# 职业性苯中毒预防与治疗实务指南

## 一、苯的基本特性

### 1.1 苯是什么

苯是一种有机溶剂，常用于：
- 制鞋行业：鞋底粘合
- 印刷行业：油墨稀释
- 电子行业：清洗剂
- 化工行业：溶剂

### 1.2 苯的危害

| 危害类型 | 具体表现 |
|----------|----------|
| 急性中毒 | 头晕、恶心、昏迷 |
| 慢性中毒 | 贫血、白细胞减少 |
| 致癌性 | 白血病 |
| 生殖毒性 | 影响生育 |

### 1.3 高危行业

- 制鞋厂
- 印刷厂
- 电子厂
- 油漆涂料厂
- 橡胶制品厂

## 二、苯中毒的诊断

### 2.1 诊断标准（GBZ 91）

**诊断原则：**
1. 职业接触史（6个月以上）
2. 出现临床表现
3. 血常规异常
4. 排除其他疾病

### 2.2 诊断分期

| 分期 | 标准 |
|------|------|
| 观察对象 | 血常规异常，无明显症状 |
| 轻度中毒 | 头晕、乏力，白细胞＜4×10⁹/L |
| 中度中毒 | 贫血，血小板减少 |
| 重度中毒 | 再生障碍性贫血、白血病 |

### 2.3 临床表现

**神经系统：**
- 头晕、头痛
- 乏力、失眠
- 记忆力减退

**血液系统：**
- 贫血：面色苍白、乏力
- 出血：牙龈出血、皮下瘀斑
- 感染：反复发热

## 三、苯中毒的预防

### 3.1 企业措施

#### 替代工艺（首选）
```
用水性胶代替苯系胶
用低苯或无苯溶剂替代
```

#### 工程控制
- 通风设施：局部通风+全通风
- 隔离操作：自动化设备
- 监测报警：苯浓度监测仪

#### 个人防护
| 防护用品 | 要求 |
|----------|------|
| 防毒面具 | 有机蒸气滤罐 |
| 防护手套 | 防渗透 |
| 工作服 | 专人专用 |

### 3.2 员工防护

1. **严格防护**
   - 上班必须戴防毒面具
   - 禁止徒手操作
   - 严禁车间进食

2. **卫生习惯**
   - 班后沐浴
   - 换掉工作服
   - 单独存放工作服

3. **健康监测**
   - 每年体检
   - 血常规检查
   - 如有不适立即就医

### 3.3 工作场所标准

| 项目 | 标准值 |
|------|--------|
| 苯浓度（PC-TWA） | 6 mg/m³ |
| 苯浓度（MAC） | 10 mg/m³ |
| 监测频次 | 每月1次 |

## 四、苯中毒的治疗

### 4.1 治疗原则

```
脱离苯接触 → 对症治疗 → 营养支持 → 康复
```

### 4.2 治疗方法

#### 药物治疗

| 药物 | 作用 |
|------|------|
| 维生素B族 | 营养神经 |
| 铁剂 | 治疗贫血 |
| 升白药 | 提高白细胞 |
| 止血药 | 控制出血 |

#### 支持治疗

- 输血：严重贫血时
- 隔离：预防感染
- 营养：高蛋白饮食

### 4.3 分期治疗

**轻度中毒：**
- 脱离接触
- 门诊治疗
- 定期复查

**中度中毒：**
- 住院治疗
- 对症治疗
- 持续监测

**重度中毒：**
- 专科治疗
- 骨髓移植（必要时）
- 长期康复

## 五、案例警示

### 5.1 典型案例

某制鞋厂工人：
- 工龄：5年
- 症状：反复发热、乏力
- 诊断：职业性苯中毒（重度）
- 教训：未做好个人防护

### 5.2 防范要点

| 错误做法 | 正确做法 |
|----------|----------|
| 不戴口罩 | 正确佩戴防毒面具 |
| 车间吃饭 | 严禁在车间进食 |
| 身体不适不就医 | 及时体检就医 |

## 六、法规依据

- GBZ 91-2025 职业性苯中毒诊断标准
- GBZ 188-2025 职业健康监护技术规范
- GBZ 2.1-2007 工作场所有害因素职业接触限值
- 《职业病防治法》2018年修订

## 七、总结

苯中毒**可防可控**：

✅ **企业落实主体责任**
- 替代工艺
- 通风防护
- 健康监护

✅ **员工做好个人防护**
- 防护装备
- 卫生习惯
- 定期体检

✅ **早发现早治疗**
- 关注症状
- 及时就医
- 规范治疗
