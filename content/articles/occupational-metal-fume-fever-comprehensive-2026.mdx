# 职业性金属烟热：金属焊接作业的健康威胁

## 引言

**职业性金属烟热**（Occupational Metal Fume Fever），又称"铸造热"、"焊工热"，是劳动者在职业活动中因吸入金属氧化物烟尘而引起的以体温升高、白细胞增多、金属沉积为特征的急性职业病。这是一种常见的职业病，尤其在焊接、铸造、金属冶炼等行业中发病率较高。

金属烟热的特点是症状类似流感，发病急骤，康复较快，但反复发作可对健康造成累积影响。虽然该病通常不会造成永久性损害，但严重影响劳动者的健康和生活质量，也可能导致工作能力下降。

本文将从专业角度全面介绍职业性金属烟热的相关知识，帮助广大金属作业劳动者认识这一职业健康威胁，做好防护工作。

## 一、病因与流行病学

### 1.1 病因

**（一）主要致病毒金属**

| 金属 | 常见来源 | 发病特点 |
|------|----------|----------|
| 锌 | 镀锌、焊接、黄铜 | 最常见 |
| 铜 | 铜焊、铜合金焊接 | 常见 |
| 镁 | 镁合金焊接 | 常见 |
| 铁 | 焊接、铸造 | 常见 |
| 铝 | 铝焊、铝铸造 | 较常见 |
| 锰 | 焊接、炼钢 | 较少见 |
| 镍 | 不锈钢焊接 | 较少见 |
| 铬 | 不锈钢焊接 | 较少见 |

**（二）发病条件**

1. **高温作业**：金属在高温下产生氧化物烟尘
2. **通风不良**：作业环境通风差，烟尘浓度高
3. **密闭空间**：管道、罐体内作业
4. **新作业环境**：久未作业后重新开工

### 1.2 流行病学特点

- **行业分布**：焊接、铸造、冶炼、表面处理
- **工种分布**：焊工、铸造工、冶炼工、表面处理工
- **发病特点**：集体发病多见，症状相似
- **易感人群**：新工人、临时工发病率高

## 二、发病机制

### 2.1 发病机制

**（一）金属烟尘吸入**

- 金属在高温（1200-3000°C）下蒸发
- 蒸发的金属迅速氧化形成细小氧化物颗粒
- 颗粒直径多在0.1-1μm，易吸入肺泡
- 吸入量与浓度、时间成正相关

**（二）免疫反应学说**

1. **抗原抗体反应**
   - 金属氧化物作为抗原
   - 刺激机体产生抗体
   - 形成免疫复合物

2. **致热原反应**
   - 金属烟尘激活中性粒细胞
   - 释放内源性致热原
   - 引起发热反应

**（三）金属毒性作用**

- 金属直接毒性作用
- 氧化应激反应
- 细胞损伤
- 炎症反应

### 2.2 影响发病的因素

| 因素 | 影响 |
|------|------|
| 金属种类 | 锌、铜、镁最易引起 |
| 烟尘浓度 | 浓度越高，发病越快 |
| 暴露时间 | 时间越长，发病越重 |
| 通风条件 | 通风差易发病 |
| 个体差异 | 敏感性存在差异 |
| 吸烟 | 可能加重病情 |

## 三、临床表现

### 3.1 症状特点

**（一）潜伏期**

- 通常在接触金属烟尘后4-8小时发病
- 短者仅1-2小时
- 长者可达12小时

**（二）典型症状**

| 症状 | 特点 |
|------|------|
| 发热 | 体温可达38-40°C，畏寒战栗 |
| 头痛 | 额部、双侧为主 |
| 全身酸痛 | 肌肉、关节酸痛 |
| 乏力 | 全身倦怠感 |
| 咽痛 | 咽喉干燥、疼痛 |
| 咳嗽 | 干咳或少痰 |
| 胸闷 | 胸部紧闷感 |
| 恶心 | 可伴呕吐 |

**（三）发作特点**

- 症状出现快
- 发热呈周期性
- 周末或休息后易发病
- 再次接触易复发

### 3.2 体征

- **体温升高**：38-40°C
- **面色潮红**
- **咽部充血**
- **肺部呼吸音粗**
- **心率加快**
- **血压轻度升高**

### 3.3 实验室检查

**（一）血常规**

- 白细胞升高：10-20×10⁹/L
- 中性粒细胞比例增高
- 嗜酸性粒细胞可增多

**（二）血清学检查**

- 血清金属含量可升高
- 肝功能：ALT、AST可轻度升高
- 肌酸激酶可升高

**（三）影像学检查**

- 胸部X线：多数正常
- 少数可见肺纹理增粗
- 严重者可见肺水肿

### 3.4 临床过程

```
接触金属烟尘（4-8小时）
        ↓
发热前驱期（畏寒、乏力）
        ↓
急性发作期（高热、全身症状）
        ↓
退热期（大量出汗、体温下降）
        ↓
恢复期（症状消失）
```

## 四、诊断与鉴别诊断

### 4.1 诊断标准

根据GBZ 20-2019《职业性金属及其化合物烟热（金属烟热）的诊断》：

**诊断依据：**

1. 职业接触史：4-8小时内有大量金属氧化物烟尘吸入史
2. 临床表现：典型的流感样症状
3. 体温升高：≥38°C
4. 白细胞升高：≥10×10⁹/L
5. 排除其他疾病

**诊断要点：**

- 有明确的金属烟尘接触史
- 症状在接触后4-8小时出现
- 发热伴有全身症状
- 血白细胞升高
- 排除上呼吸道感染、肺炎等

### 4.2 鉴别诊断

需要与以下疾病鉴别：

| 疾病 | 鉴别要点 |
|------|----------|
| 上呼吸道感染 | 无金属接触史，病程较长 |
| 肺炎 | 咳嗽、咳痰，X线异常 |
| 流感 | 无金属接触史，流行季节 |
| 疟疾 | 周期性发热，血涂片阳性 |
| 药物热 | 用药史，无金属接触史 |
| 热射病 | 高温环境史，意识障碍 |

### 4.3 诊断注意事项

1. **发病时间**：症状出现在接触后4-8小时
2. **集体发病**：同一工作场所多人发病
3. **反复发作**：多次接触反复发热
4. **周末效应**：周末后首次上班易发病

## 五、治疗方案

### 5.1 治疗原则

1. **立即脱离接触**
2. **对症治疗**
3. **防治并发症**
4. **休息与观察**

### 5.2 治疗方法

**（一）一般处理**

- 立即脱离金属烟尘环境
- 移至空气新鲜处
- 卧床休息
- 补充水分和电解质

**（二）药物治疗**

| 药物 | 作用 | 用法 |
|------|------|------|
| 物理降温 | 降低体温 | 冰敷、温水擦浴 |
| 解热镇痛药 | 缓解症状 | 对乙酰氨基酚、布洛芬 |
| 维生素C | 抗氧化 | 口服/静脉 |
| 葡萄糖酸钙 | 缓解症状 | 静脉 |
| 抗生素 | 防治感染 | 必要时使用 |

**（三）对症治疗**

- 头痛：解热镇痛药
- 咳嗽：镇咳药
- 咽痛：含片、雾化
- 胸闷：吸氧（必要时）

### 5.3 预后

- **病程**：一般24-48小时恢复
- **预后**：良好，无后遗症
- **复发**：再次接触可复发
- **长期影响**：反复发作可能影响健康

## 六、预防措施

### 6.1 工程控制

**（一）工艺改革**

- 采用低烟尘焊接工艺
- 替代高毒性金属材料
- 自动化焊接
- 湿式作业

**（二）通风设施**

- 局部通风：焊烟净化器
- 全面通风：稀释烟尘浓度
- 通风系统维护：定期检测

**（三）作业环境**

- 保持干燥清洁
- 控制作业温度
- 减少金属表面油污

### 6.2 个人防护

**（一）呼吸防护**

| 防护用品 | 适用情况 |
|----------|----------|
| 过滤式防毒面具 | 一般焊接作业 |
| 焊烟净化器 | 通风不良环境 |
| 正压式呼吸器 | 密闭空间作业 |

**滤芯选择：**

- 针对金属烟尘选择合适滤芯
- 定期更换
- 正确佩戴

**（二）其他防护**

- 防护服：阻燃、隔热
- 防护手套
- 防护眼镜
- 耳塞（防噪声）

### 6.3 卫生保健措施

**（一）就业前体检**

- 排除禁忌证
- 呼吸系统疾病
- 心血管疾病
- 发热性疾病

**（二）在岗期间体检**

- 每年一次
- 询问症状
- 必要时胸部X线

**（三）健康教育**

- 金属烟热知识
- 防护知识
- 早期症状识别

### 6.4 安全管理

1. **作业规程**
   - 制定安全操作规程
   - 严格执行
   - 监督落实

2. **环境监测**
   - 定期检测烟尘浓度
   - 评估暴露风险
   - 及时改进

3. **应急处理**
   - 制定应急预案
   - 配备急救设备
   - 定期演练

## 七、特殊作业环境

### 7.1 密闭空间作业

- 必须进行气体检测
- 佩戴正压式呼吸器
- 专人监护
- 设置通风设备

### 7.2 高空作业

- 注意安全
- 防止烫伤
- 避免坠落

### 7.3 临时作业

- 同样需要防护
- 不可忽视
- 做好准备

## 八、法律保护与权益

### 8.1 职业病诊断

- 诊断机构：省级职业病诊断机构
- 诊断依据：GBZ 20-2019
- 需要材料：职业史、临床资料

### 8.2 工伤认定

- 属于职业病
- 认定为工伤
- 享受工伤保险待遇

### 8.3 用人单位责任

- 提供防护设施
- 组织健康检查
- 进行安全培训
- 落实防护措施

### 8.4 劳动者权利

- 知情权
- 拒绝违章权
- 检举权
- 依法索赔权

## 九、案例分析

### 案例一：焊接作业集体发病

**案情：** 某船厂焊工在密闭船舱内进行焊接作业，多人出现发热、头痛、全身酸痛症状。

**原因分析：**

- 通风不良
- 烟尘浓度高
- 未佩戴防护用具
- 作业时间长

**处理：**

- 脱离现场
- 对症治疗
- 加强通风
- 佩戴防护用具

**启示：**

- 密闭空间作业必须防护
- 通风是关键措施

### 案例二：新手焊工首次发病

**案情：** 某工厂新招聘焊工，首次进行镀锌钢管焊接，出现典型金属烟热症状。

**原因分析：**

- 新工人缺乏经验
- 防护意识不足
- 对金属烟热不了解

**启示：**

- 新工人培训很重要
- 了解疾病特点

## 十、总结

职业性金属烟热是完全可以预防的职业病。

**核心要点：**

1. 金属氧化物烟尘是致病因素
2. 锌、铜、镁最易引起
3. 症状在接触后4-8小时出现
4. 典型流感样表现
5. 预后良好
6. 预防是关键

**防护要点：**

- 做好通风设施
- 正确佩戴防护用具
- 了解早期症状
- 出现症状及时就医

让我们共同关注金属烟热防护，守护劳动者健康！
