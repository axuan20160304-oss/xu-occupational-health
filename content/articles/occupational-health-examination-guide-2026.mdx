# 职业健康检查全攻略：守护劳动者健康的第一道防线

## 引言

**职业健康检查**是职业卫生工作的重要组成部分，是预防职业病的第一道防线。通过系统的健康检查，可以早期发现职业禁忌证、疑似职业病，及时采取干预措施，保护劳动者身心健康。《中华人民共和国职业病防治法》明确规定，对从事接触职业病危害作业的劳动者，用人单位应当按照国务院卫生行政部门的规定组织职业健康检查。

本文将系统介绍职业健康检查的相关知识，帮助广大劳动者和企业正确理解和执行这项重要的健康保障制度。

## 一、职业健康检查的法律依据

### 1.1 主要法律法规

| 法律法规 | 相关条款 | 主要内容 |
|----------|----------|----------|
| 《职业病防治法》 | 第三十五条 | 用人单位必须组织职业健康检查 |
| 《职业健康检查管理办法》 | 全文 | 规范职业健康检查工作 |
| GBZ 188-2014 | 全文 | 职业健康监护技术规范 |
| 《用人单位职业健康监护监督管理办法》 | 全文 | 用人单位责任与义务 |

### 1.2 法律强制性与否

职业健康检查具有法律强制性：

1. **上岗前检查**：必须检查，费用由用人单位承担
2. **在岗期间检查**：定期检查，强制执行
3. **离岗时检查**：必须检查，脱离有害岗位时进行
4. **应急检查**：事故发生后必须检查
5. **离岗后检查**：针对某些职业病需要随访

### 1.3 不承担检查的法律后果

用人单位未组织职业健康检查的法律责任：

- 给予警告，责令限期改正
- 处5万元以上30万元以下罚款
- 情节严重的，责令停止产生职业病危害的作业
- 劳动者有权拒绝从事接触职业病危害的作业

## 二、职业健康检查的类型

### 2.1 上岗前职业健康检查

**目的：**

- 发现职业禁忌证
- 评价劳动者是否适合从事该作业
- 建立基础健康档案

**对象：**

- 新录用劳动者
- 变换工作岗位劳动者
- 变换工作内容劳动者
- 特殊作业人员

**特点：**

- 强制性，必须在上岗前完成
- 针对性，根据接触危害因素确定检查项目
- 筛选性，排除不适合接触该危害因素的人员

### 2.2 在岗期间职业健康检查

**目的：**

- 早期发现职业病或职业禁忌证
- 监测劳动者健康状况变化
- 评估防护措施效果

**分类 类型 | 检查周期 | 适用：**

|情况 |
|------|----------|----------|
| 定期检查 | 1年/2年/3年 | 一般危害因素 |
| 重点监护 | 6个月 | 高危危害因素 |
| 动态观察 | 缩短周期 | 异常发现需复查 |

**周期确定原则：**

- 危害因素毒性越强，检查周期越短
- 暴露浓度越高，检查周期越短
- 已有异常发现者需缩短周期

### 2.3 离岗时职业健康检查

**目的：**

- 明确劳动者离开时的健康状况
- 分清健康损害责任
- 为后续健康随访提供依据

**要点：**

- 必须在劳动者离开岗位前完成
- 如实在个月内进行，离岗后仍需检查
- 检查项目与在岗期间相同

### 2.4 应急职业健康检查

**发生下列情况时必须进行：**

1. 发生急性职业病事故
2. 接触史不明确但可能存在大量暴露
3. 特殊应急情况需要确认健康状况
4. 上级部门要求的应急检查

**特点：**

- 立即开展，时间紧迫
- 针对性检查，聚焦可能损害
- 扩大检查范围

### 2.5 离岗后健康检查

**适用情况：**

- 职业性苯中毒、砷中毒、镉中毒等
- 尘肺病
- 放射性疾病
- 某些慢性职业病

**随访要求：**

- 尘肺病患者：每年1次
- 苯接触者：离开后继续随访3年
- 镉接触者：离开后继续随访7年

## 三、主要危害因素的健康检查项目

### 3.1 粉尘作业

**必检项目：**

- 既往病史
- 症状询问（咳嗽、胸闷等）
- 内科检查
- 胸部X线检查
- 肺功能检查

**选检项目：**

- 心电图
- 血常规
- 尿常规

**职业禁忌证：**

1. 活动性肺结核
2. 慢性阻塞性肺疾病
3. 弥漫性肺纤维化
4. 严重慢性呼吸系统疾病

### 3.2 噪声作业

**必检项目：**

- 既往听力病史
- 耳科检查
- 纯音气导听阈测试
- 症状询问（耳鸣、听力下降等）

**选检项目：**

- 声导抗测试
- 耳声发射

**职业禁忌证：**

1. 永久性感音神经性听力损失
2. 中度以上传导性听力损失
3. 各种原因引起的平衡功能障碍

### 3.3 高温作业

**必检项目：**

- 既往病史
- 内科检查
- 血压测量
- 心电图
- 血常规、尿常规

**选检项目：**

- 肝功能
- 肾功能

**职业禁忌证：**

1. Ⅱ期高血压
2. 活动性消化性溃疡
3. 慢性肾炎
4. 未控制的甲亢
5. 大面积皮肤疤痕

### 3.4 苯作业

**必检项目：**

- 既往血液病史
- 内科检查
- 血常规（必检，包含血小板）
- 肝功能

**选检项目：**

- 尿常规
- 肾功能

**职业禁忌证：**

1. 血常规异常（白细胞、中性粒细胞、血小板）
2. 造血系统疾病
3. 严重的肝肾疾病

### 3.5 铅作业

**必检项目：**

- 神经系统症状询问
- 血常规
- 尿铅
- 血铅

**选检项目：**

- 尿δ-ALA
- 神经肌电图
- 肾功能

**职业禁忌证：**

1. 贫血
2. 神经系统疾病
3. 肝肾功能异常

### 3.6 放射性作业

**必检项目：**

- 既往放射线接触史
- 内科检查
- 血常规
- 晶状体检查
- 甲状腺检查

**选检项目：**

- 肝肾功能
- 染色体畸变分析

**职业禁忌证：**

1. 严重的血液系统疾病
2. 活动性肝肾疾病
3. 晶状体混浊
4. 甲状腺功能异常

## 四、体检结果分析与处理

### 4.1 体检结论类型

| 结论 | 含义 | 处理 |
|------|------|------|
| 未见异常 | 检查结果正常 | 正常从事接触危害作业 |
| 职业禁忌证 | 不适合从事该作业 | 调离原岗位 |
| 疑似职业病 | 可能有职业病 | 进一步诊断 |
| 其他疾病 | 与职业无关的疾病 | 临床诊治 |
| 复查 | 检查结果不明确 | 按时复查 |

### 4.2 职业禁忌证的处理

**原则：**

- 立即调离禁忌证相应岗位
- 安排适当的其他工作
- 不得降低其工资、福利待遇
- 不得解除劳动合同

**常见职业禁忌证处置：**

| 危害因素 | 禁忌证 | 调离岗位 |
|----------|--------|----------|
| 粉尘 | 活动性肺结核 | 粉尘作业 |
| 噪声 | 中度以上听力损失 | 噪声作业 |
| 高温 | Ⅱ期高血压 | 高温作业 |
| 苯 | 血常规异常 | 苯作业 |
| 铅 | 贫血 | 铅作业 |

### 4.3 疑似职业病的处理

**处理流程：**

1. 及时向用人单位报告
2. 填写《职业病报告卡》
3. 到职业病诊断机构申请诊断
4. 用人单位应当如实提供相关材料
5. 诊断期间，用人单位不得解除劳动合同

**劳动者权利：**

- 免费进行职业病诊断
- 诊断费用由用人单位承担
- 对诊断结论有异议可申请鉴定

## 五、体检注意事项

### 5.1 劳动者注意事项

**检查前：**

1. 携带本人身份证
2. 核对体检表个人信息
3. 如实填写职业史、病史
4. 按照规定禁食（部分检查需要）
5. 避免剧烈运动

**检查中：**

1. 如实回答医生询问
2. 配合各项检查
3. 有不适及时告知
4. 不要隐瞒病史

**检查后：**

1. 确认体检表填写完整
2. 了解体检结论
3. 如有异常及时复查
4. 妥善保管体检结果

### 5.2 用人单位注意事项

**组织准备：**

1. 制定年度体检计划
2. 选择有资质体检机构
3. 提前通知劳动者
4. 准备劳动者名册

**现场配合：**

1. 提供准确危害因素信息
2. 安排好生产工作
3. 配合体检机构工作
4. 维持体检秩序

**后续处理：**

1. 及时获取体检报告
2. 告知劳动者体检结果
3. 落实禁忌证调离
4. 妥善保管档案

## 六、职业健康监护档案管理

### 6.1 档案内容

| 内容 | 说明 |
|------|------|
| 劳动者个人信息 | 姓名、身份证、联系方式 |
| 职业史 | 岗位、接触危害因素、工龄 |
| 危害因素接触史 | 浓度/强度、防护措施 |
| 历次体检结果 | 各项检查数据 |
| 诊断证明 | 职业病诊断资料 |
| 劳动能力鉴定 | 伤残等级资料 |

### 6.2 档案管理要求

**保存期限：**

- 劳动者离岗后至少保存20年
- 发生职业病事故的永久保存
- 粉尘作业档案终身保存

**管理要求：**

1. 专人负责
2. 保密管理
3. 便于查阅
4. 及时更新

**劳动者权利：**

- 有权查阅本人档案
- 有权获取复印件
- 离开时可要求提供

## 七、体检机构选择与管理

### 7.1 机构资质要求

**必须具备：**

1. 《医疗机构执业许可证》
2. 职业健康检查机构批准证书
3. 相应检查项目资质
4. 专业人员配备
5. 仪器设备条件

### 7.2 机构选择要点

1. 核查资质证书
2. 了解专业能力
3. 考察设备条件
4. 评估服务质量
5. 考虑距离便利
6. 合理价格

### 7.3 违规行为识别

**常见问题：**

1. 超资质范围检查
2. 简化检查项目
3. 出具虚假结论
4. 不配合调离
5. 泄露劳动者隐私

**应对措施：**

- 拒绝违规检查
- 向卫生行政部门举报
- 依法维护权益

## 八、常见问题解答

### Q1：职业健康检查费用由谁承担？

A：根据法律规定，职业健康检查费用由用人单位承担，包括检查费、复查费、诊断费等。劳动者不需要承担任何费用。

### Q2：拒绝体检会怎样？

A：劳动者应当配合用人单位进行职业健康检查。无正当理由拒绝检查的，用人单位有权按照规章制度处理。但对于从事高危危害作业的劳动者，拒绝检查可能影响其从事该作业的资格。

### Q3：体检发现异常但不是职业病怎么办？

A：对于发现的异常，应根据具体情况处理：
- 属于职业禁忌证的，应调离原岗位
- 属于其他疾病的，应建议临床诊治
- 用人单位应给予必要关心和适当安排

### Q4：对体检结论有异议怎么办？

A：劳动者对体检结论有异议的，可以：
- 向体检机构提出复查
- 向卫生行政部门投诉
- 申请职业病诊断鉴定

### Q5：换单位后档案怎么办？

A：劳动者离开原单位时，有权要求原单位提供职业健康监护档案复印件。新用人单位应当为劳动者建立新的健康监护档案，并继续进行健康监护。

## 九、企业实施要点

### 9.1 建立管理制度

1. 制定职业健康检查管理制度
2. 明确职责分工
3. 落实责任到人
4. 建立考核机制

### 9.2 制定年度计划

1. 梳理接触危害因素岗位
2. 确定检查人员名单
3. 安排检查时间
4. 落实经费预算

### 9.3 落实后续处理

1. 及时获取并分析报告
2. 告知劳动者结果
3. 落实调离要求
4. 记录处理情况

### 9.4 持续改进

1. 分析体检数据
2. 评估防护效果
3. 改进防护措施
4. 优化检查安排

## 十、总结

职业健康检查是保护劳动者健康的重要制度，需要企业和劳动者共同参与：

**企业责任：**

- 依法组织职业健康检查
- 承担全部费用
- 落实禁忌证调离
- 妥善保管档案

**劳动者义务：**

- 配合参加检查
- 如实告知健康状况
- 遵守防护要求
- 关注自身健康

**核心要点：**

1. 上岗前、在岗期间、离岗时三类检查必须执行
2. 职业禁忌证必须调离原岗位
3. 疑似职业病应及时诊断
4. 档案要妥善保管至少20年
5. 劳动者有知情权和复查权

让我们共同重视职业健康检查，为劳动者筑起一道坚实的健康防线！
