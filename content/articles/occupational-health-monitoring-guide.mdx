# 职业健康监护实务指南

## 一、职业健康监护的概念

职业健康监护是以预防为目的，对接触职业病危害因素的劳动者进行医学检查。

### 1.1 监护目的

- 早期发现职业病、职业禁忌证
- 评价职业病危害因素接触程度
- 为制定预防措施提供依据

### 1.2 监护类别

| 类别 | 时间 | 目的 |
|------|------|------|
| 上岗前检查 | 入职前 | 发现禁忌证 |
| 在岗期间检查 | 工作期间 | 早期发现疾病 |
| 离岗时检查 | 离职前 | 确认健康状况 |
| 应急检查 | 事故发生时 | 评估健康影响 |

## 二、上岗前职业健康检查

### 2.1 检查目的

- 了解劳动者健康状况
- 发现职业禁忌证
- 为后续检查提供基础数据

### 2.2 必检项目

| 检查项目 | 主要目的 |
|----------|----------|
| 血常规 | 发现血液系统疾病 |
| 尿常规 | 发现泌尿系统疾病 |
| 肝功能 | 发现肝脏疾病 |
| 肾功能 | 发现肾脏疾病 |
| 胸片 | 发现肺部疾病 |
| 心电图 | 发现心脏疾病 |

### 2.3 常见职业禁忌证

**铅作业禁忌证：**
- 神经系统疾病
- 贫血
- 肝肾功能不全

**苯作业禁忌证：**
- 血液系统疾病
- 肝病

**粉尘作业禁忌证：**
- 肺结核
- 严重肺气肿
- 支气管哮喘

## 三、在岗期间职业健康检查

### 3.1 检查周期

| 危害因素 | 检查周期 |
|----------|----------|
| 铅 | 每年1次 |
| 苯 | 每年1次 |
| 粉尘（≥10倍PC-TWA） | 每年2次 |
| 噪声 | 每年1次 |
| 高温 | 夏季前 |

### 3.2 检查项目

**铅作业：**
- 症状询问（头晕、乏力、四肢麻木等）
- 体格检查（神经系统）
- 血常规
- 尿铅、血铅
- 肝肾功能

**噪声作业：**
- 症状询问（耳鸣、听力下降）
- 纯音气导听阈测试
- 内科检查

### 3.3 异常结果处理

| 结果 | 处理 |
|------|------|
| 职业禁忌证 | 调离原岗位 |
| 疑似职业病 | 职业病诊断 |
| 异常 | 复查、跟踪观察 |

## 四、离岗时职业健康检查

### 4.1 检查时间

- 劳动者离开接触职业病危害因素岗位前
- 或在离岗后180天内

### 4.2 检查项目

- 同在岗期间检查
- 必要时增加特殊检查项目

## 五、应急职业健康检查

### 5.1 适用情况

- 事故性接触
- 急性职业中毒
- 放射事故

### 5.2 检查内容

- 症状和体征检查
- 必要的实验室检查
- 必要时进行职业病诊断

## 六、职业健康检查档案管理

### 6.1 档案内容

- 劳动者健康检查表
- 检查报告
- 职业病诊断证明
- 复查记录

### 6.2 保存期限

- **不少于30年**
- 离岗后继续保存10年

### 6.3 档案利用

- 劳动者可查阅本人档案
- 用人单位可依法使用
- 监管部门可依法检查

## 七、用人单位的职责

### 7.1 组织义务

- 制定职业健康检查计划
- 承担检查费用
- 安排劳动者按时参检

### 7.2 结果告知

- 书面告知劳动者检查结果
- 职业禁忌证书面通知劳动者本人
- 疑似职业病报告卫生行政部门

### 7.3 处置措施

- 职业禁忌证：调离原作业岗位
- 疑似职业病：安排职业病诊断
- 职业病：依法享受待遇

## 八、常见问题解答

### Q1: 职业健康检查费用由谁承担？

A: 由用人单位承担。

### Q2: 劳动者可以不参加职业健康检查吗？

A: 不可以。用人单位应当组织劳动者参加职业健康检查。

### Q3: 对检查结果有异议怎么办？

A: 可向用人单位提出复检要求，或向卫生行政部门投诉。
