# 职业性砷中毒的诊断标准与驱砷治疗

## 概述

职业性砷中毒（Occupational Arsenic Poisoning）是指劳动者在职业活动中因接触砷及其化合物而引起的中毒性疾病。砷及其化合物广泛应用于冶金、农药、化工、电子、玻璃、皮革等行业。《职业病分类和目录》将砷及其化合物中毒列为法定职业病。

砷中毒的特点：
- **蓄积性强**：砷可在体内蓄积，长期低剂量暴露也可引起中毒
- **多系统损害**：可引起神经系统、消化系统、皮肤等多系统损害
- **慢性中毒常见**：职业性砷中毒多为慢性中毒

## 中毒机制

### 毒理学特点

砷（As）是一种类金属元素，原子量74.92。职业性砷中毒主要因接触三氧化二砷（砒霜）、砷酸盐、亚砷酸盐等化合物引起。

### 病理生理

1. **巯基结合**：砷与体内含巯基（-SH）的酶结合，抑制酶活性
2. **氧化损伤**：砷可引起细胞氧化应激反应
3. **DNA损伤**：砷具有致癌性和致畸性
4. **血管损伤**：砷可引起血管内皮细胞损伤

### 毒性分级

| 尿砷含量（mg/L） | 病情程度 |
|-------------------|----------|
| <0.2 | 正常范围 |
| 0.2-0.5 | 轻度砷负荷 |
| 0.5-1.0 | 中度砷负荷 |
| >1.0 | 砷中毒 |

## 诊断标准

### 诊断原则

根据《GBZ 83-2013 职业性砷中毒的诊断》，诊断需满足以下条件：

1. **明确的砷接触史**：有确切的职业性砷接触史
2. **符合临床表现**：出现头痛、头晕、乏力、消化道症状、皮肤损害等
3. **实验室检查异常**：尿砷、发砷含量增高
4. **排除其他疾病**：排除其他原因引起的类似疾病

### 诊断依据

#### 一、职业接触史调查

| 行业 | 接触砷化物 | 常见工种 |
|------|------------|----------|
| 冶金 | 砷矿石、冶炼烟尘 | 冶炼工、烧结工 |
| 农药 | 砷农药生产 | 农药制作工 |
| 化工 | 砷酸盐、亚砷酸盐 | 化工操作工 |
| 电子 | 砷化镓 | 电子材料制备工 |
| 玻璃 | 砷制剂 | 玻璃熔制工 |

#### 二、临床表现

##### 急性砷中毒

- **消化系统**：恶心、呕吐、腹痛、腹泻（水样便或米泔水样便）
- **神经系统**：头痛、眩晕、意识障碍
- **心血管系统**：心律失常、血压下降
- **肾脏**：蛋白尿、血尿，甚至急性肾衰竭

##### 慢性砷中毒

- **神经系统**：头痛、头晕、记忆力减退、末梢神经炎
- **消化系统**：食欲不振、消化不良、肝肿大
- **皮肤**：色素沉着、角化过度、疣状赘生物（砷疔）
- **指甲**：Mees纹（指甲白色横纹）
- **呼吸道**：鼻咽部刺激症状、萎缩性鼻炎

#### 三、实验室检查

1. **尿砷测定**：尿砷>0.2mg/L提示砷负荷增加
2. **发砷测定**：发砷>1μg/g提示砷接触史
3. **血砷测定**：血砷>0.1mg/L有诊断意义
4. **肝肾功能**：转氨酶升高、肌酐尿素氮升高
5. **心电图**：心律失常、QT间期延长

### 诊断分级

#### 急性砷中毒

| 分级 | 临床表现 |
|------|----------|
| 轻度 | 头痛、头晕、恶心、呕吐、腹泻，尿砷增高 |
| 重度 | 严重消化道症状、意识障碍、心律失常、肾衰竭 |

#### 慢性砷中毒

| 分级 | 临床表现 |
|------|----------|
| 轻度 | 头痛、头晕、乏力，尿砷或发砷增高 |
| 中度 | 明显神经系统症状，皮肤色素沉着、角化过度 |
| 重度 | 末梢神经炎、肝肾功能损害、砷疔（癌前病变） |

## 鉴别诊断

### 需要鉴别的疾病

1. **急性胃肠炎**：食物中毒、细菌性痢疾
2. **病毒性肝炎**：甲肝、乙肝等
3. **末梢神经炎**：糖尿病性、药物性神经炎
4. **皮肤病**：湿疹、银屑病

### 鉴别要点

| 疾病 | 关键鉴别特征 |
|------|--------------|
| 急性胃肠炎 | 无砷接触史，群发性，细菌培养阳性 |
| 病毒性肝炎 | 肝炎病毒标志物阳性，肝功能明显异常 |
| 糖尿病神经炎 | 血糖升高，无砷接触史 |

## 驱砷治疗

### 治疗原则

1. **立即终止接触**：脱离砷污染环境
2. **特效解毒**：使用金属络合剂驱砷
3. **对症支持治疗**：保护重要脏器功能
4. **皮肤损害处理**：局部治疗为主

### 特效解毒剂

#### 一、二巯基丙磺酸钠（Na-DMPS）

| 项目 | 内容 |
|------|------|
| 适应症 | 急慢性砷中毒 |
| 用法 | 5mg/kg/d，分2-3次肌肉注射 |
| 疗程 | 急性中毒7-10天，慢性中毒15-20天 |
| 不良反应 | 头晕、恶心、过敏反应 |

#### 二、二巯基丁二酸（DMSA）

| 项目 | 内容 |
|------|------|
| 适应症 | 砷、汞、铅等金属中毒 |
| 用法 | 成人1g/d，分2-4次口服 |
| 疗程 | 5-7天，间隔2-3天可重复 |
| 不良反应 | 胃肠道反应、肝功能异常 |

#### 三、依地酸钙钠（CaNa₂EDTA）

| 项目 | 内容 |
|------|------|
| 适应症 | 铅、镉等金属中毒，对砷效果较差 |
| 用法 | 1g/d，静脉滴注或肌肉注射 |
| 疗程 | 3-5天，间隔3-4天 |
| 注意 | 需要与DMSA或Na-DMPS联合使用 |

### 对症支持治疗

#### 一、急性砷中毒

- **补液**：纠正脱水和电解质紊乱
- **镇吐**：甲氧氯普胺、维生素B6
- **保护肝脏**：还原型谷胱甘肽、硫普罗宁
- **保护肾脏**：碱化尿液，必要时血液净化
- **心律失常处理**：抗心律失常药物

#### 二、慢性砷中毒

- **营养神经**：维生素B1、B6、B12
- **皮肤损害**：维A酸软膏、尿素软膏
- **砷疔处理**：手术切除或光动力治疗，定期随访

### 康复治疗

1. **营养支持**：高蛋白、高维生素饮食
2. **运动康复**：适度锻炼，改善体质
3. **心理康复**：缓解焦虑、抑郁情绪
4. **长期随访**：监测尿砷、肝肾功能

## 劳动能力鉴定

### 鉴定标准

根据《劳动能力鉴定 职工工伤与职业病致残等级》（GB/T 16180）：

| 伤残等级 | 鉴定标准 |
|----------|----------|
| 四级 | 慢性砷中毒合并肝硬化、肾衰竭 |
| 五级 | 慢性砷中毒合并严重神经系统损害 |
| 六级 | 慢性砷中毒合并明显皮肤损害 |

### 鉴定要点

- 脱离接触后症状是否持续
- 脏器功能损害程度
- 是否需要长期治疗

## 预防措施

### 工程控制

- **湿式作业**：减少粉尘飞扬
- **通风排毒**：安装局部通风设施
- **自动化生产**：减少人工操作
- **监测检测**：定期检测空气中砷浓度

### 个人防护

- 防尘口罩（过滤效率≥95%）
- 防护服和防护手套
- 作业后沐浴更衣
- 严禁在车间进食、吸烟

### 健康监护

- **上岗前体检**：筛选肝病、皮肤病
- **定期在岗体检**：每年1次，尿砷、发砷检测
- **离岗体检**：砷负荷评估

## 总结

职业性砷中毒的诊断需要综合职业接触史、临床表现和实验室检查结果。驱砷治疗是特效治疗，常用二巯基丙磺酸钠和二巯基丁二酸。慢性砷中毒需长期随访，监测可能发生的皮肤癌变。用人单位应落实职业卫生防护措施，预防砷中毒的发生。
