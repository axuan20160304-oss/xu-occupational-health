# 职业性手臂振动病：手持振动工具工人的健康威胁

## 引言

**职业性手臂振动病**（Occupational Vibration-Induced White Finger，VWF），又称振动性白指（Vibration-Induced Raynaud's Phenomenon），是劳动者长期从事手持振动工具作业而引起的以手部血管和神经损害为主的职业病。这种疾病主要表现为手指发白、发紫、麻木、疼痛等症状，严重影响劳动者的生活质量和工作能力。

手臂振动病是典型的职业病，在凿岩工、采矿工、铸造工、林业伐木工、装修工人等群体中发病率较高。随着工业化进程加快和手持振动工具的广泛应用，该疾病的发病率呈上升趋势。本文将从专业角度全面介绍职业性手臂振动病的相关知识，帮助广大劳动者认识这一职业健康威胁。

## 一、振动概述

### 1.1 振动的基本概念

振动是指物体在外力作用下沿直线或弧线相对于平衡位置进行的往复运动。在职业卫生中，主要关注的是传递到人体的机械振动。

**振动的主要参数：**

| 参数 | 定义 | 单位 |
|------|------|------|
| 频率 | 每秒振动的次数 | Hz |
| 振幅 | 振动位移的最大值 | mm |
| 加速度 | 速度变化的快慢 | m/s² |
| 位移 | 振动时位置的变化 | mm |

### 1.2 振动的分类

**（一）按振动方向分类**

- 垂直振动：上下方向的振动
- 水平振动：前后或左右方向的振动
- 复合振动：多方向同时振动

**（二）按振动频率分类**

| 频率范围 | 特点 |
|----------|------|
| <1 Hz | 低频振动，前庭刺激明显 |
| 1-20 Hz | 中频振动，共振风险 |
| 20-1000 Hz | 高频振动，感觉神经损伤 |
| >1000 Hz | 超声振动，实际意义不大 |

**（三）按振动来源分类**

- 机械振动：凿岩机、电钻、风铲等
- 运输振动：车辆、船舶、飞机
- 建筑振动：打桩机、压路机

### 1.3 常见振动工具

| 工种 | 振动工具 | 振动特点 |
|------|----------|----------|
| 凿岩工 | 凿岩机、风钻 | 频率高、振幅大 |
| 采矿工 | 采煤机、掘进机 | 复合振动 |
| 铸造工 | 造型机、清砂机 | 低频振动 |
| 伐木工 | 油锯、电锯 | 高频振动 |
| 装修工 | 电钻、冲击钻 | 高频振动 |
| 混凝土工 | 振动棒 | 低频振动 |
| 铆工 | 铆钉枪 | 脉冲振动 |
| 纺织工 | 织布机 | 低频振动 |

## 二、病因与发病机制

### 2.1 致病因素

**（一）振动因素**

- 振动强度：加速度越大，危害越重
- 振动频率：20-200 Hz危害最大
- 振幅：振幅越大，能量越高
- 暴露时间：每日接触时间越长，危害越大

**（二）环境因素**

- 寒冷：低温是重要诱因
- 潮湿：加重血管收缩
- 噪声：协同作用

**（三）个体因素**

- 性别：女性更敏感
- 年龄：年龄越大，敏感性越高
- 吸烟：加重血管损害
- 职业年限：工龄越长，发病率越高

### 2.2 发病机制

**（一）血管机制**

1. **交感神经反应**
   - 振动刺激引起交感神经兴奋
   - 导致血管收缩
   - 长期作用导致血管结构改变

2. **血管内皮损伤**
   - 振动直接损伤血管内皮细胞
   - 引起内皮功能障碍
   - 导致血管舒缩异常

3. **血液流变学改变**
   - 血小板聚集增加
   - 血液粘稠度增高
   - 微循环障碍

**（二）神经机制**

1. **感觉神经损伤**
   - 振动引起神经纤维脱髓鞘
   - 导致感觉减退、麻木
   - 严重者出现感觉障碍

2. **运动神经损伤**
   - 肌力下降
   - 肌肉萎缩
   - 精细动作障碍

**（三）肌肉骨骼机制**

1. **肌肉损伤**
   - 长期振动导致肌肉疲劳
   - 肌纤维萎缩
   - 肌肉力量下降

2. **骨关节损伤**
   - 关节软骨磨损
   - 骨质增生
   - 退行性改变

### 2.3 影响发病的因素

| 因素 | 影响 |
|------|------|
| 振动强度 | 强度越大，发病越早 |
| 振动频率 | 20-200Hz危害最大 |
| 暴露时间 | 每日数小时易发病 |
| 低温 | 显著加重病情 |
| 吸烟 | 加重血管损害 |
| 工龄 | 工龄越长，发病率越高 |
| 个体差异 | 敏感性存在差异 |

## 三、临床表现

### 3.1 典型症状

**（一）血管症状——白指**

- **发作特点**：
  - 突发性、对称性
  - 从指尖开始，向近端蔓延
  - 发作时手指苍白、发凉
  - 发作后转为紫绀、潮红

- **诱发因素**：
  - 接触冷水
  - 低温环境
  - 情绪紧张

- **发作过程**：
  1. 苍白期：手指突然变白
  2. 紫绀期：转为青紫
  3. 潮红期：恢复充血变红

**（二）神经症状**

| 症状 | 表现 |
|------|------|
| 感觉减退 | 手掌、手指感觉迟钝 |
| 麻木 | 持续性或阵发性麻木 |
| 疼痛 | 夜间或寒冷时加重 |
| 感觉异常 | 刺痛、蚁走感 |

**（三）运动症状**

- 握力下降
- 手指灵活性降低
- 精细动作障碍
- 肌肉萎缩

### 3.2 体征

**（一）血管检查**

- 毛细血管镜检查：可见管袢畸形
- 指动脉检查：搏动减弱
- 皮温测定：发作时皮温降低
- 甲皱微循环：异常

**（二）神经检查**

- 感觉阈值升高
- 肌力下降
- 肌肉萎缩
- 神经传导速度减慢

**（三）其他检查**

- 冷水激发试验阳性
- 握力计测定握力下降
- 手指灵活性测试异常

### 3.3 临床分期

根据GBZ 7-2014《职业性手臂振动病的诊断》：

| 分期 | 临床表现 |
|------|----------|
| 轻度 | 偶发手指发白，局限于一个手指 |
| 中度 | 多个手指经常发白，伴麻木 |
| 重度 | 频繁发作，累及所有手指，伴肌肉萎缩 |

## 四、诊断与鉴别诊断

### 4.1 诊断标准

根据GBZ 7-2014《职业性手臂振动病的诊断》：

**诊断依据：**

1. 职业史：长期从事手持振动工具作业
2. 临床表现：手指发白、麻木、疼痛等
3. 激发试验：冷水激发试验阳性
4. 排除其他疾病

**诊断要点：**

- 从事振动作业2年以上
- 出现典型的白指症状
- 冷水激发试验阳性
- 排除其他引起雷诺现象的疾病

### 4.2 鉴别诊断

需要与以下疾病鉴别：

| 疾病 | 鉴别要点 |
|------|----------|
| 雷诺病 | 无振动作业史，发病与振动无关 |
| 血栓闭塞性脉管炎 | 有吸烟史，下肢为主 |
| 系统性红斑狼疮 | 自身抗体阳性，多系统损害 |
| 硬皮病 | 皮肤硬化，自身抗体阳性 |
| 糖尿病神经病变 | 血糖升高史 |

### 4.3 辅助检查

**（一）专科检查**

- 冷水激发试验
- 指端皮温测定
- 甲皱微循环检查
- 毛细血管镜检查

**（二）神经电生理检查**

- 肌电图
- 神经传导速度
- 感觉阈值测定

**（三）影像学检查**

- X线检查：排除骨关节病变
- CT/MRI：必要时检查

**（四）实验室检查**

- 自身抗体：排除自身免疫性疾病
- 血糖：排除糖尿病

## 五、治疗方案

### 5.1 治疗原则

1. **终止振动作业**
2. **改善血液循环**
3. **营养神经**
4. **对症治疗**
5. **康复锻炼**

### 5.2 治疗方法

**（一）药物治疗**

| 药物 | 作用 | 用法 |
|------|------|------|
| 钙通道阻滞剂 | 扩张血管 | 硝苯地平、氨氯地平 |
| 前列腺素E1 | 扩张血管 | 静脉滴注 |
| 维生素B族 | 营养神经 | 口服/肌肉注射 |
| 银杏叶提取物 | 改善微循环 | 口服 |
| 丹参 | 活血化瘀 | 口服/静脉 |
| 疼痛明显可加用镇痛药物 |

**（二）物理治疗**

- 温热疗法：红外线、热敷
- 蜡疗：改善血液循环
- 超声治疗：促进组织修复
- 电刺激：兴奋肌肉

**（三）运动疗法**

- 手指屈伸练习
- 握力训练
- 精细动作训练
- 关节活动度训练

**（四）手术治疗**

- 严重者可考虑交感神经切除术
- 血管重建手术（极少数）

### 5.3 康复建议

1. **保暖**：避免寒冷刺激
2. **戒烟**：烟草加重病情
3. **避免接触振动**：调离振动作业
4. **适度锻炼**：增强体质
5. **心理调适**：保持乐观

### 5.4 预后

- **轻度**：早期治疗，可完全恢复
- **中度**：积极治疗，部分恢复
- **重度**：难以完全恢复，需长期管理
- 关键是早期发现、早期治疗

## 六、预防措施

### 6.1 工程控制

**（一）改革工艺**

- 采用低振动工具
- 自动化替代手持操作
- 改进工具设计

**（二）减振措施**

- 安装减振装置
- 使用隔振材料
- 优化工具把手

**（三）作业方式**

- 轮流使用双手
- 缩短连续作业时间
- 增加休息次数

### 6.2 个人防护

**（一）防寒保暖**

- 保暖手套
- 发热手套
- 保持手部温暖

**（二）防振手套**

- 专用防振手套
- 减少振动传递
- 注意：不建议使用普通厚手套

**（三）其他防护**

- 耳塞（防噪声）
- 防护服
- 工作后做手部放松操

### 6.3 卫生保健措施

**（一）就业前体检**

- 排除禁忌证
- 周围血管疾病
- 神经系统疾病
- 手部外伤史

**（二）在岗期间体检**

- 每年一次
- 询问症状
- 冷水激发试验
- 感觉检查

**（三）健康教育**

- 振动危害知识
- 防护知识
- 早期症状识别

### 6.4 作业环境管理

1. **控制环境温度**
   - 作业环境温度≥16°C
   - 避免低温作业

2. **保持干燥**
   - 工作服干燥
   - 手部保持干燥

3. **减少噪声**
   - 噪声与振动协同作用
   - 采取降噪措施

## 七、法律保护与权益

### 7.1 职业病诊断

- 诊断机构：省级职业病诊断机构
- 诊断依据：GBZ 7-2014
- 需要材料：职业史、临床资料

### 7.2 工伤认定

- 属于职业病
- 认定为工伤
- 享受工伤保险待遇

### 7.3 用人单位责任

- 提供低振动工具
- 组织健康检查
- 进行安全培训
- 落实防护措施

### 7.4 劳动者权利

- 知情权
- 拒绝违章权
- 检举权
- 依法索赔权

## 八、总结

职业性手臂振动病是完全可以预防的职业病。

**核心要点：**

1. 长期接触手持振动工具是病因
2. 手指发白（白指）是典型症状
3. 冷水激发试验是重要诊断方法
4. 早期治疗效果较好
5. 保暖是重要预防措施
6. 调离振动作业是根本

**防护要点：**

- 使用低振动工具
- 做好保暖措施
- 缩短作业时间
- 定期健康检查
- 出现症状及时就医

让我们共同关注手臂振动病防护，守护劳动者健康！
