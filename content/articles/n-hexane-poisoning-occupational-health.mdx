## 一、适用范围

本指南适用于从事正己烷作业的用人单位及劳动者，主要涉及以下行业：

### 1.1 主要行业

- **电子行业**：PCB清洗、线路板制造
- **印刷行业**：油墨调配、印版清洗
- **制鞋行业**：鞋底粘合、皮革处理
- **箱包制造**：粘胶作业
- **橡胶行业**：橡胶制品加工
- **油漆涂料**：有机溶剂调配
- **塑料加工**：塑料薄膜粘合
- **文具制造**：粘合剂使用

### 1.2 正己烷作业岗位

| 岗位 | 正己烷暴露形式 |
|------|-------------|
| 清洗工 | 正己烷蒸气 |
| 粘合工 | 正己烷蒸气 |
| 印刷工 | 正己烷蒸气 |
| 涂胶工 | 正己烷蒸气 |

## 二、正己烷的基本特性

### 2.1 物理特性

| 特性 | 说明 |
|------|------|
| 外观 | 无色透明液体 |
| 熔点 | -95.3℃ |
| 沸点 | 68.7℃ |
| 密度 | 0.659 g/cm³ |
| 挥发性 | 易挥发 |
| 气味 | 轻微特殊气味 |

### 2.2 毒性特点

- **主要化合物**：正己烷（n-hexane）
- **毒性**：神经毒性，蓄积性
- **侵入途径**：呼吸道、皮肤
- **靶器官**：神经系统、周围神经
- **中毒表现**：多发性周围神经病、运动障碍

### 2.3 毒代动力学

| 过程 | 特点 |
|------|------|
| 吸收 | 呼吸道90%，皮肤少量 |
| 分布 | 脂肪组织含量高 |
| 代谢 | 肝脏代谢，生成2,5-己二酮 |
| 排泄 | 尿液排出 |

## 三、职业健康检查

### 3.1 上岗前检查（GBZ 188-2025）

从事正己烷作业的劳动者应进行上岗前职业健康检查，以下疾病或情况为职业禁忌症：

**绝对禁忌症：**

| 系统 | 疾病 |
|------|------|
| 神经系统 | 周围神经病、多发性神经病 |
| 肝脏 | 慢性肝炎、肝硬化 |
| 肾脏 | 慢性肾炎、肾功能不全 |

**相对禁忌症：**

- 妊娠期、哺乳期妇女

### 3.2 在岗检查

| 检查周期 | 要求 |
|----------|------|
| 正己烷作业 | 每年1次 |

**必检项目：**

1. **症状询问**
   - 头晕、头痛、乏力
   - 四肢麻木、疼痛
   - 肌肉无力、萎缩
   - 食欲不振

2. **体格检查**
   - 神经系统检查（肌力、感觉）
   - 肌营养状况检查

3. **辅助检查**
   - 血常规
   - 尿常规
   - 肝功能
   - 肾功能
   - 神经肌电图（重点）

### 3.3 离岗检查

劳动者离开正己烷作业岗位时，应进行离岗职业健康检查。

### 3.4 应急检查

发生正己烷中毒事故时，应对暴露人群进行应急检查。

## 四、职业病诊断

### 4.1 诊断标准（GBZ 68-2013）

**诊断原则：**

1. 职业接触史（6个月以上）
2. 出现周围神经病症状
3. 尿2,5-己二酮升高
4. 排除其他疾病

### 4.2 诊断分期

| 分期 | 诊断标准 |
|------|----------|
| **观察对象** | 长期接触史，无明显症状 |
| **轻度中毒** | 出现周围神经病症状，肌力下降 |
| **中度中毒** | 肌力明显下降，运动功能障碍 |
| **重度中毒** | 瘫痪、肌肉萎缩 |

### 4.3 临床表现

#### 4.3.1 神经系统

| 症状 | 表现 |
|------|------|
| 感觉障碍 | 四肢麻木、疼痛、感觉减退 |
| 运动障碍 | 肌无力、肌肉萎缩 |
| 周围神经病 | 对称性远端感觉减退 |
| 自主神经 | 出汗异常、皮肤温度改变 |

#### 4.3.2 实验室检查

| 指标 | 表现 |
|------|------|
| 尿2,5-己二酮 | 升高 |
| 神经肌电图 | 神经源性损害 |

## 五、接触限值

### 5.1 工作场所空气中正己烷浓度限值

| 标准 | PC-TWA | MAC |
|------|---------|-----|
| 正己烷 | 100 mg/m³ | - |

### 5.2 生物监测指标

| 指标 | 正常值 | 异常值 |
|------|---------|---------|
| 尿2,5-己二酮 | <1.5 mg/L | ≥1.5 mg/L |

### 5.3 监测要求

| 监测类型 | 频次 |
|----------|------|
| 日常监测 | 每月至少1次 |
| 定期监测 | 每年至少1次 |

## 六、预防控制措施

### 6.1 工程控制措施

| 措施 | 说明 |
|------|------|
| **替代** | 用低毒溶剂替代正己烷 |
| **通风** | 局部通风、全通风 |
| **隔离** | 自动化隔离操作 |
| **湿式作业** | 减少挥发 |

### 6.2 防护设施

| 设施 | 说明 |
|------|------|
| 通风罩 | 蒸气收集 |
| 过滤装置 | 活性炭过滤 |
| 隔离操作 | 远程控制 |

### 6.3 个人防护

| 防护用品 | 要求 |
|----------|------|
| **防毒口罩** | 有机蒸气滤毒罐 |
| **防护手套** | 防渗透手套 |
| **防护眼镜** | 防护眼镜 |

### 6.4 卫生措施

- 班后沐浴
- 更换工作服
- 禁止在车间进食
- 定期体检

## 七、急救处理

### 7.1 急性中毒

**症状**：头晕、恶心、呕吐、意识障碍

**急救：**

1. 脱离正己烷作业环境
2. 立即就医
3. 对症治疗
4. 维生素B族营养神经

### 7.2 慢性中毒

**处理原则：**

1. 脱离正己烷接触
2. 营养神经治疗（维生素B1、B6、B12）
3. 物理治疗
4. 运动康复

### 7.3 治疗药物

| 药物 | 用法 |
|------|------|
| 维生素B1 | 10-20mg，口服，每日3次 |
| 维生素B6 | 10-20mg，口服，每日3次 |
| 维生素B12 | 500μg，肌注，每日或隔日1次 |
