## 一、适用范围

本指南适用于从事N,N-二甲基甲酰胺(DMF)作业的用人单位及劳动者，主要涉及以下行业：

- **化工行业**：有机合成、溶剂萃取
- **制药行业**：原料药合成、制剂
- **电子行业**：PCB清洗、覆铜板制造
- **纺织行业**：合成纤维染色
- **皮革行业**：皮革涂饰
- **印刷行业**：油墨溶剂

## 二、DMF的基本特性

### 2.1 物理特性

| 特性 | 说明 |
|------|------|
| 外观 | 无色透明液体 |
| 气味 | 轻微氨味 |
| 沸点 | 153℃ |
| 闪点 | 58℃ |
| 挥发性 | 中等挥发 |
| 溶解性 | 与水混溶，溶于多数有机溶剂 |

### 2.2 毒性特点

- **主要成分**：N,N-二甲基甲酰胺
- **毒性**：中等毒性，主要损害肝脏
- **侵入途径**：呼吸道、皮肤、消化道
- **代谢**：经肝脏代谢，肾脏排出
- **靶器官**：肝脏、胃肠道、皮肤

## 三、职业健康检查

### 3.1 上岗前检查

从事DMF作业的劳动者应进行上岗前职业健康检查，以下疾病或情况为职业禁忌症：

**禁忌症：**

- **肝脏疾病**：急慢性肝炎、肝硬化、肝功能异常
- **消化系统疾病**：慢性胃炎、胃溃疡
- **肾脏疾病**：慢性肾炎、肾功能不全
- **皮肤病**：慢性湿疹、皮炎
- **心血管疾病**：严重心脏病、心律失常

### 3.2 在岗检查

| 检查周期 | 要求 |
|----------|------|
| 接触DMF浓度≥PC-TWA | 每年1次 |
| 接触DMF浓度＜PC-TWA | 每2年1次 |

**检查项目：**

1. **症状询问**
   - 乏力、食欲不振、恶心
   - 右上腹不适、胀痛
   - 头晕、头痛
   - 皮肤瘙痒、皮疹

2. **体格检查**
   - 肝脏触诊（大小、质地、压痛）
   - 皮肤检查
   - 神经系统检查

3. **辅助检查**
   - 肝功能：ALT、AST、γ-GT、ALP、TBIL
   - 肾功能：肌酐、尿素氮
   - 血常规
   - 腹部B超（肝胆脾肾）

### 3.3 离岗检查

劳动者离开DMF作业岗位时，应进行离岗职业健康检查。

## 四、职业病诊断

### 4.1 诊断标准

依据GBZ 85-2002《职业性二甲基甲酰胺中毒诊断标准》：

**诊断原则：**

1. 短期内接触较高浓度DMF或长期密切接触DMF史
2. 出现肝脏损害为主的临床表现
3. 排除其他疾病
4. 结合职业卫生调查资料

### 4.2 诊断分期

| 分期 | 主要表现 |
|------|----------|
| **观察对象** | 肝功能异常，无明显症状 |
| **轻度中毒** | 乏力、食欲不振、恶心、呕吐、肝功能异常（ALT≥2倍正常上限） |
| **中度中毒** | 上述症状加重，出现肝脾肿大、黄疸 |
| **重度中毒** | 肝功能严重异常、凝血障碍、肝昏迷 |

### 4.3 慢性DMF中毒

长期接触DMF可导致慢性中毒：

- **肝脏损害**：慢性肝炎、肝酶升高
- **消化系统**：慢性胃炎、消化不良
- **皮肤损害**：皮炎、皮肤干燥
- **生殖系统**：月经紊乱（女性）

## 五、接触限值

### 5.1 工作场所空气中DMF浓度限值

| 标准 | PC-TWA | PC-STEL |
|------|--------|---------|
| GBZ 2.1-2007 | 20 mg/m³ | 40 mg/m³ |

### 5.2 监测要求

- **日常监测**：每月至少监测1次
- **定期监测**：每年至少进行1次全面监测
- **个体监测**：接触浓度≥50%PC-TWA者

## 六、预防控制措施

### 6.1 工程控制措施

| 措施 | 说明 |
|------|------|
| **密闭化** | 生产设备密闭化，减少泄漏 |
| **通风** | 设置全面通风和局部通风 |
| **替代** | 用低毒性溶剂替代DMF |
| **自动化** | 实现自动化生产，减少人工操作 |

### 6.2 管理措施

- **危害告知**：告知劳动者DMF危害及防护措施
- **培训教育**：开展DMF防护知识培训
- **健康监护**：严格执行职业健康检查制度
- **应急处理**：制定DMF中毒应急预案

### 6.3 个人防护

| 防护用品 | 要求 |
|----------|------|
| **防毒面具** | 有机蒸气滤罐 |
| **防护手套** | 丁腈橡胶或氯丁橡胶手套 |
| **防护服** | 防渗透工作服 |
| **防护眼镜** | 防护眼镜或防护面罩 |

### 6.4 作业环境要求

- 工作场所DMF浓度≤PC-TWA
- 设置警示标识
- 配置洗眼器和淋浴设备
- 禁止明火作业

## 七、急救处理

### 7.1 急性DMF中毒急救

**吸入中毒：**

1. 迅速脱离现场，移至空气新鲜处
2. 保持呼吸道通畅
3. 吸氧
4. 对症处理
5. 及时送医

**皮肤接触：**

1. 立即脱去被污染衣物
2. 用大量清水或肥皂水冲洗
3. 皮肤破损处按化学烧伤处理
4. 就医

**眼睛接触：**

1. 立即用流动清水冲洗
2. 冲洗时间≥15分钟
3. 就医

### 7.2 现场急救箱配置

| 物品 | 用途 |
|------|------|
| 有机蒸气口罩 | 呼吸防护 |
| 洗眼液 | 眼睛冲洗 |
| 急救手册 | 指导使用 |

## 八、健康监护档案管理

### 8.1 档案内容

- 职业史和接触史
- 既往病史
- 家族史
- 检查结果汇总
- 诊断结论
- 防护措施

### 8.2 档案保存

- 离岗后保存至少20年
- 发生职业病者终身保存
