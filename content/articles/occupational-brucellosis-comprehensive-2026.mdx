# 职业性布鲁氏菌病：人畜共患的职业传染病

## 引言

**职业性布鲁氏菌病**（Occupational Brucellosis），简称"布病"，是由布鲁氏菌引起的人畜共患传染病。布鲁氏菌是一种革兰氏阴性球杆菌，主要感染牛、羊、猪等家畜，人类通过接触感染动物或其产品而患病。在职业活动中，兽医、畜牧工作者、肉类加工工人、实验室人员等群体接触布鲁氏菌的机会较多，因此布病被列入职业病目录。

布病是一种全球范围内广泛分布的传染病，在我国主要流行于牧区、半牧区和有牛羊养殖的地区。该病病程较长，易转为慢性，反复发作，严重影响劳动者的身体健康和生活质量。本文将从专业角度全面介绍职业性布鲁氏菌病的相关知识，帮助相关职业人群认识这一人畜共患的职业传染病。

## 一、病原学概述

### 1.1 布鲁氏菌的分类

**（一）主要菌种**

| 菌种 | 主要宿主 | 致病性 |
|------|----------|--------|
| 羊种菌（B. melitensis） | 羊 | 病情最重 |
| 牛种菌（B. abortus） | 牛 | 病情较轻 |
| 猪种菌（B. suis） | 猪 | 病情较重 |
| 犬种菌（B. canis） | 狗 | 病情较轻 |

**（二）生物学特性**

- 革兰氏阴性球杆菌
- 无芽孢、无荚膜
- 需氧或兼性厌氧
- 对外界环境有较强抵抗力
- 在土壤、皮毛中可存活数周

### 1.2 感染途径

**（一）主要传播途径**

| 途径 | 说明 |
|------|------|
| 接触传播 | 接触病畜及其分泌物 |
| 呼吸道传播 | 吸入含菌气溶胶 |
| 消化道传播 | 食入未消毒的奶制品 |
| 皮肤破损 | 破损皮肤接触感染 |

**（二）职业感染途径**

- 接生患病动物
- 屠宰病畜
- 加工肉类
- 兽医诊疗
- 实验室操作

## 二、流行病学特点

### 2.1 流行概况

**（一）全球分布**

- 主要分布于 Mediterranean、中东、拉丁美洲等地区
- 我国主要分布于内蒙古、新疆、青海、宁夏等牧区

**（二）职业分布**

| 职业 | 感染风险 |
|------|----------|
| 兽医 | 高 |
| 畜牧工人 | 高 |
| 屠宰工人 | 高 |
| 肉类加工工人 | 中 |
| 实验室人员 | 中 |
| 农民 | 低 |

### 2.2 流行特点

1. **季节性**：春末夏初高发
2. **职业性**：特定职业人群高发
3. **聚集性**：家庭聚集性明显
4. **隐匿性**：轻症或不典型病例多

## 三、发病机制

### 3.1 感染过程

**（一）入侵途径**

1. 皮肤破损处入侵
2. 呼吸道黏膜入侵
3. 消化道黏膜入侵

**（二）体内播散**

- 局部淋巴结繁殖
- 进入血液循环
- 播散至全身器官

### 3.2 致病机制

**（一）细胞内寄生**

- 可在巨噬细胞内生存繁殖
- 逃避机体免疫清除
- 导致慢性感染

**（二）免疫反应**

- 体液免疫：产生抗体
- 细胞免疫：主要清除机制
- 免疫复合物：导致关节炎等

**（三）病理变化**

- 肝脾淋巴结肿大
- 肉芽肿形成
- 关节炎
- 睾丸炎

### 3.3 影响发病的因素

| 因素 | 影响 |
|------|------|
| 菌种 | 羊种菌致病性最强 |
| 感染量 | 菌量越多，发病越快 |
| 宿主因素 | 免疫力低下者易发病 |
| 防护措施 | 防护好可避免感染 |

## 四、临床表现

### 4.1 急性布鲁氏菌病

**（一）潜伏期**

- 一般1-3周
- 可短至数天，长达数月

**（二）全身症状**

| 症状 | 特点 |
|------|------|
| 发热 | 波状热最具特征 |
| 多汗 | 夜间或凌晨大量出汗 |
| 乏力 | 全身倦怠 |
| 头痛 | 持续性头痛 |
| 肌肉疼痛 | 大肌肉群疼痛 |
| 关节疼痛 | 膝、髋、肩关节 |

**（三）局部症状**

- 肝脾肿大
- 淋巴结肿大
- 睾丸炎（男性）
- 附睾炎（男性）
- 卵巢炎（女性）

### 4.2 慢性布鲁氏菌病

**（一）慢性期表现**

- 低热
- 乏力
- 关节疼痛
- 肌肉疼痛

**（二）后遗损害**

- 关节炎
- 脊柱炎
- 神经炎
- 睾丸萎缩
- 疲劳综合征

### 4.3 并发症

1. **骨关节损害**
   - 关节炎
   - 脊柱炎
   - 骨髓炎

2. **神经系统损害**
   - 脑膜炎
   - 神经炎
   - 精神障碍

3. **心血管损害**
   - 心内膜炎
   - 血管炎

4. **其他**
   - 肝炎
   - 肺炎

## 五、诊断与鉴别诊断

### 5.1 诊断标准

根据GBZ 227-2016《职业性布鲁氏菌病诊断标准》：

**诊断依据：**

1. 职业接触史：从事布鲁氏菌相关职业
2. 临床表现：发热、多汗、关节痛等
3. 实验室检查：血清学或细菌学阳性
4. 排除其他疾病

**诊断分级：**

| 分级 | 标准 |
|------|------|
| 疑似病例 | 职业史+临床表现 |
| 确诊病例 | 疑似+实验室阳性 |
| 临床诊断病例 | 疑似+虎红平板阳性 |

### 5.2 实验室检查

**（一）血清学检查**

| 方法 | 特点 |
|------|------|
| 虎红平板凝集试验 | 初筛试验 |
| 试管凝集试验 | 确诊试验 |
| ELISA | 敏感度高 |
| 补体结合试验 | 特异性高 |

**（二）细菌学检查**

- 血培养阳性
- 骨髓培养阳性
- 脓液培养阳性

**（三）分子生物学检查**

- PCR检测
- 基因测序

### 5.3 鉴别诊断

需要与以下疾病鉴别：

| 疾病 | 鉴别要点 |
|------|----------|
| 伤寒 | 血培养、肥达试验 |
| 结核病 | 结核菌素试验、X线 |
| 风湿热 | ASO、关节症状 |
| 类风湿关节炎 | RF、抗CCP |
| 流感 | 流行病学、病毒检测 |

## 六、治疗方案

### 6.1 治疗原则

1. **早期治疗**
2. **联合用药**
3. **足疗程**
4. **对症处理**

### 6.2 抗菌治疗

**（一）常用方案**

| 方案 | 药物 | 疗程 |
|------|------|------|
| 标准方案 | 多西环素+利福平 | 6周 |
| WHO推荐 | 多西环素+利福平+链霉素 | 6周 |
| 替代方案 | 多西环素+复方新诺明 | 6周 |

**（二）特殊人群治疗**

- 儿童：利福平+复方新诺明
- 孕妇：利福平+阿莫西林

### 6.3 对症治疗

1. **解热镇痛**：缓解发热和疼痛
2. **保肝治疗**：肝功能异常时
3. **免疫调节**：严重者可使用
4. **局部处理**：脓肿切开引流

### 6.4 慢性期治疗

- 抗生素治疗（延长疗程）
- 对症治疗
- 免疫治疗
- 功能锻炼

### 6.5 预后

- **急性期**：大多可治愈
- **慢性期**：难以根治，易复发
- **死亡**：极少（<1%）
- **影响**：可影响劳动能力

## 七、预防措施

### 7.1 控制传染源

**（一）动物管理**

- 定期检疫
- 病畜淘汰
- 健康畜群培育

**（二）病畜处理**

- 病畜隔离
- 淘汰病畜
- 无害化处理

### 7.2 切断传播途径

**（一）个人防护**

| 防护用品 | 用途 |
|----------|------|
| 防护手套 | 接触动物及产品 |
| 防护服 | 作业时穿戴 |
| 口罩 | 防止吸入 |
| 防护眼镜 | 防止溅入 |

**（二）作业防护**

- 作业区与生活区分开
- 定期消毒
- 保持清洁

**（三）食品卫生**

- 不食生奶
- 不食未熟肉
- 巴氏消毒奶制品

### 7.3 卫生保健措施

**（一）就业前体检**

- 排除禁忌证
- 免疫系统疾病
- 严重慢性病

**（二）在岗期间体检**

- 每年一次
- 血清学检查
- 临床检查

**（三）健康教育**

- 布鲁氏菌病知识
- 防护知识
- 早期症状识别

### 7.4 免疫预防

- 动物疫苗接种
- 高危人群疫苗接种（可选）
- 疫苗接种注意事项

## 八、职业防护要点

### 8.1 兽医

1. 接生时戴手套
2. 诊疗后洗手消毒
3. 医疗器械消毒
4. 定期体检

### 8.2 畜牧工人

1. 接触病畜时防护
2. 不直接接触流产胎儿
3. 作业后消毒
4. 定期体检

### 8.3 屠宰工人

1. 穿戴防护用品
2. 工具消毒
3. 作业后清洁
4. 定期体检

### 8.4 实验室人员

1. 生物安全柜操作
2. 废弃物高压灭菌
3. 人员培训
4. 定期体检

## 九、法律保护与权益

### 9.1 职业病诊断

- 诊断机构：省级职业病诊断机构
- 诊断依据：GBZ 227-2016
- 需要材料：职业史、临床资料、实验室检查

### 9.2 工伤认定

- 属于职业病
- 认定为工伤
- 享受工伤保险待遇

### 9.3 用人单位责任

- 提供防护设施
- 组织健康检查
- 进行安全培训
- 落实防护措施

### 9.4 劳动者权利

- 知情权
- 拒绝违章权
- 检举权
- 依法索赔权

## 十、总结

职业性布鲁氏菌病是完全可以预防的职业病。

**核心要点：**

1. 人畜共患传染病
2. 羊种菌致病性最强
3. 职业人群易感染
4. 治疗需联合用药
5. 预防重于治疗
6. 慢性期治疗困难

**防护要点：**

- 做好个人防护
- 遵守操作规程
- 注意食品卫生
- 定期健康检查
- 出现症状及时就医

让我们共同关注布鲁氏菌病防护，守护劳动者健康！
