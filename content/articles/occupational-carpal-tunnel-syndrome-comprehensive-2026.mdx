# 职业性腕管综合征：现代办公族的职业病

## 引言

**职业性腕管综合征**（Occupational Carpal Tunnel Syndrome，CTS）是指劳动者在职业活动中因长期、反复用手部进行精细操作而导致正中神经在腕管内受压所引起的疾病。这是一种常见的职业性肌肉骨骼疾病，在办公室工作人群、制造业工人、医护人员等群体中发病率较高。

随着电脑的普及和工作方式的改变，腕管综合征的发病率逐年上升，已成为一种典型的"现代职业病"。这种疾病不仅影响劳动者的工作效率，严重者还可导致手部功能永久性损害，需要引起高度重视。

本文将从专业角度全面介绍职业性腕管综合征的相关知识，帮助广大劳动者认识这一常见职业健康问题。

## 一、解剖学基础

### 1.1 腕管结构

**（一）腕管的组成**

| 结构 | 位置 | 作用 |
|------|------|------|
| 腕骨 | 腕管底部 | 构成骨性底壁 |
| 腕横韧带 | 腕管顶部 | 形成纤维性顶壁 |
| 肌腱 | 腕管内 | 屈肌腱通过 |
| 正中神经 | 腕管内 | 支配手部感觉运动 |

**（二）内容物**

- 正中神经
- 屈肌腱（共9条）
- 腱鞘滑膜

### 1.2 正中神经功能

**（一）运动功能**

- 拇指对掌
- 拇指外展
- 食指、中指屈曲

**（二）感觉功能**

- 拇指
- 食指
- 中指
- 环指桡侧半

## 二、病因与发病机制

### 2.1 病因

**（一）职业因素**

| 职业 | 风险因素 |
|------|----------|
| 办公室工作者 | 键盘操作、鼠标使用 |
| 制造业工人 | 重复性手部动作 |
| 医护人员 | 注射器操作 |
| 收银员 | 键盘操作 |
| 流水线工人 | 重复性装配 |
| 裁缝 | 穿针引线 |

**（二）工作方式因素**

- 长时间保持手腕屈曲
- 反复屈伸手腕
- 用力握持工具
- 振动暴露

**（三）其他因素**

- 性别（女性多发）
- 妊娠
- 糖尿病
- 甲状腺功能异常

### 2.2 发病机制

**（一）机械性压迫**

- 腕管内压力升高
- 正中神经受压
- 神经缺血缺氧

**（二）慢性劳损**

- 反复摩擦
- 肌腱肿胀
- 腱鞘炎

**（三）病理变化**

- 神经外膜增厚
- 髓鞘变性
- 轴突萎缩

### 2.3 影响发病的因素

| 因素 | 影响 |
|------|------|
| 重复性手部活动 | 增加发病风险 |
| 腕部姿势 | 屈曲位增加压力 |
| 用力程度 | 用力越大，风险越高 |
| 振动 | 加重神经损伤 |
| 时间 | 长期从事易发病 |

## 三、临床表现

### 3.1 症状

**（一）感觉症状**

| 症状 | 特点 |
|------|------|
| 麻木 | 手指尖、手掌麻木 |
| 刺痛 | 夜间或清晨加重 |
| 灼痛 | 手指感觉异常 |
| 感觉减退 | 严重时出现 |

**（二）运动症状**

- 拇指无力
- 精细动作障碍
- 握力下降
- 写字、用筷困难

**（三）疼痛症状**

- 腕部疼痛
- 前臂疼痛
- 夜间疼痛
- 疼痛可放射至上臂

### 3.2 体征

**（一）感觉检查**

- 痛觉减退
- 触觉减退
- 两点辨别觉减退

**（二）运动检查**

- 拇指对掌力减弱
- 大鱼际肌萎缩
- 肌力下降

**（三）特殊检查**

| 检查 | 方法 | 阳性表现 |
|------|------|----------|
| Tinel征 | 叩击腕部 | 手指麻木 |
| Phalen征 | 屈腕60秒 | 麻木加重 |
| 止血带试验 | 充气后麻木 | 出现麻木 |

### 3.3 临床分级

| 分级 | 表现 |
|------|------|
| 轻度 | 间歇性麻木，夜间加重 |
| 中度 | 持续麻木，日常活动受限 |
| 重度 | 肌肉萎缩，感觉丧失 |

## 四、诊断与鉴别诊断

### 4.1 诊断标准

**诊断依据：**

1. 职业史：长期反复手部活动
2. 临床表现：麻木、刺痛、无力
3. 体征：Tinel征、Phalen征阳性
4. 辅助检查：电生理检查异常

### 4.2 电生理检查

**（一）肌电图**

- 正中神经传导速度减慢
- 潜伏期延长
- 肌肉纤颤

**（二）神经传导速度**

| 指标 | 异常表现 |
|------|----------|
| 运动神经传导 | 潜伏期延长 |
| 感觉神经传导 | 传导速度减慢 |

### 4.3 鉴别诊断

| 疾病 | 鉴别要点 |
|------|----------|
| 颈椎病 | 颈椎影像学异常 |
| 胸出口综合征 | 臂丛神经受压 |
| 旋前圆肌综合征 | 前臂压痛 |
| 末梢神经炎 | 对称性、多因素 |

## 五、治疗方案

### 5.1 治疗原则

1. **早期治疗**：防止进展
2. **综合治疗**：多种方法结合
3. **去除病因**：改变工作方式
4. **功能恢复**：康复锻炼

### 5.2 非手术治疗

**（一）一般治疗**

| 方法 | 作用 |
|------|------|
| 休息 | 减少进一步损伤 |
| 冰敷 | 减轻肿胀 |
| 抬高 | 促进静脉回流 |

**（二）支具治疗**

- 夜间中立位支具
- 白天工作时使用
- 持续3-6个月

**（三）药物治疗**

| 药物 | 作用 | 用法 |
|------|------|------|
| 非甾体抗炎药 | 消炎止痛 | 口服、外用 |
| 维生素B族 | 营养神经 | 口服 |
| 糖皮质激素 | 抗炎消肿 | 局部注射 |

**（四）物理治疗**

- 超声波
- 电刺激
- 激光治疗
- 蜡疗

**（五）运动疗法**

- 腕部伸展运动
- 手指活动训练
- 肌力锻炼

### 5.3 手术治疗

**（一）适应证**

- 保守治疗无效
- 肌肉萎缩
- 严重功能障碍

**（二）手术方法**

- 腕横韧带切开减压
- 内镜下手术
- 开放手术

**（三）术后康复**

- 早期活动
- 逐步功能锻炼
- 避免过度使用

### 5.4 预后

- **早期治疗**：预后良好
- **中晚期**：可能遗留功能障碍
- **关键**：早发现、早治疗

## 六、预防措施

### 6.1 工作场所设计

**（一）工作站设计**

- 键盘高度调节
- 鼠标位置合理
- 显示器高度适宜
- 光线充足

**（二）工具选择**

- 符合人体工程学
- 重量适中
- 减少振动

### 6.2 个人防护

**（一）工作姿势**

- 保持手腕中立位
- 避免长时间屈曲
- 定时休息
- 活动手腕

**（二）工间休息**

| 方法 | 时间 |
|------|------|
| 短暂休息 | 每小时5-10分钟 |
| 手部活动 | 伸展、屈曲 |
| 全身活动 | 走动、伸展 |

### 6.3 运动锻炼

**（一）预防性锻炼**

- 手腕伸展运动
- 手指屈伸运动
- 握力训练

**（二）工间操**

- 3-5分钟
- 每天数次
- 简单易行

### 6.4 健康教育

1. 正确使用键盘鼠标
2. 识别早期症状
3. 及时就医意识
4. 预防知识普及

## 七、职业康复

### 7.1 康复目标

1. 缓解症状
2. 恢复功能
3. 预防复发
4. 重返工作

### 7.2 康复方案

**（一）急性期**

- 休息
- 支具固定
- 冰敷
- 消炎止痛

**（二）恢复期**

- 物理治疗
- 运动疗法
- 作业治疗

**（三）维持期**

- 功能锻炼
- 姿势矫正
- 工作适应

### 7.3 重返工作

1. 评估工作能力
2. 调整工作岗位（必要时）
3. 逐步增加工作量
4. 长期随访

## 八、法律保护与权益

### 8.1 职业病认定

- 严重腕管综合征可认定为职业病
- 需符合相关诊断标准
- 享受工伤待遇

### 8.2 用人单位责任

- 提供符合人机工程学的工作环境
- 合理安排工作时间
- 组织工间休息
- 落实劳动保护

### 8.3 劳动者权利

- 知情权
- 改善工作条件权
- 拒绝违章权
- 依法索赔权

## 九、总结

职业性腕管综合征是完全可以预防的职业病。

**核心要点：**

1. 反复手部活动是主要病因
2. 早期症状以麻木、刺痛为主
3. 早发现、早治疗是关键
4. 预防重于治疗
5. 改变工作方式是根本

**预防要点：**

- 保持手腕中立位
- 避免长时间屈曲
- 定时休息活动
- 加强手部锻炼
- 出现症状及时就医

让我们共同关注腕管综合征防护，守护劳动者手部健康！
