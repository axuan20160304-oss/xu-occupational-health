# 职业性噪声聋预防与治疗实务指南

## 一、噪声的基本知识

### 1.1 什么是噪声

噪声是指生产过程中产生的一切声音。

**噪声强度单位**：分贝(dB)

### 1.2 噪声的危害

| 危害类型 | 具体表现 |
|----------|----------|
| 听力损伤 | 暂时性/永久性听力下降 |
| 耳鸣 | 耳内持续性蝉鸣声 |
| 神经系统 | 头痛、头晕、失眠 |
| 心血管 | 血压升高、心率加快 |

### 1.3 高危行业

- 机械制造
- 矿山开采
- 建筑施工
- 纺织印染
- 金属加工

## 二、噪声聋的诊断

### 2.1 诊断标准

**诊断原则：**
1. 职业接触史（≥85dB，3年以上）
2. 纯音听阈测试异常
3. 排除其他原因

### 2.2 诊断分期

| 分期 | 标准 |
|------|------|
| 观察对象 | 听力下降，未达标准 |
| 轻度 | 26-40dB |
| 中度 | 41-55dB |
| 重度 | ≥56dB |

### 2.3 临床表现

- 进行性听力下降
- 双侧对称性
- 初期耳鸣
- 远离噪声后症状持续

## 三、噪声聋的预防

### 3.1 企业措施

#### 工程控制
- 隔声：安装隔声墙
- 消声：安装消声器
- 吸声：使用吸声材料

#### 替代方案
- 用低噪声设备
- 自动化改造

### 3.2 个人防护

| 防护用品 | 防护效果 |
|----------|----------|
| 耳塞 | 衰减15-25dB |
| 耳罩 | 衰减20-35dB |

### 3.3 健康监护

| 检查类别 | 周期 |
|----------|------|
| 上岗前 | 入职前 |
| 在岗期间 | 每年1次 |
| 离岗时 | 离职前 |

## 四、噪声聋的治疗

### 4.1 治疗原则

```
脱离噪声 → 营养神经 → 高压氧 → 康复
```

### 4.2 治疗方法

#### 药物治疗
- 维生素B族
- 银杏叶提取物
- 血管扩张剂

#### 物理治疗
- 高压氧舱
- 声治疗

### 4.3 康复

- 佩戴助听器
- 听力训练
- 心理疏导

## 五、案例警示

### 5.1 典型案例

某机械厂冲压工：
- 工龄：10年
- 症状：双耳听力明显下降
- 诊断：职业性噪声聋（中度）

### 5.2 教训

- 未正确佩戴耳塞
- 车间噪声超标未治理
