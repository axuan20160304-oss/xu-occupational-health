# 职业性乙酸甲酯中毒的防护与应急处置

## 概述

乙酸甲酯（Methyl Acetate，CH3COOCH3）是一种常用的有机溶剂，广泛应用于油漆、涂料、胶粘剂、合成纤维等化工行业。在职业生产环境中，如果防护措施不当，劳动者可能通过呼吸道吸入或皮肤接触的方式接触到乙酸甲酯蒸气或液体，导致职业性中毒。

## 乙酸甲酯的理化性质

### 基本特性

| 项目 | 参数 |
|------|------|
| 化学式 | C3H6O2 |
| 分子量 | 74.08 g/mol |
| 外观 | 无色透明液体 |
| 气味 | 略有芳香味 |
| 闪点 | -10°C（闭杯） |
| 沸点 | 57-58°C |
| 爆炸极限 | 3.1%-16%（体积分数） |
| 溶解性 | 难溶于水，易溶于醇、醚等有机溶剂 |

### 主要用途

- 油漆和涂料溶剂
- 胶粘剂稀释剂
- 合成纤维纺丝溶剂
- 塑料和橡胶制品生产
- 电子行业清洗剂

## 健康危害

### 急性中毒症状

高浓度乙酸甲酯接触可引起急性中毒，主要表现为：

1. **中枢神经系统症状**：
   - 头痛、头晕
   - 恶心、呕吐
   - 视力模糊、流泪
   - 严重者可出现嗜睡、昏迷

2. **呼吸道刺激**：
   - 咳嗽、咽干
   - 胸闷、气促
   - 支气管炎表现

3. **皮肤和眼睛刺激**：
   - 皮肤干燥、脱脂
   - 眼睛刺痛、流泪
   - 长时间接触可引起皮炎

### 慢性影响

长期低浓度接触可能引起：

- 慢性神经系统功能紊乱
- 肝脏解毒功能下降
- 视力持续性下降
- 皮肤慢性刺激性皮炎

## 职业接触限值

根据《GBZ 2.1-2019 工作场所有害因素职业接触限值 第1部分：化学有害因素》：

| 项目 | PC-TWA | PC-STEL |
|------|--------|---------|
| 乙酸甲酯 | 200 mg/m³ | 500 mg/m³ |

**说明**：
- PC-TWA：时间加权平均浓度
- PC-STEL：短时间接触浓度

## 职业健康监护

### 体检要求

根据《GBZ 188-2014 职业健康监护技术规范》：

1. **岗前体检**：
   - 职业禁忌证筛查
   - 基础健康状况评估

2. **在岗体检**：
   - 每年1次
   - 重点检查神经系统、肝脏功能、视力

3. **离岗体检**：
   - 接触终止时进行
   - 全面评估健康状况

### 职业禁忌证

以下人员不宜从事乙酸甲酯作业：

- 神经系统器质性疾病
- 严重肝脏疾病
- 视力明显异常者
- 皮肤慢性炎症性疾病
- 过敏性体质者

## 工程控制措施

### 通风防护

1. **全面通风**：
   - 车间保持良好通风
   - 换气次数≥12次/小时

2. **局部通风**：
   - 安装局部排风罩
   - 通风设备应设在污染源附近

3. **通风系统维护**：
   - 定期检测通风效果
   - 保持通风设施正常运行

### 监测预警

- 安装乙酸甲酯浓度检测仪
- 设置报警阈值（≥100 mg/m³）
- 定期进行环境监测

## 个人防护用品

### 呼吸防护

| 作业环境浓度 | 防护用品 |
|-------------|---------|
| ≤200 mg/m³ | 防毒面具（活性炭滤盒） |
| >200 mg/m³ | 正压式空气呼吸器 |

### 皮肤防护

- 佩戴防护手套（丁腈或氯丁橡胶材质）
- 穿着防护服
- 涂抹护肤霜

### 眼睛防护

- 佩戴防护眼镜
- 接触液体时佩戴防护面罩

## 应急处置

### 吸入中毒应急处理

1. 迅速脱离现场至空气新鲜处
2. 保持呼吸道通畅
3. 给予氧气吸入
4. 及时就医

### 皮肤接触应急处理

1. 立即脱去污染衣物
2. 用大量清水冲洗（≥15分钟）
3. 使用温和的清洁剂清洗
4. 必要时就医

### 眼睛接触应急处理

1. 立即用流动清水冲洗（≥15分钟）
2. 翻开眼睑彻底清洗
3. 尽快就医

### 食入应急处理

1. 立即催吐（意识清醒时）
2. 饮温水或牛奶
3. 立即就医

## 管理和培训

1. **制度建设**：
   - 制定乙酸甲酯职业卫生操作规程
   - 建立应急救援预案

2. **培训教育**：
   - 上岗前培训
   - 定期职业卫生培训

3. **警示标识**：
   - 车间设置明显警示标志
   - 告知卡上墙

4. **健康档案**：
   - 建立劳动者健康监护档案
   - 保存职业病诊断资料

## 结语

职业性乙酸甲酯中毒是完全可以预防的。通过加强工程控制、落实个人防护、完善健康监护和提高安全意识，可以有效降低乙酸甲酯职业中毒的发生率，保障劳动者的身体健康和企业安全生产。

企业应严格遵守国家职业卫生标准，持续改进防护措施，为劳动者创造安全健康的工作环境。
