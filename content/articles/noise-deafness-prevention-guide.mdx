## 一、噪声聋概述

### 1.1 什么是职业性噪声聋

职业性噪声聋是指长期在超过85dB(A)的噪声环境下工作，内耳毛细胞受损，导致永久性听力损失的职业病。

### 1.2 症状表现

- 耳鸣：持续性或间歇性耳内鸣响
- 听力下降：从高频开始，逐渐影响日常交流
- 耳闷：耳朵发闷、堵塞感
- 头晕：严重时可能出现眩晕

### 1.3 危害

| 危害类型 | 影响 |
|----------|------|
| 听力损伤 | 永久性，不可逆 |
| 交流障碍 | 影响工作和生活 |
| 心理健康 | 焦虑、烦躁 |
| 安全隐患 | 听觉警报无法识别 |

## 二、噪声作业行业

### 2.1 主要行业

| 行业 | 噪声源 | 典型声级 |
|------|--------|----------|
| 机械制造 | 冲压、锻造、切割 | 85-110dB |
| 建筑施工 | 打桩、混凝土振捣 | 90-115dB |
| 交通运输 | 机场地勤、铁路 | 80-100dB |
| 电子制造 | SMT生产线 | 75-90dB |
| 木材加工 | 锯木、刨削 | 85-110dB |
| 纺织 | 织布机 | 80-95dB |
| 矿山 | 凿岩、爆破 | 95-120dB |

### 2.2 噪声类型

| 类型 | 特点 |
|------|------|
| 稳态噪声 | 强度基本不变 |
| 脉冲噪声 | 突然爆发，峰值很高 |
| 间歇噪声 | 时有时无 |

## 三、预防措施

### 3.1 工程控制措施

#### 3.1.1 隔声

| 措施 | 适用场景 |
|------|----------|
| 隔声墙 | 车间分隔 |
| 隔声罩 | 设备包裹 |
| 隔声门 | 噪声房间 |

#### 3.1.2 消声

| 措施 | 适用场景 |
|------|----------|
| 消声器 | 气流噪声 |
| 声屏障 | 户外噪声 |
| 吸声材料 | 车间内壁 |

#### 3.1.3 减振

| 措施 | 适用场景 |
|------|----------|
| 减振垫 | 设备基础 |
| 隔振沟 | 振动传播 |
| 弹性支撑 | 管道振动 |

#### 3.1.4 合理布局

- 噪声源远离工作区域
- 高噪声设备集中管理
- 自动化替代人工操作

### 3.2 管理措施

#### 3.2.1 危害告知

- 设置警示标识
- 告知噪声危害
- 培训防护知识

#### 3.2.2 作业时间控制

| 噪声级[dB(A)] | 最大接触时间 |
|-------------|--------------|
| 85 | 8小时 |
| 88 | 4小时 |
| 91 | 2小时 |
| 94 | 1小时 |
| 97 | 30分钟 |

#### 3.2.3 轮换作业

- 减少个人暴露时间
- 定期轮换岗位
- 限制在高噪声区停留

### 3.3 个人防护

#### 3.3.1 防噪声耳塞

| 类型 | SNR值 | 适用 |
|------|-------|------|
| 泡沫耳塞 | 25-35dB | 一般噪声 |
| 硅胶耳塞 | 20-30dB | 舒适型 |
| 个性化耳塞 | 30-40dB | 特殊作业 |

#### 3.3.2 防噪声耳罩

| 类型 | NRR值 | 适用 |
|------|-------|------|
| 耳罩式 | 25-35dB | 极高噪声 |
| 带帽耳罩 | 20-30dB | 兼顾头部防护 |

#### 3.3.3 耳塞+耳罩组合

- 组合使用可达40dB以上防护
- 适用于极端噪声环境

## 四、听力保护计划

### 4.1 行动计划

当噪声≥80dB(A)时启动：

1. 危害告知
2. 配发防护用品
3. 健康监护

### 4.2 防护计划

当噪声≥85dB(A)时：

1. 工程控制
2. 严格使用防护用品
3. 减少暴露时间

### 4.3 应急计划

当噪声≥100dB(A)时：

1. 立即停止作业
2. 撤离人员
3. 采取工程措施

## 五、职业健康监护

### 5.1 上岗前检查

#### 5.1.1 必检项目

- 纯音测听（500-6000Hz）
- 耳科检查
- 症状询问

#### 5.1.2 禁忌证

| 疾病 | 说明 |
|------|------|
| 永久性感音神经性听力损失 | 不宜从事噪声作业 |
| 中度以上传导性耳聋 | 不宜从事噪声作业 |
| 眩晕综合征 | 不宜从事噪声作业 |

### 5.2 在岗检查

#### 5.2.1 检查周期

| 噪声级 | 周期 |
|--------|------|
| ≥85dB(A) | 每年1次 |
| 80-85dB(A) | 每2年1次 |

#### 5.2.2 检查项目

- 纯音测听
- 耳科检查
- 症状询问

### 5.3 离岗检查

- 离岗时进行纯音测听
- 评估听力损失

---

## 六、急救处理

### 6.1 急性听力损伤

1. 立即脱离噪声环境
2. 安静休息
3. 及时就医

### 6.2 耳鸣处理

1. 避免进一步噪声暴露
2. 放松休息
3. 就医检查

## 八、总结

噪声聋预防要点：

1. **工程控制**是根本
2. **个人防护**是最后一道防线
3. **健康监护**早发现、早处理
4. **依法维权**保护自身权益
