# 职业性白内障的诊断标准与手术治疗

## 概述

职业性白内障（Occupational Cataract）是指劳动者在职业活动中因接触物理因素（如电离辐射、非电离辐射、微波、红外线、紫外线等）或化学因素（如苯、酚、硝基苯等）引起的晶状体混浊。《职业病分类和目录》将职业性白内障列为法定职业病，是常见的职业性眼病之一。

根据致病因素，职业性白内障可分为：

- **电离辐射性白内障**：X射线、γ射线、中子等电离辐射引起
- **非电离辐射性白内障**：微波、红外线、紫外线等辐射引起
- **化学毒物性白内障**：苯、酚、硝基苯、三硝基甲苯等化学物质引起
- **热损伤性白内障**：高温作业环境下长期暴露引起

## 诊断标准

### 诊断原则

根据《GBZ 35-2010 职业性白内障的诊断》，诊断需满足以下条件：

1. **明确的职业接触史**：有确切的职业性致白内障因素接触史
2. **晶状体改变**：裂隙灯显微镜下可见晶状体混浊
3. **符合临床表现**：出现进行性视力下降、视物模糊、屈光改变等症状
4. **排除其他因素**：排除年龄相关性白内障（老年性白内障）、先天性白内障、外伤性白内障等

### 诊断依据

#### 一、职业接触史调查

| 致病因素 | 接触行业 | 典型工种 |
|----------|----------|----------|
| 电离辐射 | 放射科、核工业、射线探伤 | 放射科医师、核电站操作工 |
| 微波 | 通信、电子、雷达 | 雷达操作员、微波设备维修工 |
| 红外线 | 玻璃加工、冶金 | 熔炉工、玻璃吹制工 |
| 紫外线 | 电焊、照排、塑料加工 | 电焊工、照排操作工 |
| 化学毒物 | 化工、染料、制药 | 苯作业工人、制药工 |

#### 二、临床表现

- **视力下降**：进行性视力减退，屈光不正加重
- **视物模糊**：视物模糊不清，有雾状感
- **单眼复视或多视**：晶状体不均匀混浊导致
- **眩光**：在强光下视力明显下降
- **色觉改变**：部分患者出现色觉异常

#### 三、眼科检查

1. **视力检查**：远视力、近视力
2. **裂隙灯显微镜检查**：观察晶状体混浊部位、形态和程度
3. **眼底检查**：排除视网膜病变
4. **眼压测量**：排除青光眼
5. **眼部超声检查**：评估眼内情况

### 诊断分期

| 分期 | 晶状体改变 | 视力影响 |
|------|------------|----------|
| 早期 | 晶状体周边部点状或片状混浊 | 视力轻度下降或正常 |
| 中期 | 晶状体皮质楔形混浊或核混浊 | 视力中度下降 |
| 晚期 | 晶状体完全混浊 | 视力严重下降或失明 |

## 鉴别诊断

### 需要鉴别的疾病

1. **年龄相关性白内障（老年性白内障）**：通常50岁以上发病，进展缓慢
2. **先天性白内障**：出生时或婴幼儿时期发病
3. **外伤性白内障**：有明确眼部外伤史
4. **代谢性白内障**：糖尿病、甲亢等全身疾病引起
5. **药物性白内障**：长期使用糖皮质激素等药物引起

### 鉴别要点

| 疾病 | 关键鉴别特征 |
|------|--------------|
| 老年性白内障 | 发病年龄晚，无职业接触史 |
| 先天性白内障 | 发病早，家族史 |
| 外伤性白内障 | 明确外伤史，局限性混浊 |
| 糖尿病性白内障 | 糖尿病病史，血糖控制不良 |

## 手术治疗

### 手术适应症

1. 晶状体混浊导致视力下降到0.3以下
2. 晶状体膨胀引起青光眼
3. 晶状体脱位或半脱位
4. 患者有强烈的复明愿望

### 手术方法

#### 一、白内障超声乳化吸除术

是目前最常用的手术方法：

- **优点**：切口小、恢复快、并发症少
- **步骤**：角膜切口 → 连续环形撕囊 → 超声乳化吸除 → 人工晶体植入
- **适用**：各类型白内障

#### 二、白内障囊外摘除术

适用于硬核白内障：

- **优点**：可取出较大晶状体核
- **缺点**：切口较大，恢复较慢

#### 三、人工晶体植入

- **单焦点人工晶体**：适合远视力或近视力矫正
- **多焦点人工晶体**：可同时矫正远近视力和中视力
- **Toric人工晶体**：矫正角膜散光

### 术前检查

1. 全身检查：血常规、血糖、心电图等
2. 眼部检查：角膜内皮计数、眼轴长度测量、角膜曲率等
3. 人工晶体度数计算

### 术后处理

1. 按时使用抗生素和糖皮质激素滴眼液
2. 避免眼部外伤和感染
3. 定期复查视力恢复情况
4. 术后3个月验光配镜

## 劳动能力鉴定

### 鉴定标准

根据《劳动能力鉴定 职工工伤与职业病致残等级》（GB/T 16180），职业性白内障的劳动能力鉴定分级如下：

| 伤残等级 | 鉴定标准 |
|----------|----------|
| 四级 | 职业性白内障手术后，双眼视力均<0.1 |
| 五级 | 职业性白内障手术后，单眼视力<0.05或双眼视力<0.2 |
| 六级 | 职业性白内障手术后，单眼视力<0.3或双眼视力<0.4 |
| 七级 | 职业性白内障手术后，视力有一定程度下降 |

### 鉴定要点

- 手术治疗效果
- 矫正视力水平
- 双眼视力差异

## 职业病待遇

### 工伤保险待遇

- **医疗费用**：手术费用由工伤保险基金支付
- **停工留薪期**：住院期间原工资福利待遇不变
- **伤残津贴**：根据伤残等级按月发放
- **一次性伤残补助金**：根据伤残等级一次性支付

## 预防措施

### 工程控制

- 电离辐射工作场所：设置防护屏蔽、距离防护、时间防护
- 红外线/紫外线作业：使用防护玻璃或隔热屏障
- 化学毒物作业：通风排毒、湿式作业

### 个人防护

- 佩戴防护眼镜：防辐射眼镜、防护面罩
- 防护眼镜定期更换：确保防护效果
- 作业后清洁面部：避免化学物质残留

### 健康监护

- 上岗前体检：筛查眼部疾病
- 定期在岗体检：每年1次眼科检查
- 早发现、早诊断、早治疗

## 总结

职业性白内障的诊断需要综合职业接触史和晶状体检查结果。手术治疗是目前有效的治疗方法，预后良好。早期诊断和规范治疗可最大程度恢复视力功能。用人单位应加强职业防护措施，预防职业性白内障的发生。
