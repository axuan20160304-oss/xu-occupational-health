## 一、正己烷的危害有多大

### 1.1 什么是正己烷

正己烷是一种有机溶剂，常用于：
- 电子行业清洗PCB板
- 印刷行业油墨稀释
- 制鞋行业粘合鞋底
- 橡胶、塑料加工

**特性**：易挥发、脂溶性强、蓄积毒性

### 1.2 对人体的危害

| 危害类型 | 具体表现 |
|----------|----------|
| 神经系统 | 周围神经病变、肌肉萎缩 |
| 长期影响 | 运动障碍、瘫痪 |
| 不可逆 | 神经损伤难以恢复 |

### 1.3 高危行业

| 行业 | 风险岗位 | 暴露风险 |
|------|----------|----------|
| 电子厂 | 清洗工 | 高 |
| 印刷厂 | 调墨工 | 高 |
| 制鞋厂 | 粘胶工 | 极高 |
| 箱包厂 | 贴合工 | 高 |

## 二、如何预防正己烷中毒

### 2.1 企业应该做什么

#### 1. 替代工艺（首选）
```
用低毒溶剂替代正己烷
推荐：乙醇、丙酮、异丙醇
```

#### 2. 工程控制
- **局部通风**：通风罩 + 活性炭吸附
- **全通风**：每小时换气6-12次
- **隔离操作**：自动化设备远程控制

#### 3. 个人防护
| 防护用品 | 防护部位 |
|----------|----------|
| 防毒面具 | 呼吸道 |
| 防护手套 | 皮肤 |
| 防护服 | 身体 |

#### 4. 健康监护
- **上岗前**：排除禁忌症
- **在岗期间**：每年体检，重点查神经肌电图
- **离岗时**：健康评估

### 2.2 员工应该怎么做

1. **严格防护**
   - 上班必须戴防毒面具
   - 禁止徒手接触正己烷
   - 严禁车间内进食

2. **卫生习惯**
   - 班后立即沐浴
   - 换掉工作服再回家
   - 单独存放工作服

3. **身体警觉**
   - 如有四肢麻木、无力，立即报告
   - 早期发现，早期治疗

### 2.3 工作场所标准

| 项目 | 标准值 |
|------|--------|
| 空气中浓度 | <100 mg/m³ (PC-TWA) |
| 监测频次 | 每月1次 |
| 体检频次 | 每年1次 |

## 三、正己烷中毒的治疗

### 3.1 治疗原则

```
脱离接触 → 药物治疗 → 康复训练 → 定期随访
```

### 3.2 药物治疗

#### 营养神经药物

| 药物 | 剂量 | 用法 |
|------|------|------|
| 维生素B1 | 10-20mg | 口服，每日3次 |
| 维生素B6 | 10-20mg | 口服，每日3次 |
| 维生素B12 | 500μg | 肌注，隔日1次 |
| 甲钴胺 | 500μg | 口服/肌注，每日1次 |

#### 辅助治疗

| 治疗方式 | 作用 |
|----------|------|
| 针灸 | 改善循环，促进神经修复 |
| 按摩 | 防止肌肉萎缩 |
| 运动训练 | 维持关节活动度 |
| 物理治疗 | 电刺激、蜡疗等 |

### 3.3 分期治疗

#### 轻度中毒
- 脱离接触
- 维生素B族治疗
- 门诊随访

#### 中度中毒
- 住院治疗
- 综合性康复
- 持续随访

#### 重度中毒
- 专科治疗
- 长期康复
- 职业评定

### 3.4 康复要点

1. **坚持服药**：维生素B族需长期服用
2. **循序渐进**：康复训练不能操之过急
3. **营养均衡**：高蛋白饮食有助于修复
4. **定期复查**：每3-6个月复查神经肌电图

## 四、案例与警示

### 4.1 典型案例

**案例：某电子厂清洗工**
- 工龄：3年
- 症状：四肢麻木、肌肉萎缩
- 诊断：职业性正己烷中毒（重度）
- 教训：未做好个人防护

### 4.2 警示

| 错误做法 | 后果 |
|----------|------|
| 不戴防毒面具 | 吸入过量正己烷 |
| 徒手接触溶剂 | 皮肤吸收中毒 |
| 车间内吃饭 | 摄入正己烷 |
| 身体不适不及时就医 | 病情加重 |
