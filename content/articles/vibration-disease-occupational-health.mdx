# 职业性振动病（手臂振动病）：诊断、预防与处理指南

振动病是因长期接触生产性振动引发的职业性疾病，我国将其列为法定职业病，分为手传振动和全身振动两种类型。手传振动主要引发末梢循环障碍及神经功能障碍，典型症状为振动性白指（Raynaud's phenomenon）。本文依据GBZ 7-2014《职业性手臂振动病的诊断》等国家标准，系统阐述其危害特点、诊断标准及防控措施。

## 一、适用范围

本标准（GBZ 7-2014《职业性手臂振动病的诊断》）适用于：

- **凿岩作业**：凿岩工、掘进工
- **金属加工**：砂轮磨光工、抛光工、切割工
- **木材加工**：电锯操作工、木工
- **建筑施工**：混凝土振捣工、风钻工
- **矿山作业**：采矿工、爆破工
- **机械制造**：锻压工、冲压工
- **交通运输**：驾驶员（长期驾驶振动机械）
- **农林作业**：拖拉机手、收割机操作手

> **注意**：本标准主要适用于手臂振动病，全身振动病可参照执行。

## 二、毒性特点

### 2.1 振动参数

| 参数 | 描述 | 对健康的影响 |
|------|------|--------------|
| **频率** | 振动每秒完成的次数（Hz） | 低频（<10Hz）主要影响内脏；高频（>20Hz）主要影响手臂 |
| **强度** | 振动加速度（m/s²） | 强度越大，危害越严重 |
| **方向** | 振动传递方向 | 多方向振动危害更大 |
| **暴露时间** | 每日接触振动的时间 | 时间越长，危害越严重 |

### 2.2 发病机制

1. **血管痉挛**：振动引起手指血管收缩痉挛，导致末梢循环障碍
2. **神经损伤**：长期振动导致手臂神经脱髓鞘改变
3. **肌肉骨骼损伤**：引起肌肉劳损、骨关节退行性改变
4. **局部效应**：振动导致组织毛细血管损伤、变性
5. **全身效应**：长期暴露可影响心血管系统、内分泌系统

### 2.3 临床表现

**手传振动病（手臂振动病）典型症状**：

| 系统 | 症状 |
|------|------|
| **末梢循环障碍** | 手指发凉、发白、发绀，寒冷或情绪激动时加重（振动性白指） |
| **感觉障碍** | 手指麻木、疼痛、感觉减退、感觉过敏 |
| **运动障碍** | 手指无力、精细动作障碍、握力下降 |
| **骨关节损伤** | 指关节、腕关节疼痛、变形、活动受限 |
| **肌肉损伤** | 前臂肌肉萎缩、肌力下降 |

**全身振动影响**：

| 系统 | 症状 |
|------|------|
| **神经系统** | 头晕、眩晕、恶心、视力模糊 |
| **心血管** | 血压变化、心率加快 |
| **消化系统** | 食欲减退、消化不良 |
| **脊柱** | 腰背痛、椎间盘突出 |

## 三、健康检查

### 3.1 上岗前体检

**必检项目**：
- 常规体检（内科、外科）
- 神经系统检查
- 末梢循环功能检查
- 骨关节检查

**职业禁忌证**：
- 周围神经系统疾病
- 雷诺病或雷诺现象
- 严重的心血管疾病
- 手指或手腕骨关节疾病
- 明显的神经症

### 3.2 在岗期间体检

**体检周期**：
- 接触振动强度合格：每年1次
- 接触振动强度超标：每半年1次
- 出现症状者：缩短体检间隔

**体检内容**：
- 详细病史询问（手指发白、麻木、疼痛等）
- 神经系统检查（感觉、运动）
- 末梢循环功能检查（冷水复温试验）
- 神经-肌电图检查
- 骨关节X线检查（必要时）

### 3.3 应急健康检查

接触强振动后，应进行：
- 神经系统检查
- 末梢循环功能评估
- 必要时进行肌电图检查

## 四、诊断标准

### 4.1 诊断原则

依据GBZ 7-2014，职业性手臂振动病的诊断需根据：

1. **一年以上连续从事手传振动作业的职业史**
2. **手部末梢循环障碍、手臂神经功能障碍和（或）骨关节肌肉损伤为主的临床表现**
3. **末梢循环功能、神经-肌电图检查结果**
4. **参考作业环境的职业卫生学资料**
5. **排除其他病因所致类似疾病**

### 4.2 诊断分级

| 分级 | 诊断条件 |
|------|----------|
| **观察对象** | 从事振动作业期间出现手部疼痛、麻木、发凉、僵硬等异常感觉，但未达到诊断分期 |
| **轻度振动病** | 从事振动作业1年以上，出现手部疼痛、麻木、僵硬等异常感觉，末梢循环功能检查或神经-肌电图检查有异常改变 |
| **中度振动病** | 出现明显的振动性白指，或手部麻木、疼痛、僵硬明显加重，末梢循环功能障碍或神经-肌电图检查明显异常 |
| **重度振动病** | 出现多指或全手坏死，或出现明显的肌肉萎缩、骨关节变形，神经-肌电图检查严重异常 |

### 4.3 诊断试验

| 试验名称 | 方法 | 阳性标准 |
|----------|------|----------|
| **冷水复温试验** | 将手浸入4℃水中2分钟后测量复温时间 | 复温时间超过30分钟为阳性 |
| **缩血管试验** | 冷刺激后测量指动脉血流 | 血流恢复延迟 |

### 4.4 鉴别诊断

- 雷诺病（原发性血管痉挛性疾病）
- 血栓闭塞性脉管炎
- 糖尿病周围神经病变
- 颈椎病所致上肢麻木
- 类风湿性关节炎

## 五、接触限值

### 5.1 工作场所振动接触限值

根据GBZ 2.2-2005《工作场所有害因素职业接触限值 物理因素》及GBZ/T 247-2013：

| 振动类型 | 接触限值 |
|----------|----------|
| **手传振动** | 日接振时间≤4h时，加速度≤5 m/s² |
| **全身振动（垂直）** | 8小时或等效日接触限值为0.62 m/s² |
| **全身振动（水平）** | 8小时或等效日接触限值为0.30 m/s² |

### 5.2 振动强度监测

- **频率计权加速度**（ahv）：综合评价振动对人体的影响
- **日接振时间**：每日累计接触振动的时间
- **振动病风险评估**：需综合考虑振动强度和暴露时间

### 5.3 环境监测要求

- 定期检测工作场所振动强度
- 重点监测点：手持振动工具操作位、驾驶室
- 建立振动监测档案

## 六、预防措施

### 6.1 工程控制

1. **选用低振动设备**：采用低振动工具和设备
2. **减振措施**：
   - 安装减振垫或减振手柄
   - 使用隔振材料
3. **湿式作业**：减少摩擦产生的振动
4. **自动化**：实现自动化操作，减少人工直接持握振动工具

### 6.2 个人防护

| 防护用品 | 适用场景 |
|----------|----------|
| **防振手套** | 常规作业（需具有减振效果） |
| **保暖手套** | 寒冷环境作业 |
| **耳塞或耳罩** | 振动伴随噪声时 |
| **护腕** | 保护腕关节 |

> **注意**：普通保暖手套不能替代专业的防振手套。

### 6.3 卫生措施

- **限制日接振时间**：每工作1-2小时休息10-15分钟
- **避免连续暴露**：采用轮换作业制度
- **保暖**：寒冷环境下保持手部温暖
- **戒烟**：吸烟会加重血管痉挛
- **加强锻炼**：促进末梢血液循环

### 6.4 健康管理

- 建立振动作业人员健康档案
- 开展职业健康教育（特别是早期症状识别）
- 实施定期健康检查
- 及时发现和处理早期患者
- 对观察对象密切随访

## 七、急救处理

### 7.1 现场急救

**步骤一：迅速脱离振动环境**
- 停止手持振动工具作业
- 移至温暖环境

**步骤二：保暖**
- 佩戴保暖手套
- 用温水浸泡受影响的四肢
- 避免接触冰冷物体

**步骤三：促进血液循环**
- 轻轻按摩受影响的部位
- 抬高四肢

**步骤四：就医**
- 及时就医
- 告知医生振动暴露史

### 7.2 医院治疗

**主要治疗原则**：

1. **脱离振动接触**：调离振动作业岗位
2. **末梢循环障碍治疗**：
   - 血管扩张药物（如硝苯地平）
   - 改善微循环药物（如银杏叶提取物）
   - 局部保暖
3. **神经功能障碍治疗**：
   - 营养神经药物（如维生素B1、B12）
   - 物理治疗（蜡疗、红外线）
4. **对症治疗**：
   - 镇痛药物
   - 肌肉松弛剂
5. **康复治疗**：
   - 运动疗法
   - 作业治疗

### 7.3 注意事项

- 振动性白指一旦发生，难以逆转
- 早期发现、早期干预是关键
- 避免长期连续接触振动
- 寒冷会加重症状

### 8.2 法律法规

1. **《中华人民共和国职业病防治法》（2018年修正）**
   - 规定用人单位职业病防治责任
   - 要求开展职业健康检查、职业病危害因素检测

2. **《工作场所职业卫生监督管理规定》（国家安全生产监督管理总局令第47号）**
   - 用人单位职业卫生管理要求

3. **《职业病诊断与鉴定管理办法》（国家卫生健康委员会令第6号）**
   - 职业病诊断程序规定

4. **《工伤保险条例》（国务院令第586号）**
   - 职业病享受工伤保险待遇

5. **《职业病分类和目录》（国卫疾控发〔2013〕48号）**
   - 手臂振动病列为法定职业病

### 8.3 用人单位责任

- 开展职业病危害项目申报
- 实施作业场所职业病危害因素检测与评价
- 为劳动者提供符合防治职业病要求的个人防护用品
- 组织劳动者进行职业健康检查（上岗前、在岗期间）
- 建立职业健康监护档案
- 报告职业病病例
