# 职业性急性化学物中毒：应急处置与救援指南

## 引言

**职业性急性化学物中毒**是指劳动者在短时间内接触大量化学毒物引起的中毒性疾病。这类中毒往往起病急骤、病情凶险、死亡率高，是职业卫生领域的重大安全隐患。近年来化工行业快速发展，化学物质种类日益增多，急性中毒事件时有发生。

根据国家突发公共卫生事件报告，职业性急性化学物中毒事故主要集中在化工、冶金、电子、印刷等行业。正确掌握应急处置知识，对于降低中毒死亡率、减少伤残具有重要意义。本文将从专业角度系统介绍职业性急性化学物中毒的应急处置与救援知识。

## 一、常见致毒化学物分类

### 1.1 按毒作用机制分类

| 类别 | 代表物质 | 主要毒性 |
|------|----------|----------|
| 窒息性气体 | 一氧化碳、硫化氢、氰化氢 | 阻断氧的摄取、利用 |
| 刺激性气体 | 氯气、氨气、光气、氮氧化物 | 呼吸道损伤 |
| 挥发性有机溶剂 | 苯、甲苯、正己烷 | 神经系统损害 |
| 重金属及其化合物 | 汞、铅、砷 | 多系统损伤 |
| 农药 | 有机磷、氨基甲酸酯 | 胆碱酯酶抑制 |

### 1.2 按毒性程度分类

| 剧毒 | 高毒 | 中等毒 | 低毒 |
|------|------|--------|------|
| 氰化氢 | 氯气 | 氨气 | 丙酮 |
| 硫化氢 | 光气 | 苯 | 甲苯 |
| 汞蒸气 | 氮氧化物 | 甲醇 | 乙醇 |

## 二、急性中毒的共同特点

### 2.1 发病特点

1. **起病急骤**：多在接触后数分钟至数小时内发病
2. **病情危重**：短时间内大量毒物进入体内
3. **症状明显**：出现特征性中毒表现
4. **变化迅速**：病情可在短时间内急剧恶化

### 2.2 临床特点

- 往往累及多个系统
- 存在剂量-效应关系
- 可能存在迟发性毒性
- 特效解毒剂有限

## 三、现场应急处置原则

### 3.1 基本原则

**（一）立即脱离现场**

- 首要原则是将中毒者移至空气新鲜处
- 抢救者需做好个人防护后方可进入现场
- 避免盲目进入造成二次中毒

**（二）快速评估**

- 评估中毒者意识、呼吸、循环状态
- 初步判断毒物种类和接触途径
- 及时呼叫专业救援

**（三）尽快清除毒物**

- 皮肤污染：立即脱去污染衣物，大量清水冲洗
- 眼睛污染：立即用流动清水冲洗至少15分钟
- 口服中毒：谨慎洗胃，视毒物特性决定

**（四）及时应用解毒剂**

- 已知特效解毒剂时应尽快使用
- 如氰化物中毒用亚硝酸钠+硫代硫酸钠
- 有机磷中毒用阿托品+氯磷定

**（五）支持治疗**

- 保持呼吸道通畅
- 维持循环稳定
- 对症处理并发症

### 3.2 现场急救流程

```
发现中毒者
    ↓
确保现场安全（通风、个人防护）
    ↓
快速评估生命体征
    ↓
↓           ↓
意识清醒    意识丧失
    ↓           ↓
移至空气    开放气道
新鲜处      人工呼吸
    ↓           ↓
初步问诊    胸外按压
    ↓           ↓
对症处理    立即送医
    ↓
呼叫120/专业救援
    ↓
送往医院
```

## 四、常见急性化学物中毒的处置

### 4.1 窒息性气体中毒

**（一）一氧化碳（CO）中毒**

*中毒特点：*

- 无色无味，不易察觉
- 与血红蛋白结合形成碳氧血红蛋白（HbCO）
- 组织严重缺氧
- 皮肤黏膜樱桃红色

*急救措施：*

1. 立即将患者移至通风处
2. 吸氧（最好高压氧舱）
3. 防治脑水肿：甘露醇、激素
4. 对症支持治疗
5. 严重者进行血液净化

**（二）硫化氢（H₂S）中毒**

*中毒特点：*

- 臭鸡蛋味（高浓度时反而无味）
- 强烈刺激性
- 瞬间大量吸入可致“电击样”死亡
- 可引起肺水肿、化学性肺炎

*急救措施：*

1. 立即脱离现场
2. 给予高流量氧气
3. 防治肺水肿：激素、利尿
4. 亚硝酸钠（注意：高铁血红蛋白形成剂，慎用）
5. 忌用亚甲蓝（会加重病情）

**（三）氰化氢（HCN）中毒**

*中毒特点：*

- 杏仁味
- 毒性剧烈，死亡率极高
- 细胞呼吸链阻断
- 抢救时间窗极短

*急救措施：*

1. 立即脱离现场
2. 特效解毒剂：
   - 亚硝酸异戊酯吸入
   - 3%亚硝酸钠静脉注射
   - 25%硫代硫酸钠静脉注射
3. 吸氧
4. 人工呼吸（必要时）
5. 及时送医

### 4.2 刺激性气体中毒

**（一）氯气（Cl₂）中毒**

*中毒特点：*

- 黄绿色刺激性气体
- 强烈刺激性：流泪、咳嗽、胸闷
- 严重者：喉头水肿、肺水肿、ARDS
- 可能在24-48小时后发生迟发性肺水肿

*急救措施：*

1. 立即脱离现场
2. 绝对卧床休息
3. 吸氧，保持呼吸道通畅
4. 防治肺水肿：
   - 糖皮质激素
   - 消泡剂（有机硅）
   - 限制液体入量
5. 眼部冲洗：2%碳酸氢钠
6. 密切观察48小时

**（二）氨气（NH₃）中毒**

*中毒特点：*

- 强烈刺激性臭味
- 碱性腐蚀作用
- 引起呼吸道、眼睛严重损伤
- 皮肤灼伤

*急救措施：*

1. 脱离现场
2. 皮肤污染：大量清水冲洗至少20分钟
3. 眼部冲洗：流动清水或生理盐水冲洗至少15分钟
4. 吸氧，解除支气管痉挛
5. 防治肺水肿：激素、限制液体
6. 防治感染

**（三）光气（COCl₂）中毒**

*中毒特点：*

- 无色气体，有霉草味
- 潜伏期较长（0.5-24小时）
- 迟发性肺水肿是主要死因
- 病情可能突然加重

*急救措施：*

1. 绝对卧床休息
2. 吸氧，保持安静
3. 防治肺水肿：
   - 早期大剂量激素
   - 限制液体
   - 利尿剂
4. 镇静剂（禁用吗啡）
5. 密切观察72小时

### 4.3 有机溶剂中毒

**（一）苯中毒**

*中毒特点：*

- 挥发性强，有特殊芳香气味
. 中枢神经系统抑制
- 严重者：意识障碍、昏迷
- 可能引起心律失常

*急救措施：*

1. 脱离现场
2. 脱去污染衣物
3. 皮肤清洗
4. 吸氧
5. 葡萄糖醛酸（禁用肾上腺素）
6. 对症支持治疗

**（二）正己烷中毒**

*中毒特点：*

- 慢性中毒多见，急性中毒相对较少
. 周围神经损害
- 乏力、麻木、肌肉萎缩

*急救措施：*

1. 脱离现场
2. 吸氧
3. 维生素B族、能量合剂
4. 物理治疗、功能锻炼

### 4.4 农药中毒

**（一）有机磷中毒**

*中毒特点：*

- 胆碱酯酶抑制
- 三大综合征：
  - 毒蕈碱样症状：出汗、流泪、流涎、腹泻
  - 烟碱样症状：肌颤、肌无力
  - 中枢神经系统症状：意识障碍
- 蒜臭味

*急救措施：*

1. 脱离现场
2. 洗胃（彻底清洗）
3. 特效解毒剂：
   - 阿托品（抗胆碱能）
   - 氯磷定/解磷定（复能剂）
4. 对症支持
5. 监测胆碱酯酶

**（二）百草枯中毒**

*中毒特点：*

- 无特效解毒剂
- 多器官功能衰竭
- 肺纤维化是主要死因
- 死亡率极高

*急救措施：*

1. 立即催吐
2. 洗胃：彻底
3. 吸附剂：活性炭
4. 导泻：硫酸镁
5. 血液净化：早期使用
6. 免疫抑制剂（争议）
7. 对症支持治疗

## 五、化学灼伤的处置

### 5.1 化学性皮肤灼伤

**处理原则：**

1. 立即脱离现场
2. 脱去污染衣物
3. 大量流动清水冲洗：
   - 酸灼伤：冲洗至少30分钟
   - 碱灼伤：冲洗至少60分钟
4. 勿用中和剂
5. 保护创面，勿弄破水泡
6. 送医进一步处理

### 5.2 化学性眼灼伤

**处理原则：**

1. 立即用流动清水冲洗至少15分钟
2. 翻开眼睑，彻底清洗结膜囊
3. 勿用中和剂
4. 覆盖保护
5. 尽快送眼科专科处理

## 六、群体性中毒事件的处置

### 6.1 现场指挥

1. 启动应急预案
2. 划定警戒区域
3. 组织人员疏散
4. 调配救援资源

### 6.2 医学救援

1. 检伤分类
2. 分级救治
3. 批量患者分流
4. 信息上报

### 6.3 现场调查

1. 毒物种类确认
2. 浓度检测
3. 暴露人群调查
4. 证据保全

## 七、预防措施

### 7.1 工程控制

1. 密闭化生产
2. 通风排毒设施
3. 自动化操作
4. 紧急切断装置

### 7.2 管理措施

1. 建立应急预案
2. 配备救援设备
3. 定期演练
4. 培训员工

### 7.3 个人防护

1. 正确选择防护用品
2. 定期检查维护
3. 正确佩戴使用
4. 及时更换

### 7.4 监测预警

1. 毒物浓度监测
2. 健康监护
3. 预警系统
4. 定期评估

## 八、常用解毒剂一览

| 毒物 | 解毒剂 | 用法 |
|------|--------|------|
| 氰化物 | 亚硝酸钠+硫代硫酸钠 | 静脉注射 |
| 有机磷 | 阿托品+氯磷定 | 静脉/肌肉 |
| 亚硝酸盐 | 亚甲蓝 | 静脉注射 |
| 汞、砷 | 二巯丁二酸 | 口服/静脉 |
| 铅 | 依地酸钙钠 | 静脉滴注 |
| 吗啡类 | 纳洛酮 | 静脉/肌肉 |
| 苯二氮卓类 | 氟马西尼 | 静脉注射 |

## 九、总结

职业性急性化学物中毒的处置关键在于：

1. **快速响应**：争分夺秒脱离现场
2. **正确判断**：识别毒物类型
3. **科学施救**：避免二次伤害
4. **及时送医**：尽早专业治疗
5. **预防为主**：落实防护措施

企业应建立完善的应急救援体系，劳动者应掌握基本的急救知识，只有做到“防患于未然”，才能有效保障生命安全。
