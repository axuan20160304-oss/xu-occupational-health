# 职业性甲醛中毒：诊断、预防与处理指南

甲醛（Formaldehyde，HCHO）是一种常见的有机化合物，在工业生产中广泛应用。职业性甲醛中毒是短期内接触较高浓度甲醛气体引起的以眼和呼吸系统损害为主的全身性疾病。本文依据GBZ 33-2002《职业性急性甲醛中毒诊断标准》等国家标准，系统阐述其危害特点、诊断标准及防控措施。

## 一、适用范围

本标准（GBZ 33-2002《职业性急性甲醛中毒诊断标准》）适用于：

- **木材加工行业**：人造板（胶合板、刨花板、纤维板）生产
- **家具制造**：板材粘合、涂装作业
- **纺织行业**：防皱整理、染料固定
- **皮革加工**：鞣制工序
- **化工行业**：甲醛及衍生物生产
- **建筑材料**：保温材料、涂料生产
- **医疗领域**：福尔马林（35-40%甲醛溶液）使用
- **殡葬行业**：遗体防腐处理
- **实验室**：标本固定保存

> **注意**：本标准主要适用于急性甲醛中毒，慢性甲醛暴露的健康影响可参照相关标准。

## 二、毒性特点

### 2.1 甲醛的理化性质

| 项目 | 特性 |
|------|------|
| 化学式 | HCHO |
| 外观 | 无色气体 |
| 气味 | 刺激性气味（阈值约0.5-1.0 ppm） |
| 溶解性 | 易溶于水（37%水溶液即福尔马林） |
| 沸点 | -19.5℃ |
| 挥发性 | 常温下易挥发 |

### 2.2 毒代动力学

- **吸收**：主要经呼吸道吸入，也可经皮肤、消化道吸收
- **分布**：分布于呼吸道黏膜、肺泡、脑组织
- **代谢**：在体内迅速氧化为甲酸，部分以二氧化碳形式呼出
- **排泄**：主要经尿液排出

### 2.3 中毒机制

1. **黏膜刺激**：甲醛与蛋白质结合，形成甲醛-蛋白质复合物，刺激眼和呼吸道黏膜
2. **过敏反应**：作为半抗原，引起I型和IV型过敏反应
3. **神经毒性**：可透过血脑屏障，影响中枢神经系统
4. **基因毒性**：导致DNA交联和染色体畸变
5. **致癌性**：IARC确认甲醛为人类致癌物（Group I），可致鼻咽癌、白血病

### 2.4 临床表现

**急性中毒**（短期内接触高浓度甲醛）：

| 系统 | 症状 |
|------|------|
| **眼部** | 眼结膜充血、流泪、畏光、视物模糊、眼睑水肿 |
| **呼吸道** | 咳嗽、咽痛、声音嘶哑、胸闷、呼吸困难 |
| **消化道** | 恶心、呕吐、腹痛（误服时） |
| **神经系统** | 头晕、头痛、意识模糊、甚至昏迷 |
| **皮肤** | 红斑、丘疹、瘙痒（直接接触时） |

**甲醛哮喘**：
- 潜伏期可长达48小时
- 发作前可无明显症状
- 属过敏性疾病

## 三、健康检查

### 3.1 上岗前体检

**必检项目**：
- 常规体检（内科、五官科）
- 血常规、尿常规
- 肝功能
- 胸部X线检查

**职业禁忌证**：
- 过敏性体质（哮喘、过敏性鼻炎、过敏性皮炎）
- 严重慢性呼吸系统疾病
- 明显的眼、鼻、咽喉慢性疾病
- 已知对甲醛过敏者

### 3.2 在岗期间体检

**体检周期**：
- 接触甲醛浓度合格：每年1次
- 接触甲醛浓度超标：每半年1次
- 出现症状者：缩短体检间隔

**体检内容**：
- 详细病史询问（眼症状、呼吸道症状、过敏史）
- 五官科检查（重点眼结膜、鼻黏膜、咽部）
- 肺功能检查（第一秒用力呼气容积FEV1、最大呼气流速PEF）
- 胸部X线检查

### 3.3 应急健康检查

接触大量甲醛后，应立即进行：
- 眼部检查
- 呼吸道症状评估
- 胸部X线检查
- 血气分析

## 四、诊断标准

### 4.1 诊断原则

依据GBZ 33-2002，职业性急性甲醛中毒的诊断需根据：

1. **短期内接触较高浓度甲醛的职业史**
2. **迅速发病**（通常在接触后数分钟至数小时发病）
3. **相应的临床表现**（以眼和呼吸系统损害为主）
4. **胸部X线检查结果**
5. **排除其他原因引起的类似疾病**

### 4.2 诊断分级

| 分级 | 诊断条件 |
|------|----------|
| **轻度中毒** | 具有下列情况之一者：①明显的眼及上呼吸道黏膜刺激症状，体征有眼结膜充血、水肿，两肺呼吸音粗糙；②胸部X射线检查有肺纹理增多、增粗，符合急性气管-支气管炎表现 |
| **中度中毒** | 具有下列情况之一者：①咳嗽、咳痰、胸闷、气促，两肺呼吸音粗糙；②胸部X射线检查有散在的点片状或斑片状阴影，符合急性支气管肺炎表现；③喉头水肿达二度 |
| **重度中毒** | 具有下列情况之一者：①咳嗽、咳痰、胸闷、气促、呼吸困难，明显紫绀；②胸部X射线检查呈片状阴影或大片融合阴影，符合肺炎或肺水肿表现；③喉头水肿达三度或出现四度；④昏迷或休克 |

### 4.3 鉴别诊断

- 急性呼吸道感染
- 过敏性结膜炎、鼻炎
- 支气管哮喘（发作期）
- 肺炎（细菌性、病毒性）
- 化学性肺水肿（其他气体所致）

## 五、接触限值

### 5.1 工作场所空气中的接触限值

根据GBZ 2.1-2019《工作场所有害因素职业接触限值 第1部分：化学有害因素》：

| 限值类型 | 限值 |
|----------|------|
| **时间加权平均浓度（PC-TWA）** | 0.5 mg/m³ |
| **短时间接触浓度（PC-STEL）** | 1.0 mg/m³ |

### 5.2 其他标准限值

| 标准 | 限值 |
|------|------|
| 居住区大气中甲醛浓度（日均值） | 0.05 mg/m³ |
| 室内甲醛浓度（关闭门窗12小时） | 0.08 mg/m³ |

### 5.3 环境监测要求

- 定期检测工作场所空气中甲醛浓度
- 重点监测点：生产车间、通风系统排风口、作业人员呼吸带
- 甲醛浓度超标时立即采取控制措施

## 六、预防措施

### 6.1 工程控制

1. **替代技术**：采用低甲醛或无甲醛胶粘剂替代传统脲醛树脂胶
2. **密闭化**：生产设备密闭化，减少甲醛逸散
3. **通风设施**：
   - 全面通风：保持车间空气流通
   - 局部排风：在甲醛散发源设置排风罩
4. **空气净化**：安装甲醛过滤或吸附装置

### 6.2 个人防护

| 防护用品 | 适用场景 |
|----------|----------|
| **防毒面具（甲醛专用滤罐）** | 甲醛浓度超标时必配 |
| **防护眼镜** | 防止眼刺激 |
| **防护手套** | 防止皮肤直接接触 |
| **防护服** | 防止皮肤污染 |

> **注意**：普通纱布口罩不能有效防护甲醛，必须使用专用防毒面具。

### 6.3 卫生措施

- **禁止在车间进食、饮水、吸烟**
- **工作服与个人服装分开存放**
- **下班后沐浴、更衣**
- **作业后严格洗手**
- **不在作业区居住或存放食品**

### 6.4 健康管理

- 建立甲醛作业人员健康档案
- 开展职业健康教育（特别是过敏和哮喘预防知识）
- 实施定期健康检查
- 及时发现和处理早期中毒者
- 对过敏性体质者密切观察

## 七、急救处理

### 7.1 现场急救

**步骤一：迅速脱离暴露**
- 立即将中毒者移至新鲜空气处
- 脱除被污染的衣物
- 用大量清水冲洗眼部（至少15分钟）
- 用肥皂水彻底清洗皮肤

**步骤二：保持呼吸道通畅**
- 松开衣领，头偏向一侧
- 清除口腔分泌物
- 吸氧（必要时）

**步骤三：立即就医**
- 拨打120急救电话
- 告知可能为甲醛中毒

### 7.2 医院治疗

**急性中毒处理**：
1. **脱离接触**：立即停止甲醛暴露
2. **眼处理**：
   - 大量生理盐水冲洗
   - 使用抗生素眼药水预防感染
   - 严重者请眼科会诊
3. **呼吸系统处理**：
   - 吸氧，保持呼吸道通畅
   - 雾化吸入（0.1%淡氨水，可促进甲醛转化为乌洛托品）
   - 必要时使用支气管扩张剂
   - 严重者使用糖皮质激素
4. **对症治疗**：
   - 镇咳、化痰
   - 防治感染
   - 保护肝肾功能
5. **观察**：密切观察48小时（潜伏期可能延长）

**甲醛哮喘处理**：
- 脱离暴露环境
- 吸氧
- 支气管扩张剂
- 糖皮质激素（严重时）
- 氨茶碱静脉滴注

### 7.3 注意事项

- 甲醛污染衣物应彻底清洗后再使用
- 忌用酒精擦拭受损皮肤（可能加重吸收）
- 严重中毒者需住院观察

### 8.2 法律法规

1. **《中华人民共和国职业病防治法》（2018年修正）**
   - 规定用人单位职业病防治责任
   - 要求开展职业健康检查、职业病危害因素检测

2. **《工作场所职业卫生监督管理规定》（国家安全生产监督管理总局令第47号）**
   - 用人单位职业卫生管理要求

3. **《职业病诊断与鉴定管理办法》（国家卫生健康委员会令第6号）**
   - 职业病诊断程序规定

4. **《工伤保险条例》（国务院令第586号）**
   - 职业病享受工伤保险待遇

5. **《职业病分类和目录》（国卫疾控发〔2013〕48号）**
   - 甲醛中毒列为法定职业病

### 8.3 用人单位责任

- 开展职业病危害项目申报
- 实施作业场所职业病危害因素检测与评价
- 为劳动者提供符合防治职业病要求的个人防护用品
- 组织劳动者进行职业健康检查（上岗前、在岗期间）
- 建立职业健康监护档案
- 报告职业病病例
