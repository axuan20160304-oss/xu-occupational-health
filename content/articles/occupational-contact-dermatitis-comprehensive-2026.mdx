# 职业性接触性皮炎：化工及制造业劳动者的常见皮肤病

## 引言

**职业性接触性皮炎**（Occupational Contact Dermatitis）是指劳动者在职业活动中因接触化学物质或其他职业性因素引起的皮肤炎症反应。这是化工、制造业、医疗等行业中最常见的职业性皮肤病之一，约占全部职业性皮肤病的80%以上。

职业性接触性皮炎虽然不像一些职业性肿瘤或尘肺病那样危及生命，但其发病率极高，反复发作，严重影响劳动者的生活质量和工作能力。这种疾病不仅给患者带来痛苦，也给企业带来经济损失。

本文将从专业角度全面介绍职业性接触性皮炎的相关知识，帮助广大劳动者认识这一常见职业性皮肤病。

## 一、病因与分类

### 1.1 病因

**（一）化学因素**

| 类型 | 物质举例 |
|------|----------|
| 酸类 | 硫酸、盐酸、硝酸 |
| 碱类 | 氢氧化钠、氨水 |
| 溶剂 | 苯、甲苯、丙酮 |
| 金属 | 镍、铬、汞 |
| 染料 | 偶氮染料、活性染料 |
| 树脂 | 环氧树脂、酚醛树脂 |
| 橡胶 | 硫化剂、促进剂 |

**（二）物理因素**

- 摩擦
- 压力
- 冷热刺激
- 紫外线

**（三）生物因素**

- 细菌
- 真菌
- 植物

### 1.2 分类

**（一）刺激性接触性皮炎**

- 直接损伤皮肤
- 非过敏性
- 任何人接触均可发生

**（二）过敏性接触性皮炎**

- 迟发型过敏反应
- 仅少数敏感者发生
- 需要致敏过程

## 二、发病机制

### 2.1 刺激性接触性皮炎

**（一）机制**

- 化学物质直接损伤皮肤屏障
- 引起炎症反应
- 导致细胞坏死

**（二）特点**

- 发病快
- 症状明显
- 脱离后恢复较快

### 2.2 过敏性接触性皮炎

**（一）致敏阶段**

- 过敏原穿透皮肤
- 被朗格汉斯细胞摄取
- 激活T淋巴细胞

**（二）反应阶段**

- 再次接触过敏原
- 引起迟发型超敏反应
- 导致皮炎

**（三）特点**

- 有潜伏期
- 反复发作
- 症状顽固

## 三、临床表现

### 3.1 急性期

**（一）症状**

| 症状 | 特点 |
|------|------|
| 红斑 | 境界清楚 |
| 水肿 | 程度不一 |
| 水疱 | 散在或密集 |
| 糜烂 | 渗出性 |
| 瘙痒 | 剧烈 |
| 灼痛 | 接触部位明显 |

### 3.2 慢性期

- 皮肤增厚
- 苔藓样变
- 色素沉着
- 裂纹
- 反复发作

### 3.3 好发部位

- 双手
- 前臂
- 面部
- 颈部
- 腕部

## 四、诊断与鉴别诊断

### 4.1 诊断依据

1. 职业接触史
2. 临床表现
3. 斑贴试验（过敏性皮炎）
4. 排除其他疾病

### 4.2 斑贴试验

**（一）方法**

- 将可疑过敏原贴于皮肤
- 48小时观察结果

**（二）结果判断**

| 反应 | 表现 |
|------|------|
| 阴性 | 无反应 |
| 可疑 | 轻度红斑 |
| 阳性 | 红斑、水疱 |

### 4.3 鉴别诊断

- 湿疹
- 银屑病
- 真菌感染

## 五、治疗方案

### 5.1 治疗原则

1. 立即脱离接触
2. 清除皮肤残留物
3. 对症治疗
4. 防止感染

### 5.2 治疗方法

**（一）急性期**

| 方法 | 作用 |
|------|------|
| 硼酸溶液湿敷 | 收敛、消炎 |
| 生理盐水冲洗 | 清洁 |
| 糖皮质激素 | 抗炎 |
| 抗组胺药 | 止痒 |

**（二）慢性期**

- 外用糖皮质激素软膏
- 尿素软膏保湿
- 避免刺激

### 5.3 注意事项

1. 避免水过热
2. 避免刺激性食物
3. 保持皮肤清洁
4. 避免再次接触

## 六、预防措施

### 6.1 工程控制

**（一）工艺改革**

- 采用低毒替代品
- 自动化生产
- 减少人工操作

**（二）防护设施**

- 安装通风设备
- 设置防护屏障
- 保持工作场所清洁

### 6.2 个人防护

**（一）防护手套**

| 类型 | 适用情况 |
|------|----------|
| 乳胶手套 | 一般化学品 |
| 丁腈手套 | 有机溶剂 |
| 橡胶手套 | 强酸碱 |

**（二）防护服**

- 防渗透材质
- 定期更换
- 保持清洁

### 6.3 皮肤护理

- 护肤霜
- 清洁用品
- 避免过度清洗

### 6.4 健康监护

**（一）就业前体检**

- 排除皮肤疾病
- 过敏体质评估

**（二）在岗期间体检**

- 定期皮肤检查
- 询问皮肤症状

## 七、法律保护与权益

### 7.1 职业病诊断

- 诊断机构：职业病诊断机构
- 诊断依据：GBZ 18

### 7.2 用人单位责任

- 提供防护设施
- 组织健康检查
- 落实防护措施

### 7.3 劳动者权利

- 知情权
- 防护权
- 拒绝违章权

## 八、总结

职业性接触性皮炎是完全可以预防的职业病。

**核心要点：**

1. 化学因素是最常见病因
2. 刺激性皮炎与过敏性皮炎不同
3. 早期发现很重要
4. 预防重于治疗

**预防要点：**

- 了解所接触物质危害
- 正确使用防护用品
- 保持皮肤清洁
- 出现症状及时就医

让我们共同关注职业性接触性皮炎防护，守护劳动者皮肤健康！
