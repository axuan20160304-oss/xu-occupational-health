## 一、适用范围

本指南适用于从事乙酸乙酯作业的用人单位及劳动者，主要涉及以下行业：

- **印刷包装**：油墨稀释、溶剂
- **化工行业**：有机合成、溶剂
- **电子制造**：清洗剂、溶剂
- **皮革行业**：皮革涂饰
- **橡胶行业**：橡胶溶剂
- **文具制造**：粘合剂

## 二、乙酸乙酯的基本特性

### 2.1 物理特性

| 特性 | 说明 |
|------|------|
| 外观 | 无色透明液体 |
| 气味 | 水果香味 |
| 沸点 | 77.1℃ |
| 闪点 | -4℃（易燃） |
| 挥发性 | 易挥发 |
| 溶解性 | 溶于多数有机溶剂，微溶于水 |

### 2.2 毒性特点

- **主要成分**：乙酸乙酯 (CH₃COOCH₂CH₃)
- **毒性**：低毒，刺激中枢神经系统
- **侵入途径**：呼吸道、皮肤
- **靶器官**：神经系统、肝脏
- **中毒表现**：头晕、乏力、恶心

## 三、职业健康检查

### 3.1 上岗前检查

从事乙酸乙酯作业的劳动者应进行上岗前职业健康检查，以下疾病或情况为职业禁忌症：

**禁忌症：**

- **神经系统**：多发性周围神经病、神经衰弱
- **肝脏疾病**：慢性肝炎、肝硬化
- **肾脏疾病**：慢性肾炎、肾功能不全
- **皮肤病**：慢性湿疹、皮炎
- **呼吸系统**：慢性支气管炎、哮喘

### 3.2 在岗检查

| 检查周期 | 要求 |
|----------|------|
| 接触乙酸乙酯浓度≥PC-TWA | 每年1次 |
| 接触浓度＜PC-TWA | 每2年1次 |

**检查项目：**

1. **症状询问**
   - 头晕、头痛、乏力
   - 恶心、呕吐、食欲不振
   - 四肢麻木、疼痛
   - 皮肤瘙痒、皮疹

2. **体格检查**
   - 神经系统检查
   - 肝脾触诊
   - 皮肤检查

3. **辅助检查**
   - 血常规、尿常规
   - 肝功能（ALT、AST）
   - 肾功能（肌酐、尿素氮）

### 3.3 离岗检查

劳动者离开乙酸乙酯作业岗位时，应进行离岗职业健康检查。

## 四、急性乙酸乙酯中毒

### 4.1 诊断标准

**诊断原则：**

1. 短期内接触较高浓度乙酸乙酯史
2. 出现神经系统损害为主的临床表现
3. 排除其他疾病

### 4.2 临床表现

| 程度 | 表现 |
|------|------|
| 轻度 | 头晕、头痛、乏力、恶心 |
| 中度 | 呕吐、嗜睡、步态不稳 |
| 重度 | 昏迷、呼吸抑制 |

### 4.3 慢性影响

长期接触可能引起：

- 神经系统功能紊乱
- 肝脏损害
- 皮肤刺激

## 五、接触限值

### 5.1 工作场所空气中乙酸乙酯浓度限值

| 标准 | PC-TWA | PC-STEL |
|------|---------|----------|
| GBZ 2.1-2007 | 200 mg/m³ | 300 mg/m³ |

### 5.2 监测要求

- **日常监测**：每月至少监测1次
- **定期监测**：每年至少进行1次全面监测
- **个体监测**：接触浓度≥50%PC-TWA者

## 六、预防控制措施

### 6.1 工程控制措施

| 措施 | 说明 |
|------|------|
| **密闭化** | 生产设备密闭化，减少泄漏 |
| **通风** | 设置全面通风和局部通风 |
| **替代** | 用低毒性溶剂替代乙酸乙酯 |
| **自动化** | 实现自动化生产，减少人工操作 |

### 6.2 管理措施

- **危害告知**：告知劳动者乙酸乙酯危害及防护措施
- **培训教育**：开展乙酸乙酯防护知识培训
- **健康监护**：严格执行职业健康检查制度
- **应急处理**：制定乙酸乙酯中毒应急预案

### 6.3 个人防护

| 防护用品 | 要求 |
|----------|------|
| **防毒面具** | 有机蒸气滤罐 |
| **防护手套** | 耐溶剂手套（如丁腈橡胶） |
| **防护服** | 防渗透工作服 |
| **防护眼镜** | 防护眼镜或防护面罩 |

### 6.4 作业环境要求

- 工作场所乙酸乙酯浓度≤PC-TWA
- 设置警示标识
- 配置洗眼器和淋浴设备
- 禁止明火作业

## 七、急救处理

### 7.1 急性乙酸乙酯中毒急救

**吸入中毒：**

1. 迅速脱离现场，移至空气新鲜处
2. 保持呼吸道通畅
3. 吸氧
4. 对症处理
5. 及时送医

**皮肤接触：**

1. 立即脱去被污染衣物
2. 用大量清水或肥皂水冲洗
3. 皮肤破损处按化学烧伤处理
4. 就医

**眼睛接触：**

1. 立即用流动清水冲洗
2. 冲洗时间≥15分钟
3. 就医

### 7.2 现场急救箱配置

| 物品 | 用途 |
|------|------|
| 有机蒸气口罩 | 呼吸防护 |
| 洗眼液 | 眼睛冲洗 |
| 急救手册 | 指导使用 |
