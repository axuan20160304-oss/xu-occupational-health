## 一、适用范围

本指南适用于从事噪声作业的用人单位及劳动者，主要涉及以下行业：

- **机械制造**：铸造、锻造、冲压、切割
- **建筑施工**：打桩、混凝土振捣
- **交通运输**：机场地勤、铁路养护
- **电子制造**：SMT生产线
- **木材加工**：锯木、刨削
- **纺织**：织布、印染
- **矿山**：凿岩、爆破

## 二、噪声基础知识

### 2.1 噪声定义

噪声是指一切干扰人们生活、学习、工作以及对人所从事的活动产生不良影响的声音。

### 2.2 噪声分类

| 类型 | 来源 |
|------|------|
| **机械噪声** | 机械设备运转、摩擦、撞击 |
| **气流噪声** | 气体流动、通风设备 |
| **电磁噪声** | 电机、变压器 |
| **爆炸噪声** | 爆破、冲击 |

### 2.3 噪声强度

| 强度(dB) | 典型场景 |
|-----------|------------|
| 30-40 | 安静办公室 |
| 50-60 | 一般谈话 |
| 70-80 | 繁忙交通 |
| 80-90 | 纺织车间 |
| 90-100 | 机械冲压 |
| >100 | 凿岩机、飞机 |

## 三、职业健康检查

### 3.1 上岗前检查（GBZ 188-2025）

**禁忌症：**

- 各种原因引起的永久性感音神经性听力损失
- 中度以上传导性耳聋
- 眩晕综合征
- 严重的耳鸣

### 3.2 在岗检查

| 接触强度 | 检查周期 |
|----------|----------|
| ≥85 dB(A) | 每年1次 |
| 80-85 dB(A) | 每2年1次 |

**检查项目：**

1. **症状询问**
   - 耳鸣、耳聋、眩晕
   - 噪声接触史
   - 耳部疾病史

2. **耳科检查**
   - 外耳道检查
   - 鼓膜检查

3. **纯音听力测试**
   - 500、1000、2000、3000、4000、6000Hz
   - 250、8000Hz（必要时）

### 3.3 离岗检查

- 离岗时应进行职业健康检查
- 检查项目同在岗检查

## 四、职业性噪声聋诊断

### 4.1 诊断标准（GBZ 49-2014）

**诊断原则：**

1. 连续3年以上职业性噪声接触史
2. 永久性感音神经性听力损失
3. 排除其他原因
4. 结合职业卫生调查

### 4.2 诊断分级

| 分级 | 听力损失程度 |
|------|--------------|
| 轻度 | 26-40 dB HL |
| 中度 | 41-55 dB HL |
| 重度 | 56-70 dB HL |
| 极重度 | >70 dB HL |

### 4.3 诊断要点

- 双耳高频（3000、4000、6000Hz）平均听阈≥40dB
- 听力损失为不可逆
- 排除其他原因（药物、老年等）

## 五、接触限值

### 5.1 工作场所噪声限值（GBZ 2.2-2007）

| 接触时间 | 限值 |
|----------|------|
| 8小时 | 85 dB(A) |
| 4小时 | 88 dB(A) |
| 2小时 | 91 dB(A) |
| 1小时 | 94 dB(A) |

**行动水平**：80 dB(A)

### 5.2 脉冲噪声峰值

| 工作日接触次数 | 峰值(dB) |
|-----------------|-----------|
| 100次 | 112 dB |
| 30次 | 115 dB |
| 10次 | 119 dB |
| 3次 | 122 dB |

### 5.3 监测要求

| 监测类型 | 频次 |
|----------|------|
| 日常监测 | 每月1次 |
| 定期监测 | 每年1次 |
| 个体监测 | 接触≥80dB(A)者 |

## 六、预防控制措施

### 6.1 工程控制

| 措施 | 说明 |
|------|------|
| **隔声** | 隔声墙、隔声罩 |
| **消声** | 消声器、声屏障 |
| **减振** | 减振垫、隔振沟 |
| **布局** | 噪声源远离操作区 |

### 6.2 个人防护

| 防护用品 | 防护效果 |
|----------|----------|
| 防噪声耳塞（SNR≥25dB） | 25-35 dB |
| 防噪声耳罩（NRR≥25dB） | 25-40 dB |
| 耳塞+耳罩 | >40 dB |

### 6.3 管理措施

- 制定噪声危害告知卡
- 开展噪声防护培训
- 严格职业健康检查
- 合理安排作业时间

### 6.4 健康监护

- **上岗前**：排除禁忌症
- **在岗**：定期听力监测
- **离岗**：评估听力损失

## 七、听力保护计划

### 7.1 行动水平防护

当噪声≥80dB(A)时，应：

1. 开展噪声危害告知
2. 配发防噪声耳塞
3. 实施职业健康检查

### 7.2 超过限值防护

当噪声≥85dB(A)时，应：

1. 采取工程控制措施
2. 严格使用个人防护用品
3. 减少暴露时间
4. 职业健康检查
