# 职业性氨气中毒的急救处理与临床救治

## 概述

职业性氨气中毒（Occupational Ammonia Poisoning）是指劳动者在职业活动中吸入氨气引起的以呼吸系统损害为主的全身性疾病。氨气（NH₃）是一种无色、有强烈刺激性气味的气体，广泛应用于制冷、化工、化肥、皮革、制药等行业。《职业病分类和目录》将氨气中毒列入职业性急性化学物中毒的范畴。

氨气中毒的特点：
- **刺激性极强**：对呼吸道黏膜有强烈的刺激和腐蚀作用
- **毒性分级明显**：浓度越高，病情越重
- **易造成群体中毒**：泄漏事故常导致多人中毒

## 中毒机制

### 毒理学特点

氨气分子量17.03，沸点-33.42℃，极易溶于水，形成氢氧化铵（氨水），具有强碱性：

```
NH₃ + H₂O → NH₄OH
```

### 病理生理

1. **碱性腐蚀作用**：氨气溶于水后形成强碱性氨水，对呼吸道黏膜产生腐蚀性损伤
2. **直接刺激作用**：引起呼吸道黏膜充血、水肿、痉挛
3. **肺水肿**：血管通透性增加，引起肺水肿
4. **全身毒性**：高浓度氨气可引起中枢神经系统抑制

### 毒性分级

| 浓度（ppm） | 暴露时间 | 毒性表现 |
|-------------|----------|----------|
| 25-50 | 持续 | 轻微刺激感，可感知气味 |
| 100-200 | 持续 | 明显眼部及呼吸道刺激 |
| 500-1000 | 30min内 | 剧烈刺激，呼吸困难，可能死亡 |
| >2000 | 短时间 | 闪电样死亡，喉头水肿迅速窒息 |

## 急救处理

### 现场急救原则

1. **迅速脱离现场**：立即将中毒者转移至空气新鲜处，逆风撤离
2. **保持呼吸道通畅**：清除口鼻分泌物，解除呼吸道梗阻
3. **保暖**：避免寒冷加重病情
4. **心肺复苏**：如心跳呼吸停止，立即心肺复苏
5. **及时报警**：报告相关部门控制氨气泄漏

### 院前急救措施

#### 一、轻度中毒

- 立即转移至通风处
- 给予吸氧（2-4L/min）
- 安静休息，保暖
- 清水冲洗面部和眼部
- 密切观察12-24小时

#### 二、中度中毒

- 高流量吸氧（4-8L/min）
- 支气管扩张剂雾化吸入
- 静脉注射糖皮质激素（地塞米松10-15mg）
- 镇静剂，限制活动
- 紧急送医

#### 三、重度中毒

- 高流量吸氧，必要时气管切开或机械通气
- 大剂量糖皮质激素（地塞米松20-40mg）
- 防治肺水肿（利尿、脱水）
- 心电监护
- 镇静剂（避免挣扎加重呼吸困难）
- 立即送入ICU

### 洗消处理

- **皮肤污染**：立即用大量清水冲洗，至少20分钟，再用2%硼酸溶液冲洗
- **眼部污染**：立即用流动清水冲洗至少15分钟，再用3%硼酸溶液冲洗，滴入抗生素眼药水
- **衣物污染**：立即脱除，密封处理

## 临床救治

### 治疗原则

1. **立即终止接触**：迅速脱离氨气环境
2. **清除毒物**：彻底冲洗体表和呼吸道
3. **对症支持治疗**：以对症治疗为主
4. **防治并发症**：预防感染、肺水肿、呼吸衰竭

### 药物治疗

#### 一、糖皮质激素

| 药物 | 用法 | 适应症 |
|------|------|--------|
| 地塞米松 | 10-20mg静脉注射，重症可追加 | 中重度中毒 |
| 甲泼尼龙 | 40-80mg静脉注射 | 中度中毒 |
| 氢化可的松 | 200-400mg静脉滴注 | 重度中毒 |

#### 二、呼吸系统用药

- **支气管扩张剂**：沙丁胺醇、特布他林雾化吸入
- **祛痰剂**：氨溴索、乙酰半胱氨酸
- **抗感染**：第三代头孢菌素或喹诺酮类预防感染

#### 三、眼部用药

- 抗生素眼药水（氯霉素、氧氟沙星）
- 玻璃酸钠滴眼液（保护角膜）
- 必要时请眼科会诊

### 特殊治疗

#### 一、喉头水肿处理

1. **雾化吸入**：地塞米松雾化
2. **静脉注射**：地塞米松静脉注射
3. **气管切开**：严重呼吸困难时及时行气管切开
4. **机械通气**：必要时呼吸机辅助通气

#### 二、肺水肿治疗

1. **体位**：半卧位，双下肢下垂
2. **吸氧**：高流量面罩吸氧，湿化瓶加酒精
3. **利尿**：呋塞米静脉注射
4. **血管扩张剂**：酚妥拉明、硝酸甘油
5. **强心剂**：西地兰（有心衰时）
6. **机械通气**：顽固性低氧血症时

#### 三、呼吸衰竭治疗

1. **保持气道通畅**：及时吸痰，支气管镜吸痰
2. **氧疗**：根据缺氧程度选择不同方式
3. **机械通气**：根据病情选择无创或有创通气
4. **呼吸兴奋剂**：洛贝林、尼可刹米（慎用）

### 康复治疗

1. **呼吸功能锻炼**：腹式呼吸、缩唇呼吸
2. **氧疗**：低流量长期家庭氧疗
3. **物理治疗**：超短波理疗促进肺部吸收
4. **心理康复**：氨气中毒常伴心理创伤，需心理疏导

## 诊断分级

根据《GBZ 29-2011 职业性急性化学物中毒的诊断》：

| 分级 | 临床表现 |
|------|----------|
| 刺激反应 | 一过性眼部及上呼吸道刺激症状，肺部无阳性体征 |
| 轻度中毒 | 咳嗽、咳痰，两肺闻及干啰音，支气管黏膜充血、水肿 |
| 中度中毒 | 咳嗽剧烈，咳痰或咳血性痰，呼吸困难，肺部散在干湿啰音 |
| 重度中毒 | 剧烈咳嗽，咳大量粉红色或白色泡沫痰，呼吸极度困难，肺水肿 |

## 劳动能力鉴定

### 鉴定标准

根据《劳动能力鉴定 职工工伤与职业病致残等级》（GB/T 16180）：

| 伤残等级 | 鉴定标准 |
|----------|----------|
| 四级 | 急性氨气中毒后遗留严重呼吸困难，需长期吸氧 |
| 五级 | 急性氨气中毒后遗留中度呼吸困难，体力活动明显受限 |
| 六级 | 急性氨气中毒后遗留轻度呼吸困难 |

### 鉴定要点

- 脱离接触后症状是否持续
- 肺功能损害程度（FEV1、FVC）
- 影像学改变（肺部纤维化、肺气肿）
- 是否需要长期治疗和氧疗

## 预防措施

### 工程控制

- **密闭化生产**：实现生产过程自动化、密闭化
- **通风排毒**：安装局部通风和全面通风设施
- **监测报警**：安装氨气浓度监测仪和报警装置
- **应急设施**：设置应急洗眼器、喷淋装置
- **替代技术**：采用无氨工艺或低温制冷技术

### 个人防护

- 过滤式防毒面具（配氨气滤罐）
- 空气呼吸器（进入高浓度区域）
- 防护服、防护手套、防护靴
- 作业后沐浴更衣

### 健康监护

- **上岗前体检**：筛选呼吸道疾病、皮肤病
- **定期在岗体检**：每年1次
- **应急救援人员**：专项体检

### 应急处置

- 制定氨气泄漏应急预案
- 定期组织应急演练
- 配备应急救援物资
- 设立紧急撤离路线和集合点

## 总结

职业性氨气中毒是常见的刺激性气体中毒，病情危重，进展迅速。急救处理的关键是迅速脱离现场、及时吸氧和早期使用糖皮质激素。临床救治以对症支持治疗为主，重点防治喉头水肿、肺水肿和呼吸衰竭。用人单位应落实工程防护和个人防护措施，完善应急处置预案，预防氨气中毒的发生。
