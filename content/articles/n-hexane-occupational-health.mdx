## 一、适用范围

本指南适用于从事正己烷作业的用人单位及劳动者，主要涉及以下行业：

- **电子制造**：电路板清洗、清洗剂
- **印刷行业**：油墨稀释、清洗
- **化工行业**：有机合成、溶剂
- **橡胶行业**：橡胶溶剂
- **皮革行业**：皮革涂饰
- **文具制造**：粘合剂

## 二、正己烷的基本特性

### 2.1 物理特性

| 特性 | 说明 |
|------|------|
| 外观 | 无色透明液体 |
| 气味 | 特殊石油味 |
| 沸点 | 68.74℃ |
| 闪点 | -22℃（极易燃） |
| 挥发性 | 极易挥发 |
| 溶解性 | 不溶于水，溶于有机溶剂 |

### 2.2 毒性特点

- **主要成分**：正己烷 (n-hexane)
- **毒性**：低毒，主要损害神经系统
- **侵入途径**：呼吸道、皮肤
- **靶器官**：周围神经系统
- **中毒表现**：多发性周围神经病

## 三、职业健康检查

### 3.1 上岗前检查

从事正己烷作业的劳动者应进行上岗前职业健康检查，以下疾病或情况为职业禁忌症：

**禁忌症：**

- **神经系统**：多发性周围神经病、重症神经症
- **血液系统**：贫血、血小板减少性疾病
- **肝脏疾病**：慢性肝炎、肝硬化
- **肾脏疾病**：慢性肾炎、肾功能不全
- **皮肤病**：慢性湿疹、皮炎
- **心血管疾病**：严重心脏病

### 3.2 在岗检查

| 检查周期 | 要求 |
|----------|------|
| 接触正己烷浓度≥PC-TWA | 每年1次 |
| 接触浓度＜PC-TWA | 每2年1次 |

**检查项目：**

1. **症状询问**
   - 头晕、头痛、乏力
   - 四肢麻木、疼痛
   - 食欲不振、恶心
   - 皮肤瘙痒、皮疹

2. **体格检查**
   - 神经系统检查（感觉、运动）
   - 肝脾触诊
   - 皮肤检查

3. **辅助检查**
   - 血常规、尿常规
   - 肝功能（ALT、AST）
   - 肾功能（肌酐、尿素氮）
   - 神经肌电图（必要时）

### 3.3 离岗检查

劳动者离开正己烷作业岗位时，应进行离岗职业健康检查。

## 四、职业病诊断

### 4.1 诊断标准

依据GBZ 21-2002《职业性正己烷中毒诊断标准》：

**诊断原则：**

1. 短期内接触较高浓度正己烷或长期密切接触正己烷史
2. 出现周围神经系统损害为主的临床表现
3. 排除其他疾病
4. 结合职业卫生调查资料

### 4.2 诊断分期

| 分期 | 主要表现 |
|------|----------|
| **观察对象** | 症状轻微，客观检查无阳性体征 |
| **轻度中毒** | 四肢远端麻木、疼痛，感觉减退，肌力下降 |
| **中度中毒** | 上述症状加重，出现肌肉萎缩，行走困难 |
| **重度中毒** | 四肢瘫痪，大小便失禁，生活不能自理 |

### 4.3 慢性正己烷中毒

长期吸入正己烷蒸气可导致慢性中毒：

- **神经系统**：多发性周围神经病
- **肌肉萎缩**：进行性肌无力
- **恢复缓慢**：脱离后仍可持续数月

## 五、接触限值

### 5.1 工作场所空气中正己烷浓度限值

| 标准 | PC-TWA | PC-STEL |
|------|---------|----------|
| GBZ 2.1-2007 | 100 mg/m³ | - |

### 5.2 监测要求

- **日常监测**：每月至少监测1次
- **定期监测**：每年至少进行1次全面监测
- **个体监测**：接触浓度≥50%PC-TWA者

## 六、预防控制措施

### 6.1 工程控制措施

| 措施 | 说明 |
|------|------|
| **密闭化** | 生产设备密闭化，减少泄漏 |
| **通风** | 设置全面通风和局部通风 |
| **替代** | 用低毒性溶剂替代正己烷 |
| **自动化** | 实现自动化生产，减少人工操作 |

### 6.2 管理措施

- **危害告知**：告知劳动者正己烷危害及防护措施
- **培训教育**：开展正己烷防护知识培训
- **健康监护**：严格执行职业健康检查制度
- **应急处理**：制定正己烷中毒应急预案

### 6.3 个人防护

| 防护用品 | 要求 |
|----------|------|
| **防毒面具** | 活性炭滤罐或供气式 |
| **防护手套** | 耐溶剂手套（如丁腈橡胶） |
| **防护服** | 防渗透工作服 |
| **防护眼镜** | 防护眼镜或防护面罩 |

### 6.4 作业环境要求

- 工作场所正己烷浓度≤PC-TWA
- 设置警示标识
- 配置洗眼器和冲洗设备
- 禁止明火作业

## 七、急救处理

### 7.1 急性正己烷中毒急救

**吸入中毒：**

1. 迅速脱离现场，移至空气新鲜处
2. 保持呼吸道通畅
3. 吸氧
4. 对症处理
5. 及时送医

**皮肤接触：**

1. 立即脱去被污染衣物
2. 用大量清水或肥皂水冲洗
3. 皮肤破损处按化学烧伤处理
4. 就医

**眼睛接触：**

1. 立即用流动清水冲洗
2. 冲洗时间≥15分钟
3. 就医

### 7.2 慢性中毒处理

1. 脱离正己烷接触
2. 营养神经治疗
3. 物理治疗
4. 康复训练
5. 定期复查
