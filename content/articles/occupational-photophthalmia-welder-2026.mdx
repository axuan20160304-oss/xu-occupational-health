# 职业性电光性眼炎：电焊作业的常见眼部职业病

## 引言

**职业性电光性眼炎**（Occupational Photophthalmia），又称电光性角膜炎、弧光眼炎，是电焊作业及其他接触紫外线辐射的职业人群中常见的急性眼部损伤。这种疾病虽然通常是暂时性的，但如果反复发作或处理不当，也可能造成永久性视力损害。

电光性眼炎主要发生在电焊、气焊、氩弧焊、等离子切割等作业过程中，以及使用紫外线消毒灯、碳弧灯等场合。由于焊工群体庞大，该疾病在职业性眼病中占据重要地位。本文将从专业角度全面介绍职业性电光性眼炎的相关知识，为广大焊工及相关从业人员提供科学的健康指导。

## 一、病因与发病机制

### 1.1 病因

**主要病因：**

职业性电光性眼炎的主要病因是短时间内接受大量紫外线辐射。常见于以下作业：

| 作业类型 | 紫外线来源 | 风险特点 |
|----------|-------------|----------|
| 电弧焊 | 电弧光 | 最常见，强度高 |
| 气焊 | 火焰 | 温度高，辐射强 |
| 氩弧焊 | 氩弧 | 紫外线强度高 |
| 等离子切割 | 等离子弧 | 辐射强度大 |
| 碳弧灯作业 | 碳弧光 | 辐射集中 |
| 紫外线消毒 | 紫外灯 | 长时间暴露 |

**易感因素：**

- 未佩戴或未正确佩戴防护用具
- 防护用具不合格或损坏
- 多人同时作业时观察他人焊接
- 雪地、水面等反射环境作业
- 设备故障导致紫外线泄漏

### 1.2 发病机制

**紫外线对眼部的损伤：**

1. **波长与穿透性**
   - 紫外线波长：100-400nm
   - 角膜上皮对紫外线最为敏感
   - 波长295-320nm可被角膜完全吸收

2. **损伤机制**
   - 紫外线引起角膜上皮细胞光化学反应
   - 导致细胞蛋白质变性
   - 引起角膜上皮细胞凋亡
   - 造成角膜上皮点状缺损

3. **潜伏期**
   - 通常在照射后6-12小时发病
   - 潜伏期长短与照射强度相关
   - 强度越高，潜伏期越短

### 1.3 剂量阈值

- **最小红斑剂量**：约50 mJ/cm²
- **角膜损伤阈值**：约60-100 mJ/cm²
- 短时间（数秒至数十秒）高强度暴露即可发病

## 二、临床表现

### 2.1 症状

**典型症状：**

| 症状 | 特点 |
|------|------|
| 眼部剧烈疼痛 | 沙砾感、刺痛、灼痛 |
| 畏光 | 无法睁眼，光照加重疼痛 |
| 流泪 | 大量流泪，泪液增多 |
| 异物感 | 感觉眼内有异物 |
| 视物模糊 | 角膜上皮缺损导致视力下降 |
| 眼睑痉挛 | 眼部肌肉痉挛，无法睁眼 |

**伴随症状：**

- 头痛
- 面部皮肤灼热感
- 全身不适

### 2.2 体征

**裂隙灯检查：**

- 角膜上皮点状缺损
- 荧光素染色阳性
- 角膜上皮水肿
- 结膜充血
- 眼睑肿胀

**眼部检查：**

| 检查项目 | 表现 |
|----------|------|
| 视力 | 轻至中度下降 |
| 眼压 | 正常或轻度升高 |
| 角膜荧光素染色 | 点状、弥漫性着色 |
| 结膜充血 | 弥漫性充血 |
| 眼睑 | 肿胀、痉挛 |

### 2.3 临床分型

**（一）轻度**

- 角膜上皮点状缺损<5个
- 症状轻微
- 24-48小时自愈

**（二）中度**

- 角膜上皮片状缺损
- 症状明显
- 48-72小时愈合

**（三）重度**

- 角膜上皮大范围缺损
- 症状严重
- 需要治疗，愈合较慢

## 三、诊断与鉴别诊断

### 3.1 诊断标准

根据GBZ 9-2002《职业性急性电光性眼炎（紫外线角膜结膜炎）诊断标准》：

**诊断依据：**

1. 职业接触史
2. 紫外线照射史
3. 临床表现：眼部刺激症状
4. 体征：角膜荧光素染色阳性
5. 潜伏期6-12小时

**诊断要点：**

- 有职业性紫外线接触史
- 出现典型眼部症状
- 裂隙灯检查发现角膜上皮损伤
- 排除其他病因

### 3.2 鉴别诊断

需要与以下疾病鉴别：

| 疾病 | 鉴别要点 |
|------|----------|
| 细菌性结膜炎 | 分泌物多，细菌培养阳性 |
| 病毒性结膜炎 | 耳前淋巴结肿大，病程较长 |
| 过敏性结膜炎 | 过敏史，季节性，痒感明显 |
| 机械性损伤 | 有异物史，裂隙灯可见异物 |
| 化学性损伤 | 有化学物质接触史 |
| 角膜上皮糜烂 | 有相关眼病病史 |

### 3.3 辅助检查

1. **荧光素染色**：显示角膜上皮缺损
2. **裂隙灯检查**：详细观察角膜病变
3. **眼压测量**：排除青光眼
4. **视力检查**：评估视力影响

## 四、治疗方案

### 4.1 治疗原则

1. **立即脱离紫外线环境**
2. **减轻疼痛**
3. **促进角膜上皮愈合**
4. **预防感染**
5. **对症处理**

### 4.2 治疗方法

**（一）局部治疗**

| 药物 | 作用 | 用法 |
|------|------|------|
| 表面麻醉剂 | 缓解疼痛 | 0.5%丁卡因，紧急使用 |
| 抗生素滴眼液 | 预防感染 | 氯霉素、氧氟沙星等 |
| 人工泪液 | 保持湿润 | 玻璃酸钠滴眼液 |
| 生长因子 | 促进愈合 | 重组牛碱性成纤维细胞生长因子 |
| 糖皮质激素 | 减轻炎症 | 氢化可的松滴眼液（短期） |

**（二）全身治疗**

- 镇痛药物：口服布洛芬等
- 维生素A：促进上皮修复
- 抗生素：严重者可口服

**（三）一般处理**

- 冷敷：减轻眼部肿胀
- 避光：佩戴遮光眼镜
- 休息：减少用眼
- 避免揉眼

### 4.3 治疗注意事项

1. **表面麻醉剂慎用**：仅限紧急缓解疼痛，不可长期使用
2. **糖皮质激素**：短期使用，避免并发症
3. **预防感染**：保持眼部清洁
4. **避免佩戴隐形眼镜**：治疗期间禁止佩戴
5. **定期复查**：确保角膜愈合良好

### 4.4 预后

- 绝大多数患者预后良好
- 角膜上皮通常在24-72小时内愈合
- 一般不留后遗症
- 反复发作可能导致持续性角膜病变

## 五、预防措施

### 5.1 工程控制

**（一）隔离防护**

- 设置防护屏
- 使用自动焊接
- 隔离操作室

**（二）通风改善**

- 保持工作场所通风
- 减少紫外线浓度

**（三）环境优化**

- 减少反射面
- 使用防护围挡
- 合理布局工作区

### 5.2 个人防护

**（一）焊接防护用具**

| 防护用具 | 要求 |
|----------|------|
| 焊接面罩 | 符合国家标准，滤光镜片 |
| 防护眼镜 | UV400防护 |
| 防护手套 | 防火、防辐射 |
| 防护服 | 阻燃、防护 |

**（二）滤光镜片选择**

| 焊接类型 | 滤光号 |
|----------|--------|
| 电流<30A | 5号 |
| 30-75A | 7号 |
| 75-200A | 9号 |
| 200-400A | 11号 |
| >400A | 12号以上 |

**（三）正确佩戴**

- 面罩必须放下
- 眼镜贴合面部
- 不作业时及时佩戴
- 多人作业时互相提醒

### 5.3 健康监护

**（一）上岗前体检**

- 眼部检查
- 排除禁忌证：
  - 严重结膜炎
  - 角膜炎
  - 白内障
  - 视网膜病变

**（二）在岗期间体检**

- 每年一次
- 眼部常规检查
- 视功能检查

**（三）离岗体检**

- 评估眼部健康状况

### 5.4 安全培训

1. 紫外线危害知识
2. 防护用具正确使用
3. 紧急处理方法
4. 定期安全教育

## 六、特殊人群与情况

### 6.1 新手焊工

- 发病率较高
- 防护意识不足
- 需要特别培训

### 6.2 辅助工种

- 配合焊工作业
- 同样需要防护
- 避免围观焊接

### 6.3 室内作业

- 紫外线反射
- 通风不良
- 需要更好防护

### 6.4 夜间作业

- 暗环境瞳孔大
- 更易受伤
- 防护更加重要

## 七、常见误区

### 7.1 误区一：偶尔不戴没关系

**事实：** 短时间暴露即可发病，不能心存侥幸

### 7.2 误区二：症状轻不需要治疗

**事实：** 不治疗可能继发感染延长病程

### 7.3 误区三：用人奶滴眼可以治疗

**事实：** 无科学依据，可能造成感染

### 7.4 误区四：看别人焊接没关系

**事实：** 旁观同样会受到紫外线辐射

### 7.5 误区五：症状消失就是好了

**事实：** 需要复查确保角膜完全愈合

## 八、法律保护与权益

### 8.1 职业病诊断

- 诊断机构：职业病诊断机构
- 诊断依据：GBZ 9-2002
- 需要材料：职业史、照射史、临床资料

### 8.2 工伤认定

- 属于职业病
- 认定为工伤
- 享受相关待遇

### 8.3 用人单位责任

- 提供合格防护用具
- 组织健康检查
- 落实防护措施
- 进行安全培训

### 8.4 劳动者权利

- 知情权
- 防护权
- 拒绝违章权
- 索赔权

## 九、案例分析

### 案例一：新手焊工急性发病

**案情：** 某工厂新招聘焊工，首次作业未佩戴防护面罩，约10秒后出现眼部剧痛、畏光、流泪。

**处理：**

- 立即脱离工作环境
- 冷敷眼部
- 使用抗生素滴眼液
- 24小时后症状缓解

**启示：**

- 必须正确佩戴防护用具
- 新员工培训至关重要

### 案例二：多次发病导致慢性损害

**案情：** 某焊工长期不重视防护，反复发作电光性眼炎，后出现持续性角膜上皮糜烂。

**教训：**

- 反复发作可造成永久性损害
- 预防胜于治疗

## 十、总结

职业性电光性眼炎是完全可以预防的职业病。

**核心要点：**

1. 紫外线辐射是致病原因
2. 潜伏期6-12小时
3. 主要症状：疼痛、畏光、流泪
4. 治疗以对症为主，预后良好
5. 防护是关键：面罩、眼镜
6. 反复发作可能造成永久性损害

**防护要点：**

- 正确佩戴焊接面罩
- 选择合适滤光号
- 辅助人员同样防护
- 多人作业互相提醒
- 定期进行健康检查

让我们共同关注焊工眼部健康，做好职业防护，远离电光性眼炎！
