# 职业性镉中毒的诊断标准与金属络合剂治疗

## 概述

职业性镉中毒（Occupational Cadmium Poisoning）是指劳动者在职业活动中因接触镉及其化合物而引起的中毒性疾病。镉是一种重金属，广泛应用于电池制造、电镀、化工、颜料、合金等行业。《职业病分类和目录》将镉及其化合物中毒列为法定职业病。

镉中毒的特点：
- **蓄积性强**：镉在体内半衰期长达10-30年
- **肾毒性明显**：主要损害肾脏近曲小管
- **骨损害**：可引起骨质疏松和骨软化（痛痛病）

## 中毒机制

### 毒理学特点

镉（Cd）是一种银白色金属，原子量112.41。职业性镉中毒主要因接触镉烟、镉粉尘引起，如氧化镉、硫化镉、氯化镉等。

### 病理生理

1. **巯基结合**：镉与含巯基的酶结合，抑制酶活性
2. **氧化损伤**：诱导自由基生成，引起脂质过氧化
3. **肾小管损害**：选择性损害近曲小管上皮细胞
4. **钙磷代谢障碍**：影响维生素D代谢，引起骨质疏松

### 毒性分级

| 血镉含量（μg/L） | 尿镉含量（μg/g肌酐） | 病情程度 |
|-------------------|----------------------|----------|
| <5 | <5 | 正常范围 |
| 5-10 | 5-10 | 轻度镉负荷 |
| 10-20 | 10-15 | 中度镉负荷 |
| >20 | >15 | 镉中毒 |

## 诊断标准

### 诊断原则

根据《GBZ 33-2015 职业性镉中毒的诊断》，诊断需满足以下条件：

1. **明确的镉接触史**：有确切的职业性镉接触史
2. **符合临床表现**：出现头晕、乏力、腰痛、骨痛等症状
3. **实验室检查异常**：血镉、尿镉含量增高
4. **排除其他疾病**：排除其他原因引起的肾损害和骨病

### 诊断依据

#### 一、职业接触史调查

| 行业 | 接触镉化物 | 常见工种 |
|------|------------|----------|
| 电池制造 | 镉镍电池 | 电池组装工 |
| 电镀 | 氯化镉、硫酸镉 | 电镀工 |
| 化工 | 氧化镉、硫化镉 | 化工操作工 |
| 颜料 | 镉黄、镉红 | 颜料制作工 |
| 合金 | 镉合金 | 合金熔炼工 |

#### 二、临床表现

##### 急性镉中毒

- **金属烟热**：高热、寒战、肌肉酸痛
- **呼吸道刺激**：咳嗽、胸闷、呼吸困难
- **消化系统**：恶心、呕吐、腹痛、腹泻

##### 慢性镉中毒

- **神经系统**：头晕、头痛、记忆力减退、失眠
- **泌尿系统**：蛋白尿、尿频、夜尿增多
- **骨骼系统**：腰痛、骨痛、骨质疏松
- **其他**：嗅觉减退、牙齿颈部黄染（镉环）

#### 三、实验室检查

1. **血镉测定**：血镉>5μg/L提示镉负荷增加
2. **尿镉测定**：尿镉>5μg/g肌酐有诊断意义
3. **尿β2-微球蛋白**：>1000μg/g提示肾小管损害
4. **尿NAG酶**：升高提示肾小管细胞损伤
5. **血生化**：血钙、血磷降低，碱性磷酸酶升高
6. **骨密度测定**：了解骨质疏松程度

### 诊断分级

#### 急性镉中毒

| 分级 | 临床表现 |
|------|----------|
| 轻度 | 金属烟热症状，咳嗽、胸闷，血镉增高 |
| 重度 | 严重呼吸道症状、肺水肿、肾功能损害 |

#### 慢性镉中毒

| 分级 | 临床表现 |
|------|----------|
| 轻度 | 头晕、乏力，尿镉、血镉增高，尿β2-MG轻度升高 |
| 中度 | 明显腰痛、骨痛，尿β2-MG明显升高，骨密度降低 |
| 重度 | 肾功能障碍、骨质疏松、骨折 |

## 鉴别诊断

### 需要鉴别的疾病

1. **慢性肾炎**：原发性肾脏疾病
2. **骨质疏松**：老年性、绝经后骨质疏松
3. **风湿性疾病**：类风湿关节炎、强直性脊柱炎
4. **范可尼综合征**：遗传性或获得性肾小管功能障碍

### 鉴别要点

| 疾病 | 关键鉴别特征 |
|------|--------------|
| 慢性肾炎 | 无镉接触史，尿镉正常，肾穿刺病理改变不同 |
| 老年性骨质疏松 | 发病年龄晚，无镉接触史，骨代谢标志物不同 |
| 类风湿关节炎 | 类风湿因子阳性，关节X线改变特征性 |

## 金属络合剂治疗

### 治疗原则

1. **立即终止接触**：脱离镉污染环境
2. **金属络合剂驱镉**：使用络合剂促进镉排出
3. **对症支持治疗**：保护肾功能和骨骼健康
4. **营养支持**：补充钙、维生素D

### 金属络合剂

#### 一、依地酸钙钠（CaNa₂EDTA）

| 项目 | 内容 |
|------|------|
| 特点 | 广谱金属络合剂，对镉有一定驱排作用 |
| 用法 | 1g/d，静脉滴注或肌肉注射 |
| 疗程 | 3-5天，间隔3-4天，2-3个疗程 |
| 不良反应 | 肾损害，需监测肾功能 |
| 注意 | 应在医生指导下使用 |

#### 二、二巯基丙磺酸钠（Na-DMPS）

| 项目 | 内容 |
|------|------|
| 特点 | 对重金属均有驱排作用 |
| 用法 | 5mg/kg/d，分2次肌肉注射 |
| 疗程 | 3-5天，间隔2-3天 |
| 不良反应 | 头晕、恶心、皮疹 |

#### 三、二巯基丁二酸（DMSA）

| 项目 | 内容 |
|------|------|
| 特点 | 口服制剂，依从性好 |
| 用法 | 成人1g/d，分2-4次口服 |
| 疗程 | 5-7天，间隔2-3天，2-3个疗程 |
| 不良反应 | 胃肠道反应、肝功能异常 |

### 注意事项

1. **络合剂并非首选**：镉在体内分布复杂，络合剂效果有限
2. **保护肾功能**：络合剂可能加重肾损害
3. **综合治疗**：以对症支持治疗为主
4. **长期管理**：镉中毒需长期随访

### 对症支持治疗

#### 一、肾损害治疗

- **低蛋白饮食**：减轻肾脏负担
- **ACEI/ARB**：保护肾功能
- **纠正电解质**：补钙、补磷
- **透析**：严重肾衰竭时

#### 二、骨损害治疗

- **钙剂**：碳酸钙、葡萄糖酸钙
- **维生素D**：骨化三醇、阿尔法骨化醇
- **双膦酸盐**：抑制骨吸收
- **降钙素**：缓解骨痛

#### 三、金属烟热处理

- **退热**：物理降温或药物退热
- **镇痛**：缓解肌肉酸痛
- **补液**：纠正脱水
- **休息**：对症处理后多可自愈

## 劳动能力鉴定

### 鉴定标准

根据《劳动能力鉴定 职工工伤与职业病致残等级》（GB/T 16180）：

| 伤残等级 | 鉴定标准 |
|----------|----------|
| 四级 | 慢性镉中毒合并肾衰竭，需要透析 |
| 五级 | 慢性镉中毒合并严重肾损害，尿β2-MG>5000μg/g |
| 六级 | 慢性镉中毒合并中度肾损害，骨密度显著降低 |

### 鉴定要点

- 肾小球滤过率（GFR）水平
- 尿蛋白、尿β2-MG定量
- 骨密度检查结果
- 是否需要长期治疗

## 预防措施

### 工程控制

- **湿式作业**：减少镉粉尘飞扬
- **通风排毒**：安装局部通风设施
- **自动化生产**：减少人工操作
- **监测检测**：定期检测空气中镉浓度

### 个人防护

- 防尘口罩（过滤效率≥95%）
- 防护服和防护手套
- 作业后沐浴更衣
- 严禁在车间进食、吸烟

### 健康监护

- **上岗前体检**：筛选肾病、骨病
- **定期在岗体检**：每年1次，血镉、尿镉、尿β2-MG检测
- **离岗体检**：镉负荷评估

## 总结

职业性镉中毒的诊断需要综合职业接触史、临床表现和实验室检查。金属络合剂治疗是重要的驱镉手段，但需在医生指导下使用。慢性镉中毒以肾损害和骨损害为主，需要长期管理。用人单位应落实职业卫生防护措施，预防镉中毒的发生。
