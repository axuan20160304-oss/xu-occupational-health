## 一、适用范围

本指南适用于从事铅作业的用人单位及劳动者，主要涉及以下行业：

### 1.1 主要行业

- **蓄电池制造**：铅蓄电池生产、回收
- **电子制造**：PCB焊接、电子元件
- **冶金工业**：铅冶炼、铸造
- **有色金属**：铅矿开采、加工
- **化工行业**：铅化合物生产
- **电缆行业**：铅护套电缆
- **辐射防护**：铅板、铅衣制作
- **弹药制造**：子弹、炮弹生产

### 1.2 铅作业岗位

| 岗位 | 铅暴露形式 |
|------|-------------|
| 冶炼工 | 铅烟、铅尘 |
| 焊接工 | 铅烟 |
| 涂布工 | 铅尘 |
| 熔铅工 | 铅烟 |
| 回收工 | 铅尘、铅烟 |

## 二、铅的基本特性

### 2.1 物理特性

| 特性 | 说明 |
|------|------|
| 外观 | 灰蓝色金属 |
| 熔点 | 327.4℃ |
| 沸点 | 1740℃ |
| 密度 | 11.34 g/cm³ |
| 延展性 | 良好，可轧成薄片 |

### 2.2 毒性特点

- **主要化合物**：铅烟、铅尘、铅化合物
- **毒性**：重金属毒，蓄积性
- **侵入途径**：呼吸道、消化道
- **靶器官**：神经系统、血液系统、肾脏
- **中毒表现**：神经衰弱、贫血、肾功能损害

### 2.3 毒代动力学

| 过程 | 特点 |
|------|------|
| 吸收 | 呼吸道40-50%，消化道10% |
| 分布 | 血液→软组织→骨骼 |
| 代谢 | 不代谢，蓄积体内 |
| 排泄 | 粪便、尿液，缓慢 |

## 三、职业健康检查

### 3.1 上岗前检查（GBZ 188-2025）

从事铅作业的劳动者应进行上岗前职业健康检查，以下疾病或情况为职业禁忌症：

**绝对禁忌症：**

| 系统 | 疾病 |
|------|------|
| 神经系统 | 周围神经病、神经衰弱综合征 |
| 血液系统 | 贫血、血小板减少 |
| 肝脏 | 慢性肝炎、肝硬化 |
| 肾脏 | 慢性肾炎、肾功能不全 |
| 精神 | 精神性疾病 |

**相对禁忌症：**

- 妊娠期、哺乳期妇女
- 严重口腔疾病

### 3.2 在岗检查

| 检查周期 | 要求 |
|----------|------|
| 铅烟作业 | 每年1次 |
| 铅尘作业 | 每年1次 |

**必检项目：**

1. **症状询问**
   - 头晕、头痛、乏力
   - 失眠、记忆力减退
   - 食欲不振、恶心
   - 四肢麻木、疼痛
   - 腹痛、便秘

2. **体格检查**
   - 神经系统检查
   - 腹部检查（肝脾）
   - 齿龈检查（铅线）

3. **辅助检查**
   - 血常规（重点：血红蛋白）
   - 尿铅
   - 血铅
   - 肝功能
   - 肾功能

### 3.3 离岗检查

劳动者离开铅作业岗位时，应进行离岗职业健康检查。

### 3.4 应急检查

发生铅中毒事故时，应对暴露人群进行应急检查。

## 四、职业病诊断

### 4.1 诊断标准（GBZ 37-2015）

**诊断原则：**

1. 职业接触史（半年以上）
2. 尿铅≥0.12mg/L（或0.24μmol/mol肌酐）
3. 出现神经系统、消化系统症状
4. 排除其他疾病

### 4.2 诊断分期

| 分期 | 诊断标准 |
|------|----------|
| **观察对象** | 尿铅0.12-0.24mg/L，无明显症状 |
| **轻度中毒** | 尿铅≥0.24mg/L，神经衰弱综合征 |
| **中度中毒** | 贫血，腹部绞痛，记忆力明显减退 |
| **重度中毒** | 铅麻痹，中毒性脑病 |

### 4.3 临床表现

#### 4.3.1 神经系统

| 症状 | 表现 |
|------|------|
| 神经衰弱 | 头晕、头痛、失眠、记忆力减退 |
| 周围神经病 | 四肢麻木、疼痛、肌无力 |
| 铅麻痹 | 垂腕、垂足 |

#### 4.3.2 消化系统

| 症状 | 表现 |
|------|------|
| 食欲不振 | 纳差、恶心 |
| 腹痛 | 脐周绞痛，便秘 |
| 铅线 | 齿龈灰蓝色线 |

#### 4.3.3 血液系统

| 指标 | 表现 |
|------|------|
| 血红蛋白 | 贫血 |
| 点彩红细胞 | 增多 |
| 网织红细胞 | 增多 |

## 五、接触限值

### 5.1 工作场所空气中铅浓度限值

| 标准 | PC-TWA | MAC |
|------|---------|-----|
| 铅烟 | 0.05 mg/m³ | 0.15 mg/m³ |
| 铅尘 | 0.05 mg/m³ | 0.15 mg/m³ |

### 5.2 生物监测指标

| 指标 | 正常值 | 异常值 |
|------|---------|---------|
| 尿铅 | <0.12 mg/L | ≥0.12 mg/L |
| 血铅 | <0.21 μmol/L | ≥0.21 μmol/L |

### 5.3 监测要求

| 监测类型 | 频次 |
|----------|------|
| 日常监测 | 每月至少1次 |
| 定期监测 | 每年至少1次 |
| 个体监测 | 接触铅作业者 |

## 六、预防控制措施

### 6.1 工程控制措施

| 措施 | 说明 |
|------|------|
| **湿式作业** | 降低铅尘浓度 |
| **通风排毒** | 局部通风 |
| **自动化** | 减少人工操作 |
| **替代工艺** | 用无铅工艺替代 |

### 6.2 防护设施

| 设施 | 说明 |
|------|------|
| 通风罩 | 铅烟收集 |
| 过滤装置 | 铅尘过滤 |
| 隔离操作 | 远程控制 |

### 6.3 个人防护

| 防护用品 | 要求 |
|----------|------|
| **防尘口罩** | 过滤效率≥95% |
| **防护服** | 防铅尘渗透 |
| **防护手套** | 耐铅手套 |
| **防护眼镜** | 防护眼镜 |

### 6.4 卫生措施

- 班后沐浴
- 更换工作服
- 禁止在车间进食、吸烟
- 定期体检

## 七、急救处理

### 7.1 急性铅中毒

**症状**：剧烈腹痛、呕吐、铅绞痛

**急救：**

1. 脱离铅作业环境
2. 立即就医
3. 特效药：依地酸钙钠（Na₂CaEDTA）
4. 对症治疗

### 7.2 慢性铅中毒

**处理原则：**

1. 脱离铅接触
2. 驱铅治疗（依地酸钙钠）
3. 对症治疗
4. 营养支持

### 7.3 驱铅治疗

| 药物 | 用法 |
|------|------|
| 依地酸钙钠 | 1g/d，静脉滴注，3天疗程 |
| 二巯丁二钠 | 1g/d，静脉注射 |
