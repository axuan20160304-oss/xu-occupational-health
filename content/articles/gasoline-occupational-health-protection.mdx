## 一、适用范围

本指南适用于从事汽油作业的用人单位及劳动者，主要涉及以下行业：

- **石油化工**：炼油、油品储运
- **机械制造**：金属清洗、机修
- **交通运输**：机动车维修、加油站
- **印刷包装**：油墨稀释、清洗
- **其他**：橡胶、制鞋、纺织

## 二、汽油的基本特性

### 2.1 物理特性

| 特性 | 说明 |
|------|------|
| 外观 | 无色或淡黄色液体 |
| 气味 | 特殊汽油味 |
| 沸点 | 40-200℃ |
| 闪点 | -50℃（极易燃） |
| 挥发性 | 极易挥发 |

### 2.2 毒性特点

- **主要成分**：烷烃、环烷烃、芳香烃
- **毒性**：中等毒性，主要损害神经系统
- **侵入途径**：呼吸道、皮肤、消化道
- **代谢**：经肝脏代谢，肾脏排出

## 三、职业健康检查

### 3.1 上岗前检查

从事汽油作业的劳动者应进行上岗前职业健康检查，以下疾病或情况为职业禁忌症：

**禁忌症：**

- **神经系统**：多发性周围神经病、神经衰弱综合征
- **血液系统**：贫血、血小板减少性疾病
- **肝脏疾病**：慢性肝炎、肝硬化
- **肾脏疾病**：慢性肾炎、肾功能不全
- **皮肤病**：慢性湿疹、银屑病
- **心血管疾病**：冠心病、心律失常

### 3.2 在岗检查

| 检查周期 | 要求 |
|----------|------|
| 接触汽油浓度≥PC-TWA | 每年1次 |
| 接触汽油浓度＜PC-TWA | 每2年1次 |

**检查项目：**

1. **症状询问**
   - 头晕、头痛、乏力
   - 记忆力减退、失眠
   - 恶心、食欲不振
   - 手足麻木、疼痛

2. **体格检查**
   - 神经系统检查
   - 肝脾触诊
   - 皮肤检查

3. **辅助检查**
   - 血常规、尿常规
   - 肝功能（ALT、AST）
   - 肾功能（肌酐、尿素氮）
   - 神经肌电图（必要时）

### 3.3 离岗检查

劳动者离开汽油作业岗位时，应进行离岗职业健康检查。

## 四、职业病诊断

### 4.1 诊断标准

依据GBZ 27-2013《职业性汽油中毒诊断标准》：

**诊断原则：**

1. 短期内吸入较高浓度汽油蒸气或长期密切接触汽油史
2. 出现中枢神经系统损害为主的临床表现
3. 排除其他疾病
4. 结合职业卫生调查资料

### 4.2 诊断分期

| 分期 | 主要表现 |
|------|----------|
| **观察对象** | 神经衰弱综合征，症状轻微 |
| **轻度中毒** | 头晕、头痛、恶心、呕吐、步态不稳、嗜睡 |
| **中度中毒** | 上述症状加重，出现精神障碍、抽搐、昏迷 |
| **重度中毒** | 意识障碍、反射消失、脑水肿、肺水肿 |

### 4.3 慢性汽油中毒

长期吸入汽油蒸气可导致慢性中毒：

- **神经系统**：多发性周围神经病、神经衰弱
- **血液系统**：贫血、白细胞减少
- **肝脏损害**：中毒性肝病
- **肾脏损害**：蛋白尿、血尿

## 五、接触限值

### 5.1 工作场所空气中汽油浓度限值

| 标准 | PC-TWA | PC-STEL |
|------|--------|---------|
| GBZ 2.1-2007 | 300 mg/m³ | 450 mg/m³ |

### 5.2 监测要求

- **日常监测**：每月至少监测1次
- **定期监测**：每年至少进行1次全面监测
- **个体监测**：接触浓度≥50%PC-TWA者

## 六、预防控制措施

### 6.1 工程控制措施

| 措施 | 说明 |
|------|------|
| **密闭化** | 生产设备密闭化，减少泄漏 |
| **通风** | 设置全面通风和局部通风 |
| **替代** | 用低毒性溶剂替代汽油 |
| **隔离** | 隔离操作，减少人员接触 |

### 6.2 管理措施

- **危害告知**：告知劳动者汽油危害及防护措施
- **培训教育**：开展汽油防护知识培训
- **健康监护**：严格执行职业健康检查制度
- **应急处理**：制定汽油中毒应急预案

### 6.3 个人防护

| 防护用品 | 要求 |
|----------|------|
| **防毒面具** | 活性炭滤罐或供气式 |
| **防护手套** | 耐油手套（如丁腈橡胶） |
| **防护服** | 防静电工作服 |
| **防护眼镜** | 防护眼镜或防护面罩 |

### 6.4 作业环境要求

- 工作场所汽油浓度≤PC-TWA
- 设置警示标识
- 配置急救设备（洗眼器、冲洗设备）
- 禁止明火作业

## 七、急救处理

### 7.1 急性汽油中毒急救

**吸入中毒：**

1. 迅速脱离现场，移至空气新鲜处
2. 保持呼吸道通畅
3. 吸氧
4. 对症处理
5. 及时送医

**皮肤接触：**

1. 立即脱去被污染衣物
2. 用肥皂水或清水彻底冲洗
3. 皮肤破损处按化学烧伤处理

**眼睛接触：**

1. 立即用流动清水或生理盐水冲洗
2. 冲洗时间≥15分钟
3. 及时就医

### 7.2 现场急救箱配置

| 物品 | 用途 |
|------|------|
| 活性炭口罩 | 呼吸防护 |
| 洗眼液 | 眼睛冲洗 |
| 棉纱手套 | 手部防护 |
| 急救手册 | 指导使用 |
