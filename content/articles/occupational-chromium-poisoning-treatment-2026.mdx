# 职业性铬中毒的诊断标准与解毒治疗

## 概述

职业性铬中毒（Occupational Chromium Poisoning）是指劳动者在职业活动中因接触铬及其化合物而引起的中毒性疾病。铬是一种银灰色金属，广泛应用于电镀、制革、冶金、化工、颜料等行业。《职业病分类和目录》将铬及其化合物中毒列为法定职业病。

铬中毒的特点：
- **六价铬毒性高**：六价铬（Cr⁶⁺）毒性远高于三价铬（Cr³⁺）
- **皮肤损害突出**：可引起皮肤溃疡和过敏性皮炎
- **致癌性**：六价铬是已确认的人类致癌物

## 中毒机制

### 毒理学特点

铬（Cr）原子量52.00，常见价态为三价（Cr³⁺）和六价（Cr⁶⁺）。六价铬化合物毒性高，常见的有铬酸钾、铬酸钠、重铬酸钾、重铬酸钠等。

### 病理生理

1. **氧化损伤**：六价铬在体内被还原为三价铬，产生自由基
2. **DNA损伤**：六价铬可引起DNA链断裂和交叉连接
3. **蛋白质结合**：铬与蛋白质结合引起组织损伤
4. **致癌作用**：六价铬被列为人类致癌物

### 毒性分级

| 尿铬含量（μg/L） | 血铬含量（μg/L） | 病情程度 |
|-------------------|-------------------|----------|
| <5 | <2 | 正常范围 |
| 5-10 | 2-5 | 轻度铬负荷 |
| 10-20 | 5-10 | 中度铬负荷 |
| >20 | >10 | 铬中毒 |

## 诊断标准

### 诊断原则

根据《GBZ 12-2015 职业性铬鼻病的诊断》和《GBZ 106-2016 职业性皮肤病的诊断》，诊断需满足以下条件：

1. **明确的铬接触史**：有确切的职业性六价铬接触史
2. **符合临床表现**：出现鼻黏膜损害、皮肤损害等症状
3. **实验室检查异常**：尿铬、血铬含量增高
4. **排除其他疾病**：排除其他原因引起的类似疾病

### 诊断依据

#### 一、职业接触史调查

| 行业 | 接触铬化物 | 常见工种 |
|------|------------|----------|
| 电镀 | 铬酸、铬酸盐 | 电镀工 |
| 制革 | 铬鞣剂 | 皮革工 |
| 冶金 | 铬铁、铬矿 | 冶炼工 |
| 化工 | 重铬酸盐 | 化工操作工 |
| 颜料 | 铬黄、铬绿 | 颜料制作工 |

#### 二、临床表现

##### 急性铬中毒

- **消化系统**：恶心、呕吐、腹痛、腹泻（有时便血）
- **神经系统**：头痛、眩晕、意识障碍
- **肾脏**：蛋白尿、血尿，严重时急性肾衰竭

##### 慢性铬中毒

- **鼻黏膜损害**：鼻中隔软骨部穿孔、萎缩性鼻炎
- **皮肤损害**：
  - 铬溃疡（"鸡眼状"溃疡）
  - 过敏性皮炎（红斑、丘疹、水疱）
  - 色素沉着
- **呼吸道**：咳嗽、咳痰、支气管炎
- **眼睛**：结膜炎、角膜炎

#### 三、实验室检查

1. **尿铬测定**：尿铬>5μg/L提示铬负荷增加
2. **血铬测定**：血铬>2μg/L有诊断意义
3. **血常规**：白细胞减少
4. **肝肾功能**：转氨酶升高、蛋白尿
5. **鼻内镜检查**：观察鼻黏膜病变

### 诊断分级

#### 职业性铬鼻病

| 分级 | 临床表现 |
|------|----------|
| 轻度 | 鼻中隔黏膜糜烂、充血 |
| 中度 | 鼻中隔溃疡、穿孔 |
| 重度 | 鼻部结构明显破坏 |

#### 职业性铬皮炎

| 分级 | 临床表现 |
|------|----------|
| 轻度 | 红斑、丘疹，面积<体表面积3% |
| 中度 | 明显红斑、水疱，面积3%-10% |
| 重度 | 大疱、糜烂，面积>10% |

## 鉴别诊断

### 需要鉴别的疾病

1. **普通鼻炎**：过敏性鼻炎、萎缩性鼻炎
2. **皮肤溃疡**：外伤性溃疡、臁疮
3. **过敏性皮炎**：湿疹、荨麻疹
4. **鼻部肿瘤**：鼻腔癌、鼻窦癌

### 鉴别要点

| 疾病 | 关键鉴别特征 |
|------|--------------|
| 过敏性鼻炎 | 无铬接触史，嗜酸粒细胞增多，特异性IgE阳性 |
| 湿疹 | 无铬接触史，无特定好发部位 |
| 鼻腔癌 | 影像学肿块，病理确诊 |

## 解毒治疗

### 治疗原则

1. **立即终止接触**：脱离铬污染环境
2. **特效解毒**：使用金属络合剂驱铬
3. **对症支持治疗**：保护肝肾功能和皮肤修复
4. **局部处理**：皮肤和鼻黏膜病变的局部治疗

### 金属络合剂

#### 一、维生素C

| 项目 | 内容 |
|------|------|
| 特点 | 还原剂，将六价铬还原为低毒三价铬 |
| 用法 | 1-3g/d，静脉滴注或口服 |
| 疗程 | 7-14天 |
| 注意事项 | 大剂量使用需监测肾功能 |

#### 二、依地酸钙钠（CaNa₂EDTA）

| 项目 | 内容 |
|------|------|
| 特点 | 广谱金属络合剂，对铬有一定驱排作用 |
| 用法 | 1g/d，静脉滴注 |
| 疗程 | 3-5天，间隔2-3天 |
| 不良反应 | 肾损害，需监测肾功能 |

#### 三、二巯基丙磺酸钠（Na-DMPS）

| 项目 | 内容 |
|------|------|
| 特点 | 对重金属均有驱排作用 |
| 用法 | 5mg/kg/d，分2次肌肉注射 |
| 疗程 | 3-5天，间隔2-3天 |

### 对症支持治疗

#### 一、皮肤损害

- **铬溃疡**：先用肥皂水冲洗，再外用维生素C溶液湿敷
- **过敏性皮炎**：外用糖皮质激素软膏，口服抗组胺药
- **色素沉着**：可自行消退，无需特殊处理

#### 二、鼻黏膜损害

- **鼻中隔穿孔**：避免抠鼻，保持鼻腔湿润
- **萎缩性鼻炎**：复方薄荷油滴鼻
- **继发感染**：抗生素治疗

#### 三、金属烟热

- **退热**：物理降温或药物退热
- **镇痛**：缓解肌肉酸痛
- **休息**：对症处理后多可自愈

### 职业性铬鼻病治疗

1. **鼻腔冲洗**：生理盐水冲洗鼻腔
2. **局部用药**：复方薄荷油、维生素A油
3. **鼻中隔穿孔**：必要时行修补手术
4. **避免再次接触**：治愈后不应再从事铬作业

## 劳动能力鉴定

### 鉴定标准

根据《劳动能力鉴定 职工工伤与职业病致残等级》（GB/T 16180）：

| 伤残等级 | 鉴定标准 |
|----------|----------|
| 六级 | 职业性铬鼻病，鼻中隔穿孔 |
| 七级 | 职业性铬皮炎，反复发作 |
| 八级 | 职业性铬鼻病，轻度鼻黏膜病变 |

### 鉴定要点

- 鼻中隔穿孔大小和位置
- 皮肤损害面积和程度
- 脱离接触后症状是否持续

## 预防措施

### 工程控制

- **替代技术**：采用三价铬电镀工艺替代六价铬
- **通风排毒**：安装局部通风设施
- **湿式作业**：减少铬粉尘飞扬
- **监测检测**：定期检测空气中铬浓度

### 个人防护

- 防护口罩（防铬尘口罩）
- 防护服和防护手套
- 防护眼镜
- 作业后沐浴更衣

### 健康监护

- **上岗前体检**：筛选鼻炎、皮肤病
- **定期在岗体检**：每年1次，尿铬检测，鼻部检查
- **皮肤检查**：发现皮疹及时处理

## 总结

职业性铬中毒的诊断需要综合职业接触史、临床表现和实验室检查。解毒治疗以金属络合剂为主，同时需重视皮肤和鼻黏膜病变的局部治疗。六价铬是确认的人类致癌物，用人单位应优先采用替代技术，预防铬中毒的发生。
