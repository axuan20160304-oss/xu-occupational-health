# 职业性尘肺病的预防措施与粉尘控制技术

## 概述

职业性尘肺病（Occupational Pneumoconiosis）是指劳动者在职业活动中长期吸入生产性粉尘，并在肺内潴留引起的以肺组织弥漫性纤维化为主的全身性疾病。《职业病分类和目录》将尘肺病列为法定职业病，是我国发病人数最多的职业病。

根据致病粉尘类型，尘肺病可分为：
- 矽肺（游离二氧化硅粉尘）
- 煤工尘肺（煤尘）
- 石棉肺（石棉粉尘）
- 水泥尘肺（水泥粉尘）
- 陶工尘肺（陶瓷原料粉尘）
- 铸工尘肺（铸造粉尘）
- 电焊工尘肺（电焊烟尘）

## 粉尘危害

### 粉尘分类

| 分类标准 | 类型 | 示例 |
|----------|------|------|
| 按成分 | 无机粉尘 | 矽尘、煤尘、石棉 |
| | 有机粉尘 | 棉尘、木尘、面粉 |
| | 混合粉尘 | 铸造尘、焊烟 |
| 按粒径 | 呼吸性粉尘 | <10μm，可进入肺泡 |
| | 可吸入粉尘 | <15μm，可进入呼吸道 |
| 按游离二氧化硅 | 矽尘 | SiO₂含量>10% |
| | 非矽尘 | SiO₂含量≤10% |

### 粉尘致病性

- **粒径**：呼吸性粉尘危害最大
- **浓度**：浓度越高，致病性越强
- **游离二氧化硅含量**：含量越高，致病性越强
- **暴露时间**：接尘时间越长，发病风险越高

## 粉尘浓度标准

### 国家标准

根据《GBZ 2.1-2019 工作场所有害因素职业接触限值》：

| 粉尘类型 | 时间加权平均浓度（mg/m³） |
|----------|--------------------------|
| 矽尘（10%≤SiO₂≤50%） | 1 |
| 矽尘（50%<SiO₂≤80%） | 0.7 |
| 矽尘（SiO₂>80%） | 0.5 |
| 煤尘 | 4 |
| 水泥粉尘（游离SiO₂<10%） | 4 |
| 石棉纤维（根/厘米） | 0.8 |

## 工程控制技术

### 一、湿式作业

#### 原理
通过加水抑制粉尘产生，降低空气中粉尘浓度

#### 方法
- 湿式凿岩
- 喷雾洒水
- 湿法搅拌
- 湿式切割

#### 适用场景
- 矿山开采
- 岩石破碎
- 陶瓷加工
- 水泥制造

### 二、通风除尘

#### 局部通风
- 吸气罩：设置在粉尘源附近
- 风管：输送含尘气体
- 除尘器：分离粉尘
- 风机：提供动力

#### 全面通风
- 新风输入：稀释空气中粉尘
- 排风系统：排出含尘空气

#### 除尘器类型

| 类型 | 原理 | 适用粒径 |
|------|------|----------|
| 布袋除尘 | 过滤 | >0.1μm |
| 静电除尘 | 静电吸附 | >0.01μm |
| 旋风除尘 | 离心分离 | >5μm |
| 湿式除尘 | 液体捕集 | >1μm |

### 三、密闭抽尘

#### 原理
将粉尘源密闭，通过负压抽吸防止粉尘外逸

#### 方法
- 密闭容器
- 密闭管道
- 自动化操作
- 负压控制

### 四、替代技术

#### 方法
- 采用无粉尘或少粉尘的材料
- 改进工艺，减少粉尘产生
- 使用低二氧化硅含量的材料

#### 示例
- 铸造用钢丸替代石英砂
- 湿法生产工艺替代干法

## 个人防护

### 一、呼吸防护

#### 防尘口罩类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| KN95 | 过滤效率95% | 一般粉尘作业 |
| KN100 | 过滤效率99.97% | 高浓度粉尘 |
| 全面罩 | 防护面部 | 特殊作业 |
| 电动送风 | 防护效果好 | 长时间作业 |

#### 选型原则

1. **过滤效率**：根据粉尘浓度选择
2. **适合性**：适合佩戴者脸型
3. **舒适性**：长时间佩戴舒适
4. **维护性**：便于清洁和更换

#### 使用注意事项

1. 正确佩戴，确保密封
2. 及时更换滤料
3. 保持清洁
4. 定期检查

### 二、其他防护

- 防尘服：防止皮肤接触粉尘
- 防尘帽：防止头发沾染粉尘
- 防护眼镜：防止眼部损害

## 健康监护

### 上岗前体检

#### 目的
- 筛选禁忌证
- 建立健康基线
- 健康教育

#### 禁忌证
- 活动性肺结核
- 慢性阻塞性肺疾病
- 严重心血管疾病

### 在岗体检

#### 体检周期

| 粉尘类型 | 体检周期 |
|----------|----------|
| 矽尘 | 每年1次 |
| 煤尘 | 每年1次 |
| 石棉 | 每年1次 |
| 其他粉尘 | 每2年1次 |

#### 体检项目
1. 病史采集
2. 体格检查
3. 胸部X射线
4. 肺功能检查

#### 异常处理

| 检查结果 | 处理措施 |
|----------|----------|
| 疑似尘肺 | 进一步诊断 |
| 小阴影密集度增加 | 调离粉尘作业 |
| 肺功能明显下降 | 调离粉尘作业 |

### 离岗体检

- 全面健康检查
- 职业病诊断（如符合条件）
- 健康评估

## 用人单位职责

### 管理措施

1. **粉尘监测**：定期检测工作场所粉尘浓度
2. **工程改造**：采取防尘措施
3. **个人防护**：配备合格防尘用品
4. **健康监护**：组织定期体检
5. **培训教育**：普及防尘知识
6. **警示标识**：设置粉尘危害告知卡

### 档案管理

- 粉尘监测报告
- 职业健康检查档案
- 个人健康档案
- 防尘用品发放记录

## 劳动者权利

### 知情权
- 粉尘危害及后果
- 防护措施
- 健康检查安排

### 建议权
- 改善工作环境
- 改进防护措施

### 拒绝权
- 拒绝违章指挥
- 拒绝无防护作业

### 职业病待遇
- 职业病诊断
- 工伤认定
- 相应待遇

## 粉尘监测

### 监测方法

1. **定点采样**：在工作场所设置采样点
2. **个体采样**：佩戴个体采样器
3. **连续监测**：安装粉尘在线监测仪

### 监测周期

| 情况 | 监测周期 |
|------|----------|
| 新建项目 | 验收时监测 |
| 正常生产 | 每年至少1次 |
| 粉尘浓度超标 | 立即监测 |
| 设备变更 | 变更后监测 |

### 评价标准

- **达标**：浓度≤职业接触限值
- **超标**：浓度>职业接触限值，需采取整改措施

## 总结

职业性尘肺病的预防需要采取综合措施：工程控制是根本，个人防护是关键，健康监护是保障。用人单位应落实粉尘防护主体责任，劳动者应正确使用防尘用品，双方共同努力预防尘肺病的发生。
