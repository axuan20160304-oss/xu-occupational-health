# 职业性皮肤病：化工业劳动者的皮肤健康威胁

## 引言

**职业性皮肤病**（Occupational Dermatoses）是指劳动者在职业活动中因接触职业病危害因素引起的皮肤及其附属器官的疾病。职业性皮肤病是常见的职业病之一，在化工、制造、印刷、电子等行业中发病率较高。这种疾病不仅影响劳动者的皮肤健康，严重者可影响工作能力和生活质量。

根据流行病学调查，职业性皮肤病约占全部职业病的20%-30%，是职业病的重要组成部分。常见的职业性皮肤病包括职业性皮炎、职业性痤疮、职业性皮肤色素变化、职业性指甲损害等。本文将从专业角度全面介绍职业性皮肤病的相关知识，帮助广大劳动者认识这一职业健康威胁。

## 一、病因与分类

### 1.1 病因

**（一）化学因素**

| 类别 | 物质举例 | 致病特点 |
|------|----------|----------|
| 酸类 | 硫酸、盐酸、硝酸 | 腐蚀性 |
| 碱类 | 氢氧化钠、氨水 | 刺激性 |
| 溶剂 | 苯、甲苯、正己烷 | 脱脂作用 |
| 金属 | 铬、镍、汞 | 致敏性 |
| 染料 | 偶氮染料、蒽醌 | 致敏性 |
| 树脂 | 环氧树脂、酚醛树脂 | 致敏性 |

**（二）物理因素**

- 摩擦：反复摩擦导致皮肤损伤
- 压力：长期压迫引起溃疡
- 热：高温导致烫伤
- 冷：低温导致冻疮
- 辐射：紫外线导致光敏性皮炎

**（三）生物因素**

- 细菌：引起皮肤感染
- 真菌：引起手癣、足癣
- 病毒：引起疣、疱疹
- 寄生虫：引起虱病、螨虫皮炎

### 1.2 分类

**（一）职业性皮炎**

- 刺激性接触性皮炎
- 过敏性接触性皮炎
- 光感性皮炎

**（二）职业性痤疮**

- 油性痤疮
- 氯痤疮

**（三）职业性皮肤色素变化**

- 色素沉着
- 色素减退

**（四）职业性皮肤溃疡**

- 化学性溃疡
- 压力性溃疡

**（五）其他**

- 毛发改变
- 指甲改变
- 疣赘

## 二、发病机制

### 2.1 刺激性接触性皮炎

**（一）发病机制**

1. 直接损伤
   - 化学物质破坏皮肤屏障
   - 引起炎症反应
   - 导致细胞坏死

2. 炎症反应
   - 释放炎性介质
   - 引起血管扩张
   - 出现红斑、水肿

**（二）特点**

- 发病急
- 症状明显
- 脱离后恢复较快

### 2.2 过敏性接触性皮炎

**（一）发病机制**

1. 致敏阶段
   - 过敏原穿透皮肤
   - 被朗格汉斯细胞摄取
   - 激活T淋巴细胞

2. 反应阶段
   - 再次接触过敏原
   - 引起迟发型超敏反应
   - 导致皮炎

**（二）特点**

- 有潜伏期
- 再次接触后发病
- 症状较顽固

### 2.3 职业性痤疮

**（一）发病机制**

1. 毛囊口角化过度
2. 皮脂分泌旺盛
3. 痤疮丙酸杆菌繁殖
4. 炎症反应

**（二）特点**

- 长期接触后发生
- 症状持续
- 脱离后恢复较慢

## 三、临床表现

### 3.1 职业性皮炎

**（一）刺激性接触性皮炎**

- 红斑：境界清楚
- 水肿：程度不一
- 水疱：大疱或小疱
- 糜烂：渗出性
- 瘙痒：程度轻重不一
- 灼痛：接触部位

**（二）过敏性接触性皮炎**

- 红斑：边缘不清
- 丘疹：散在或密集
- 水疱：可融合
- 渗出：慢性期减轻
- 瘙痒：剧烈
- 反复发作

**（三）光感性皮炎**

- 日光暴露部位发病
- 皮疹类似湿疹
- 春夏加重
- 冬季减轻

### 3.2 职业性痤疮

**（一）症状**

- 毛囊性丘疹
- 脓疱
- 囊肿
- 结节
- 瘢痕

**（二）部位**

- 面部
- 胸部
- 背部
- 上臂

### 3.3 职业性皮肤色素变化

**（一）色素沉着**

- 接触部位颜色加深
- 弥漫性或片状
- 常见于铬、苯酚作业

**（二）色素减退**

- 白斑样改变
- 常见于苯酚作业

### 3.4 职业性皮肤溃疡

**（一）症状**

- 创面坏死
- 边缘清楚
- 疼痛明显
- 愈合缓慢

**（二）常见于**

- 铬酸作业
- 氢氟酸作业

## 四、诊断与鉴别诊断

### 4.1 诊断标准

根据GBZ 18-2013《职业性皮肤病的诊断》：

**诊断依据：**

1. 职业接触史
2. 临床表现
3. 皮肤斑贴试验（过敏性皮炎）
4. 排除非职业性疾病

### 4.2 皮肤斑贴试验

**（一）方法**

- 将可疑过敏原贴于皮肤
- 48小时后观察
- 72小时再次观察

**（二）结果判断**

| 反应 | 表现 |
|------|------|
| 阴性 | 无反应 |
| 可疑 | 轻度红斑 |
| 阳性 | 红斑、浸润、水疱 |

### 4.3 鉴别诊断

需要与以下疾病鉴别：

| 疾病 | 鉴别要点 |
|------|----------|
| 湿疹 | 无职业接触史 |
| 脂溢性皮炎 | 部位特点 |
| 寻常痤疮 | 年龄、病史 |
| 银屑病 | 皮损特点 |

## 五、治疗方案

### 5.1 治疗原则

1. **立即脱离接触**
2. **清除皮肤残留物**
3. **对症治疗**
4. **防止感染**

### 5.2 治疗方法

**（一）急性皮炎**

| 药物 | 作用 | 用法 |
|------|------|------|
| 硼酸溶液 | 收敛、消炎 | 湿敷 |
| 生理盐水 | 清洁 | 冲洗 |
| 糖皮质激素 | 抗炎 | 外用/口服 |
| 抗组胺药 | 止痒 | 口服 |

**（二）慢性皮炎**

- 外用糖皮质激素软膏
- 尿素软膏保湿
- 避免刺激

**（三）职业性痤疮**

- 维A酸类药物
- 抗生素
- 对症处理

**（四）皮肤溃疡**

- 清创处理
- 湿润疗法
- 必要时植皮

### 5.3 注意事项

1. 避免水过热
2. 避免刺激性食物
3. 保持皮肤清洁
4. 避免再次接触

## 六、预防措施

### 6.1 工程控制

**（一）工艺改革**

- 采用低毒替代品
- 自动化生产
- 减少人工操作

**（二）防护设施**

- 安装通风设备
- 设置防护屏障
- 保持工作场所清洁

### 6.2 个人防护

**（一）防护手套**

| 类型 | 适用情况 |
|------|----------|
| 乳胶手套 | 一般化学品 |
| 丁腈手套 | 有机溶剂 |
| 橡胶手套 | 强酸碱 |

**（二）防护服**

- 防渗透材质
- 定期更换
- 保持清洁

**（三）护肤品**

- 护肤霜
- 遮光剂
- 按需使用

### 6.3 卫生措施

1. 作业后立即洗手
2. 不在车间进食饮水
3. 定期更换工作服
4. 保持皮肤清洁

### 6.4 健康监护

**（一）就业前体检**

- 排除皮肤疾病
- 过敏体质评估

**（二）在岗期间体检**

- 定期皮肤检查
- 询问皮肤症状

## 七、特定行业防护

### 7.1 化工行业

- 重点防护酸碱
- 做好通风
- 穿戴防护服

### 7.2 电镀行业

- 重点防护铬、镍
- 防护手套必戴
- 作业后彻底清洗

### 7.3 印刷行业

- 重点防护溶剂
- 保持通风
- 避免皮肤直接接触

### 7.4 电子行业

- 重点防护焊锡
- 防护手套
- 作业环境清洁

## 八、法律保护与权益

### 8.1 职业病诊断

- 诊断机构：职业病诊断机构
- 诊断依据：GBZ 18-2013

### 8.2 工伤认定

- 属于职业病
- 认定为工伤
- 享受待遇

### 8.3 用人单位责任

- 提供防护设施
- 组织健康检查
- 落实防护措施

### 8.4 劳动者权利

- 知情权
- 防护权
- 拒绝违章权

## 九、总结

职业性皮肤病是完全可以预防的职业病。

**核心要点：**

1. 化学因素是最常见病因
2. 刺激性皮炎与过敏性皮炎不同
3. 早期发现很重要
4. 预防重于治疗
5. 防护是关键措施

**防护要点：**

- 了解所接触物质危害
- 正确使用防护用品
- 保持皮肤清洁
- 出现症状及时就医

让我们共同关注职业性皮肤病防护，守护劳动者皮肤健康！
