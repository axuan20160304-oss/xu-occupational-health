# 职业性肿瘤：工作环境中的致癌风险

## 引言

**职业性肿瘤**（Occupational Cancer）是指劳动者在职业活动中长期接触职业病致癌因素而引起的恶性肿瘤。职业性肿瘤是一种严重的职业病，其特点是潜伏期长、进展缓慢、发现时往往已到晚期。根据国际癌症研究机构（IARC）的统计，约4%-5%的恶性肿瘤与职业暴露有关。

职业性肿瘤是职业病中最严重的类型之一，一旦发病，不仅治疗困难、预后差，而且给劳动者家庭带来沉重负担。因此，预防职业性肿瘤的发生具有重要意义。

本文将从专业角度全面介绍职业性肿瘤的相关知识，帮助广大劳动者和企业认识这一严重的职业健康问题。

## 一、职业性肿瘤概述

### 1.1 职业性肿瘤的特点

1. **潜伏期长**：通常10-30年
2. **病因明确明确的致癌因素**：有暴露史
3. **部位集中**：好发于特定部位
4. **可预防性**：有效预防可降低发病率

### 1.2 历史与现状

- **历史认识**：18世纪发现扫烟囱工人阴囊癌
- **IARC分类**：已确认100余种职业致癌物
- **我国情况**：法定职业性肿瘤有8种

### 1.3 我国法定职业性肿瘤

根据《职业病分类和目录》，我国法定职业性肿瘤包括：

| 序号 | 肿瘤类型 | 致癌因素 |
|------|----------|----------|
| 1 | 石棉所致肺癌 | 石棉 |
| 2 | 石棉所致间皮瘤 | 石棉 |
| 3 | 苯所致白血病 | 苯 |
| 4 | 联苯胺所致膀胱癌 | 联苯胺 |
| 5 | 氯甲醚所致肺癌 | 氯甲醚 |
| 6 | 砷所致肺癌 | 砷 |
| 7 | 砷所致皮肤癌 | 砷 |
| 8 | 焦炉工人肺癌 | 焦炉逸散物 |

## 二、常见职业致癌因素

### 2.1 物理因素

| 因素 | 致癌类型 | 职业 |
|------|----------|------|
| 石棉 | 肺癌、间皮瘤 | 建材、造船 |
| 氡 | 肺癌 | 矿山 |
| 紫外线 | 皮肤癌 | 户外作业 |
| 电离辐射 | 多种肿瘤 | 核工业、医疗 |

### 2.2 化学因素

**（一）确认致癌物（1类）**

| 物质 | 致癌类型 | 主要行业 |
|------|----------|----------|
| 苯 | 白血病 | 化工、印刷 |
| 石棉 | 肺癌、间皮瘤 | 建材、造船 |
| 联苯胺 | 膀胱癌 | 染料 |
| 砷 | 肺癌、皮肤癌 | 冶炼、农药 |
| 甲醛 | 鼻咽癌 | 家具、纺织 |
| 煤焦油 | 皮肤癌 | 炼焦 |
| 矿物油 | 皮肤癌 | 机械加工 |

**（二）可疑致癌物（2A类）**

- 苯并芘
- 氯甲醚
- 铍
- 镍

### 2.3 生物因素

- **乙型肝炎病毒**：肝细胞癌
- **丙型肝炎病毒**：肝细胞癌
- **人乳头瘤病毒**：宫颈癌
- **EB病毒**：鼻咽癌、淋巴瘤

## 三、发病机制

### 3.1 致癌机制

**（一）DNA损伤**

- 致癌物代谢产物与DNA结合
- 导致基因突变
- 影响DNA修复机制

**（二）基因表达异常**

- 激活原癌基因
- 抑癌基因失活
- 影响细胞凋亡

**（三）表观遗传改变**

- DNA甲基化
- 组蛋白修饰
- 非编码RNA调节

### 3.2 影响发病的因素

| 因素 | 影响 |
|------|------|
| 暴露剂量 | 剂量越大，风险越高 |
| 暴露时间 | 时间越长，累积越多 |
| 暴露方式 | 吸入最危险 |
| 个体差异 | 代谢酶多态性 |
| 联合因素 | 多种致癌物协同 |
| 吸烟 | 增加风险 |

### 3.3 潜伏期特点

- 石棉暴露到肺癌：20-30年
- 苯暴露到白血病：5-10年
- 联苯胺暴露到膀胱癌：15-20年
- 砷暴露到肺癌：15-25年

## 四、临床表现

### 4.1 共同特点

1. **职业接触史**：长期职业致癌物暴露史
2. **潜伏期长**：通常10年以上
3. **早期无症状**：发现时多为晚期
4. **好发部位**：与暴露途径相关

### 4.2 常见职业性肿瘤

**（一）职业性肺癌**

- 症状：咳嗽、痰血、胸痛、呼吸困难
- 体征：可闻及干湿啰音
- 检查：胸部X线、CT、肿瘤标志物

**（二）职业性白血病**

- 症状：发热、贫血、出血
- 体征：肝脾淋巴结肿大
- 检查：血常规、骨髓检查

**（三）职业性膀胱癌**

- 症状：无痛性血尿
- 检查：膀胱镜、尿细胞学

**（四）职业性皮肤癌**

- 症状：皮肤溃疡、肿块
- 检查：病理检查

### 4.3 诊断要点

1. **职业史确认**：明确致癌物暴露史
2. **暴露评估**：浓度、时间评估
3. **临床诊断**：病理确诊
4. **排除其他**：排除非职业因素

## 五、诊断与鉴别诊断

### 5.1 诊断标准

根据GBZ 94-2017《职业性肿瘤的诊断》：

**诊断原则：**

1. 肿瘤诊断明确
2. 有明确的职业致癌物暴露史
3. 暴露时间足够长
4. 排除其他可能病因

### 5.2 诊断程序

1. **职业史调查**：详细了解职业经历
2. **暴露评估**：评估暴露水平
3. **临床诊断**：病理学确诊
4. **综合判断**：确定因果关系

### 5.3 鉴别诊断

- 散发性肿瘤
- 遗传性肿瘤
- 其他病因肿瘤

## 六、治疗与预后

### 6.1 治疗原则

1. **早发现**：早期诊断是关键
2. **综合治疗**：手术、放疗、化疗结合
3. **对症支持**：减轻痛苦
4. **康复治疗**：提高生活质量

### 6.2 治疗方法

| 方法 | 适用情况 |
|------|----------|
| 手术 | 早期肿瘤 |
| 放疗 | 敏感肿瘤 |
| 化疗 | 血液系统肿瘤 |
| 靶向治疗 | 有靶点肿瘤 |
| 免疫治疗 | 适合患者 |

### 6.3 预后

- **早期发现**：预后较好
- **晚期发现**：预后差
- **部分肿瘤**：如白血病预后仍差
- **关键**：早诊早治

## 七、预防措施

### 7.1 一级预防：病因预防

**（一）消除致癌因素**

- 禁止使用致癌物质
- 替代品开发
- 工艺改革

**（二）工程控制**

- 通风设施
- 湿式作业
- 密闭化生产

**（三）个人防护**

- 防尘口罩
- 防护服
- 皮肤防护

### 7.2 二级预防：早期发现

**（一）健康监护**

- 就业前体检
- 定期体检
- 肿瘤筛查

**（二）筛查项目**

| 肿瘤类型 | 筛查方法 |
|----------|----------|
| 肺癌 | 胸部CT |
| 膀胱癌 | 尿细胞学 |
| 皮肤癌 | 皮肤检查 |
| 白血病 | 血常规 |

### 7.3 三级预防：及时治疗

- 早诊早治
- 规范治疗
- 康复治疗

### 7.4 特定措施

**（一）石棉接触**

- 禁止使用石棉
- 湿式作业
- 防护服

**（二）苯接触**

- 低苯替代
- 通风防护
- 定期血常规

**（三）联苯胺接触**

- 替代品开发
- 防护设施
- 定期尿检

## 八、健康监护

### 8.1 就业前体检

- 排除禁忌证
- 肿瘤家族史评估
- 健康状况基线

### 8.2 在岗期间体检

- 定期体检
- 肿瘤筛查
- 早期发现异常

### 8.3 特殊检查

| 检查项目 | 适用人群 |
|----------|----------|
| 胸部CT | 石棉、砷暴露 |
| 血常规 | 苯暴露 |
| 尿细胞学 | 联苯胺暴露 |
| 皮肤检查 | 焦油、矿物油暴露 |

## 九、法律保护与权益

### 9.1 职业性肿瘤诊断

- 诊断机构：省级职业病诊断机构
- 诊断依据：GBZ 94-2017
- 需要材料：职业史、暴露评估、病理诊断

### 9.2 工伤认定

- 属于职业病
- 认定为工伤
- 享受相应待遇

### 9.3 用人单位责任

- 提供安全作业环境
- 落实防护措施
- 组织健康检查
- 告知致癌风险

### 9.4 劳动者权利

- 知情权
- 拒绝违章权
- 定期体检权
- 依法索赔权

## 十、总结

职业性肿瘤是最严重的职业病之一，但完全可以预防。

**核心要点：**

1. 职业性肿瘤潜伏期长
2. 病因明确，可预防
3. 早期发现是关键
4. 八种法定职业性肿瘤
5. 预防是根本措施

**预防要点：**

- 了解职业致癌因素
- 落实防护措施
- 定期健康体检
- 早期发现异常
- 依法维护权益

让我们共同关注职业性肿瘤预防，守护劳动者生命健康！
