# 职业性锰中毒：诊断、预防与处理指南

锰（Manganese）是工业生产中广泛应用的重要金属元素。职业性慢性锰中毒是长期接触锰烟尘所引起的以神经系统损害为主的疾病，严重时可导致帕金森氏综合征。本文依据GBZ 3-2006《职业性慢性锰中毒诊断标准》等国家标准，系统阐述其危害特点、诊断标准及防控措施。

## 一、适用范围

本标准（GBZ 3-2006《职业性慢性锰中毒诊断标准》）适用于：

- **锰矿开采**：锰矿勘探、开采、选矿
- **锰冶炼**：锰铁合金、电解锰、金属锰生产
- **焊接作业**：电焊条制造、焊接作业（特别是使用含锰焊条）
- **电池制造**：锰锌电池、干电池生产
- **玻璃陶瓷**：玻璃脱色剂、陶瓷釉料生产
- **油漆涂料**：催干剂、防腐剂生产
- **农业**：锰肥、杀虫剂（代森锰）生产使用

> **注意**：本标准主要适用于慢性锰中毒，急性锰中毒较少见。

## 二、毒性特点

### 2.1 锰的理化性质

| 项目 | 特性 |
|------|------|
| 化学式 | Mn |
| 外观 | 银灰色硬脆金属 |
| 原子量 | 54.94 |
| 熔点 | 1244℃ |
| 沸点 | 2061℃ |
| 存在形式 | 锰矿、锰烟、锰尘（二氧化锰、四氧化三锰等） |

### 2.2 毒代动力学

- **吸收**：主要经呼吸道吸入，也可经消化道吸收
- **分布**：分布于脑、肝、肾、胰腺等组织；可通过血脑屏障
- **代谢**：在体内参与多种酶的合成和激活
- **排泄**：主要经胆汁和肠道排泄，尿中排泄较少

### 2.3 中毒机制

1. **多巴胺代谢障碍**：锰选择性地蓄积于基底节（尤其是苍白球），抑制多巴胺合成
2. **线粒体损伤**：影响神经元能量代谢
3. **氧化应激**：诱导自由基产生，导致脂质过氧化
4. **神经细胞凋亡**：导致神经元程序性死亡
5. **星形胶质细胞增生**：形成不可逆的神经病理改变

### 2.4 临床表现

**早期症状**（神经衰弱综合征）：

| 系统 | 症状 |
|------|------|
| **神经系统** | 头晕、头痛、乏力、失眠、记忆力减退、嗜睡 |
| **植物神经** | 多汗、心悸、心动过速或过缓、食欲减退 |
| **精神症状** | 情绪改变（易激动、淡漠、兴趣缺乏） |

**中晚期症状**（锥体外系损害）：

| 系统 | 症状 |
|------|------|
| **运动障碍** | 肌张力增高（齿轮样强直）、动作迟缓、静止性震颤 |
| **步态异常** | 步态慌张、行走不稳、跌倒 |
| **面部表情** | 表情呆滞、面具脸 |
| **言语障碍** | 言语不清、语音低微 |
| **精神症状** | 幻觉、妄想、情感障碍 |

## 三、健康检查

### 3.1 上岗前体检

**必检项目**：
- 常规体检（内科、神经科）
- 血常规、尿常规
- 肝功能
- 神经系统检查

**职业禁忌证**：
- 神经系统器质性疾病
- 明显神经症
- 肝胆系统疾病
- 严重心血管疾病

### 3.2 在岗期间体检

**体检周期**：
- 接触锰浓度合格：每年1次
- 接触锰浓度超标：每半年1次
- 出现早期症状者：缩短体检间隔

**体检内容**：
- 详细病史询问（头晕、乏力、睡眠障碍、震颤等）
- 神经系统检查（肌张力、震颤、步态）
- 精神状态评估
- 尿锰测定（可选）

### 3.3 应急健康检查

接触大量锰尘后，应进行：
- 神经系统检查
- 尿锰测定
- 肝 四、诊断标准

### 4.1 诊断原则

依据GB功能检查

##Z 3-2006，职业性慢性锰中毒的诊断需根据：

1. **长期职业接触史**（通常1年以上）
2. **相应的临床表现**（以神经系统损害为主）
3. **现场职业卫生学调查**
4. **排除其他疾病**（如帕金森病、肝豆状核变性等）

### 4.2 诊断分级

| 分级 | 诊断条件 |
|------|----------|
| **观察对象** | 出现神经衰弱综合征或轻度植物神经功能紊乱，作业环境空气中锰浓度超过国家卫生标准 |
| **轻度中毒** | 出现神经衰弱综合征，植物神经功能紊乱，手指眼睑轻微震颤，腱反射亢进 |
| **中度中毒** | 出现以下情况之一：①明显锥体外系损害，如肌张力明显增高，轻度震颤；②精神感情改变，如表情淡漠、兴趣缺乏、多语或嗜睡 |
| **重度中毒** | 出现以下情况之一：①明显的帕金森氏综合征，如面具脸、静止性震颤、慌张步态；②中毒性精神病，如幻觉、妄想 |

### 4.3 鉴别诊断

- 原发性帕金森病
- 肝豆状核变性（Wilson病）
- 一氧化碳中毒后迟发性脑病
- 药物性锥体外系综合征
- 脑炎后帕金森综合征

## 五、接触限值

### 5.1 工作场所空气中的接触限值

根据GBZ 2.1-2007《工作场所有害因素职业接触限值 第1部分：化学有害因素》：

| 限值类型 | 限值 |
|----------|------|
| **时间加权平均浓度（PC-TWA）** | 0.15 mg/m³（按Mn计） |

### 5.2 生物监测指标

- **尿锰**：参考值为<0.5μmol/L（或<3μg/L）
- **血锰**：参考值为<0.15μmol/L

> **注意**：尿锰和血锰主要反映近期接触水平，与中毒程度不一定平行。

### 5.3 环境监测要求

- 定期检测工作场所空气中锰浓度
- 重点监测点：破碎、研磨、烧结、焊接等作业区域
- 建立锰浓度监测档案

## 六、预防措施

### 6.1 工程控制

1. **替代技术**：采用低锰或无锰焊接材料
2. **湿式作业**：防止锰尘飞扬
3. **通风设施**：
   - 全面通风：保持车间空气流通
   - 局部排风：在锰尘散发源设置排风罩
4. **自动化**：实现自动化操作，减少人工直接接触

### 6.2 个人防护

| 防护用品 | 适用场景 |
|----------|----------|
| **防尘口罩** | 常规作业（需配备有效的滤膜） |
| **防护眼镜** | 防止眼刺激 |
| **防护手套** | 防止皮肤接触 |
| **防护服** | 防止皮肤污染 |

> **注意**：应使用防尘效率≥95%的口罩（如N95或KN95）。

### 6.3 卫生措施

- **禁止在车间进食、饮水、吸烟**
- **工作服与个人服装分开存放**
- **下班后沐浴、更衣**
- **作业后严格洗手**
- **不在作业区居住或存放食品**

### 6.4 健康管理

- 建立锰作业人员健康档案
- 开展职业健康教育（特别是神经系统症状识别）
- 实施定期健康检查
- 及时发现和处理早期中毒者
- 对观察对象密切随访

## 七、急救处理

### 7.1 现场急救

**步骤一：迅速脱离暴露**
- 立即将中毒者移至新鲜空气处
- 脱除被污染的衣物
- 用清水冲洗皮肤

**步骤二：保持呼吸道通畅**
- 松开衣领，头偏向一侧
- 清除口腔分泌物

**步骤三：立即就医**
- 拨打120急救电话
- 告知可能为锰中毒

### 7.2 医院治疗

**主要治疗原则**：
1. **脱离接触**：立即停止锰暴露
2. **特效驱锰治疗**：
   - **依地酸钙钠（Na₂CaEDTA）**：每日1g，静脉滴注或肌注，连用3-5天，间隔2-3天再用，共2-3个疗程
   - **二巯丁二钠（DMSA）**：口服或静脉给药
3. **对症治疗**：
   - 帕金森症状：左旋多巴、金刚烷胺、安坦等
   - 精神症状：抗精神病药物
   - 肌张力障碍：肌松剂
4. **支持治疗**：
   - 维生素B族、维生素C
   - 保护肝肾功能

### 7.3 注意事项

- 驱锰治疗对已形成的神经系统损害效果有限，关键在于早期发现
- 锰在体内蓄积时间长，治疗周期较长
- 定期监测尿锰评估驱锰效果

### 8.2 法律法规

1. **《中华人民共和国职业病防治法》（2018年修正）**
   - 规定用人单位职业病防治责任
   - 要求开展职业健康检查、职业病危害因素检测

2. **《工作场所职业卫生监督管理规定》（国家安全生产监督管理总局令第47号）**
   - 用人单位职业卫生管理要求

3. **《职业病诊断与鉴定管理办法》（国家卫生健康委员会令第6号）**
   - 职业病诊断程序规定

4. **《工伤保险条例》（国务院令第586号）**
   - 职业病享受工伤保险待遇

5. **《职业病分类和目录》（国卫疾控发〔2013〕48号）**
   - 锰中毒列为法定职业病

### 8.3 用人单位责任

- 开展职业病危害项目申报
- 实施作业场所职业病危害因素检测与评价
- 为劳动者提供符合防治职业病要求的个人防护用品
- 组织劳动者进行职业健康检查（上岗前、在岗期间）
- 建立职业健康监护档案
- 报告职业病病例
