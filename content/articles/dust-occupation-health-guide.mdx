## 一、适用范围

本指南适用于从事粉尘作业的用人单位及劳动者，主要涉及以下行业：

- **采矿**：煤矿、金属矿、非金属矿开采
- **冶金**：矿石破碎、选矿、冶炼
- **建筑**：石材加工、混凝土作业
- **建材**：水泥、耐火材料、陶瓷、玻璃
- **化工**：化肥、农药制造
- **机械**：铸造、锻造、焊接
- **纺织**：棉纺、麻纺
- **粮食**：粮食加工、仓储

## 二、粉尘分类

### 2.1 按性质分类

| 类型 | 代表粉尘 | 致病性 |
|------|----------|--------|
| **无机粉尘** | 矽尘、石棉尘、水泥尘 | 可致尘肺 |
| **有机粉尘** | 棉尘、木尘、谷尘 | 可致职业性哮喘 |
| **混合粉尘** | 煤矽尘、铸工尘 | 可致复合尘肺 |

### 2.2 常见尘肺病

| 病名 | 粉尘来源 |
|------|----------|
| 矽肺 | 含游离二氧化硅的粉尘 |
| 煤工尘肺 | 煤矿粉尘 |
| 石棉肺 | 石棉粉尘 |
| 水泥尘肺 | 水泥粉尘 |
| 陶工尘肺 | 陶瓷、耐火材料粉尘 |
| 铝尘肺 | 铝金属粉尘 |
| 电焊工尘肺 | 电焊烟尘 |
| 铸工尘肺 | 铸造粉尘 |

## 三、职业健康检查

### 3.1 上岗前检查

**禁忌症：**

- 活动性肺结核
- 慢性阻塞性肺疾病
- 慢性肺间质疾病
- 肺切除术后
- 严重心血管疾病

### 3.2 在岗检查

| 粉尘类型 | 检查周期 |
|----------|----------|
| 矽尘 | 每年1次 |
| 煤尘 | 每年1次 |
| 石棉 | 每年1次 |
| 其他粉尘 | 根据浓度 |

**检查项目：**

1. **症状询问**
   - 咳嗽、咳痰
   - 胸闷、气短
   - 胸痛
   - 发热、盗汗

2. **体格检查**
   - 呼吸系统检查
   - 心血管系统检查

3. **辅助检查**
   - 胸部X射线
   - 肺功能检查

### 3.3 离岗检查

离岗时应进行职业健康检查，检查项目同在岗检查。

## 四、尘肺病诊断

### 4.1 诊断标准（GBZ 70-2015）

**诊断原则：**

1. 可靠的生产性粉尘接触史
2. X射线胸片符合尘肺病影像学特征
3. 结合职业卫生调查资料
4. 排除其他疾病

### 4.2 诊断分期

| 分期 | X射线表现 |
|------|-----------|
| **壹期** | 总体密集度1级，分布范围达2个肺区 |
| **贰期** | 总体密集度2级，分布范围达4个肺区；或总体密集度3级，达2个肺区 |
| **叁期** | 总体密集度3级，分布范围超4个肺区；或出现大阴影 |

### 4.3 并发症

- 肺结核
- 肺部感染
- 肺气肿
- 肺心病
- 呼吸衰竭

## 五、接触限值

### 5.1 粉尘浓度限值（GBZ 2.1-2007）

| 粉尘种类 | PC-TWA (mg/m³) |
|----------|----------------|
| 总粉尘 | 4 |
| 呼吸性粉尘 | 2 |
| 游离SiO₂≥10% | 1 |
| 游离SiO₂50%-80% | 0.7 |
| 游离SiO₂＞80% | 0.5 |
| 石棉 | 0.8 (纤维/mL) |
| 煤尘 | 4 |
| 水泥粉尘 | 4 |
| 铝尘 | 3 |
| 电焊烟尘 | 4 |

### 5.2 监测要求

| 监测类型 | 频次 |
|----------|------|
| 日常监测 | 每月至少1次 |
| 定期监测 | 每年至少1次 |
| 个体监测 | 接触浓度≥50%PC-TWA者 |

## 六、预防措施

### 6.1 工程控制

| 措施 | 说明 |
|------|------|
| 湿式作业 | 降低粉尘浓度 |
| 通风除尘 | 全面通风、局部排风 |
| 密闭隔离 | 设备密闭、操作隔离 |
| 工艺改革 | 用低粉尘材料替代 |

### 6.2 个人防护

- 防尘口罩（过滤效率≥95%）
- 防护服
- 防护眼镜

### 6.3 管理措施

- 危害告知
- 培训教育
- 健康监护
- 定期监测
