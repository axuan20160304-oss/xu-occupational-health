# 职业性噪声聋的预防措施与防护用品选用指南

## 概述

职业性噪声聋（Occupational Noise-Induced Deafness，ONID）是指劳动者在工作场所中长期接触噪声（通常为稳态噪声或非稳态宽频噪声）引起的感音神经性听力损失。《职业病分类和目录》将职业性噪声聋列为法定职业病。据统计，我国噪声作业人数超过1000万，职业性噪声聋是发病率最高的职业病之一。

噪声对人体的危害：
- **听力损伤**：暂时性阈移（TTS）或永久性阈移（PTS）
- **神经系统**：头痛、头晕、失眠、记忆力减退
- **心血管系统**：血压升高、心率加快
- **消化系统**：胃肠功能紊乱

## 噪声职业接触限值

### 国家标准

根据《GBZ 2.2-2007 工作场所有害因素职业接触限值》：

| 接触时间 | 噪声限值[dB(A)] |
|----------|-----------------|
| 8小时 | 85 |
| 4小时 | 88 |
| 2小时 | 91 |
| 1小时 | 94 |
| 30分钟 | 97 |

### 噪声作业定义

- 8小时等效连续A声级≥85dB(A)的作业
- 或任何时间峰值声压级≥115dB(C)的作业

## 工程控制措施

### 一、噪声源控制

1. **设备选型**：选用低噪声设备
2. **工艺改进**：采用低噪声工艺
3. **隔声设计**：给噪声设备安装隔声罩
4. **消声装置**：安装消声器

### 二、传播途径控制

1. **隔声墙**：车间墙壁采用隔声材料
2. **吸声处理：墙面、天花板安装吸声材料**
3. **隔振措施**：设备基座安装隔振垫
4. **绿化降噪**：厂区种植树木

### 三、布局优化

1. **噪声分区**：将高噪声区域与低噪声区域分开
2. **距离防护**：工作位置尽量远离噪声源
3. **时间控制**：减少在高噪声区域的停留时间

## 个人防护用品

### 护听器类型

#### 一、耳塞

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 一次性泡沫耳塞 | 成本低，使用方便 | 短时间作业 |
| 可重复使用硅胶耳塞 | 舒适度高，寿命长 | 长时间作业 |
| 定制耳塞 | 贴合度高，舒适 | 高频接触噪声者 |
| 带线耳塞 | 防止丢失 | 频繁进出场者 |

#### 二、耳罩

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 头戴式耳罩 | 隔声效果好 | 高噪声环境 |
| 颈挂式耳罩 | 便于携带 | 间歇性噪声 |
| 贴合式耳罩 | 配合安全帽使用 | 特殊作业环境 |
| 主动降噪耳罩 | 降低低频噪声 | 复杂噪声环境 |

### 护听器选型

#### 一、选择原则

1. **SNR值**：护听器的SNR值应≥25dB
2. **舒适度**：长时间佩戴舒适
3. **适用性**：适合工作环境
4. **维护性**：便于清洁和更换

#### 二、衰减量计算

实际隔声量 = SNR值 - 8dB（考虑实际佩戴误差）

| 噪声强度[dB(A)] | 需要的SNR值[dB] |
|-----------------|------------------|
| 85 | ≥17 |
| 90 | ≥22 |
| 95 | ≥27 |
| 100 | ≥32 |

### 护听器佩戴方法

#### 泡沫耳塞佩戴步骤

1. 清洁双手
2. 将耳塞搓成细长条状
3. 向上向后拉起外耳郭
4. 将耳塞塞入耳道
5. 等待耳塞膨胀回弹
6. 检查佩戴是否正确

#### 耳罩佩戴步骤

1. 调整头带至合适长度
2. 将耳罩完全覆盖双耳
3. 调整耳罩位置确保密封
4. 检查是否有缝隙
5. 佩戴后测试隔声效果

### 护听器使用注意事项

1. **正确佩戴**：确保佩戴方法正确
2. **及时更换**：损坏或变形时及时更换
3. **清洁保养**：定期清洁，保持卫生
4. **妥善存放**：不使用时放入保护盒

## 健康监护

### 上岗前体检

#### 目标

- 筛选禁忌证
- 建立听力基线
- 健康教育

#### 禁忌证

- 永久性听力损失
- 严重耳道疾病
- 精神心理疾病

### 在岗体检

#### 体检周期

| 噪声强度[dB(A)] | 体检周期 |
|-----------------|----------|
| 85-90 | 每年1次 |
| 90-100 | 每半年1次 |
| >100 | 每3个月1次 |

#### 体检项目

1. **病史采集**：噪声接触史、耳病史
2. **耳科检查**：外耳道、鼓膜
3. **纯音测听**：500Hz、1000Hz、2000Hz、3000Hz、4000Hz、6000Hz
4. **健康评估**：综合评估

#### 听力异常处理

| 检查结果 | 处理措施 |
|----------|----------|
| 听力下降<10dB | 密切观察 |
| 听力下降10-25dB | 复查，佩戴护听器 |
| 听力下降>25dB | 调离噪声岗位 |

### 离岗体检

- 全面听力评估
- 职业病诊断（如符合条件）

## 噪声监测

### 监测方法

1. **设备：**声级计、噪声剂量计
2. **布点：**工作场所每侧设点
3. **时间：**代表正常工作时段

### 监测周期

- 新建项目：验收时监测
- 正常生产：每年至少监测1次
- 设备变更：变更后监测

### 监测记录

- 监测日期、时间
- 监测点位
- 监测结果
- 评价结论

## 用人单位职责

### 管理措施

1. **噪声危害识别**：识别所有噪声作业岗位
2. **工程改造**：优先采取工程控制
3. **个人防护**：配备合格护听器
4. **健康监护**：组织定期体检
5. **培训教育**：普及噪声防护知识
6. **警示标识**：设置噪声警示标志

### 档案管理

- 噪声监测报告
- 职业健康检查档案
- 个人听力档案
- 护听器发放记录

## 劳动者权利

### 知情权

- 噪声危害及后果
- 防护措施
- 健康检查安排

### 建议权

- 改善工作环境
- 改进防护措施

### 拒绝权

- 拒绝违章指挥
- 拒绝冒险作业

### 职业病待遇

- 职业病诊断
- 工伤认定
- 相应待遇

## 总结

职业性噪声聋的预防需要采取综合措施：工程控制是根本，个人防护是关键，健康监护是保障。用人单位应落实噪声防护主体责任，劳动者应正确使用防护用品，双方共同努力预防职业性噪声聋的发生。
