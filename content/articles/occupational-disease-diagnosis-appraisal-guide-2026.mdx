# 职业病诊断与鉴定：劳动者权益保护指南

## 引言

职业病诊断与鉴定是职业病防治工作的重要环节，也是劳动者维护自身合法权益的关键途径。当劳动者怀疑自己患有职业病时，正确、及时地进行诊断和鉴定，不仅关系到能否获得及时的医疗救治，更关系到能否依法享受相应的工伤保险待遇和职业病防治保护。

我国《职业病防治法》明确规定了职业病诊断与鉴定的法律程序和要求。然而，由于职业病诊断涉及复杂的医学和法律问题，许多劳动者对相关程序和要求了解不足，导致自身权益未能得到充分保障。本文将从专业角度全面介绍职业病诊断与鉴定的相关知识，帮助广大劳动者依法维护自身权益。

## 一、职业病诊断概述

### 1.1 职业病诊断的定义

职业病诊断是指职业病诊断机构依据国家职业病诊断标准，对劳动者的疾病是否属于职业病进行判定的专业技术行为。职业病诊断是一项严肃的医学技术行为，具有法律效力。

### 1.2 职业病诊断的法律依据

**主要法律法规：**

| 法律法规 | 相关内容 |
|----------|----------|
| 《职业病防治法》 | 诊断机构、诊断程序、法律责任 |
| 《职业病诊断与鉴定管理办法》 | 具体操作规程 |
| GBZ 系列标准 | 各类职业病诊断标准 |
| 《工伤保险条例》 | 工伤认定与待遇 |

### 1.3 职业病诊断的特点

1. **强制性**：用人单位不得拒绝或阻碍诊断
2. **免费性**：诊断费用由用人单位承担
3. **客观性**：依据国家标准和医学证据
4. **法律性**：诊断结论具有法律效力

## 二、职业病诊断机构

### 2.1 诊断机构的设置

**省级卫生健康行政部门批准的职业病诊断机构：**

- 省级职业病防治院（所）
- 省级疾病预防控制中心
- 省级人民医院职业病科
- 经批准的其他医疗机构

### 2.2 诊断机构的职责

1. 依法开展职业病诊断
2. 报告职业病
3. 职业病诊断质量控制
4. 职业病诊断知识培训

### 2.3 选择诊断机构的注意事项

- 必须是省级卫生健康行政部门批准的机构
- 优先选择专业对口的机构
- 了解机构的资质和信誉
- 确认机构的地理位置和交通便利性

## 三、职业病诊断程序

### 3.1 诊断申请

**（一）申请主体**

- 劳动者本人
- 劳动者家属
- 用人单位
- 工会组织

**（二）申请条件**

- 劳动者有职业接触史
- 出现职业病相关症状
- 怀疑患有职业病

**（三）申请材料**

| 材料 | 说明 |
|------|------|
| 职业病诊断申请书 | 填写基本情况 |
| 劳动者职业史证明 | 单位盖章确认 |
| 职业健康检查结果 | 历次体检资料 |
| 工作场所危害检测报告 | 职业卫生技术机构出具 |
| 身份证复印件 | 身份证明 |
| 病历资料 | 既往就诊资料 |

### 3.2 职业史证明

**职业史证明是职业病诊断的关键材料，应包括：**

- 用人单位名称、地址
- 劳动者的工作岗位
- 接触的职业病危害因素
- 接触时间（起止日期）
- 工作场所危害因素浓度/强度

**注意事项：**

- 用人单位应当如实提供职业史证明
- 证明材料需加盖单位公章
- 劳动者应保存好自己的劳动合同、工资单等证据

### 3.3 诊断流程

```
提交申请材料
      ↓
诊断机构受理
      ↓
资料审核
      ↓
组织诊断
      ↓
出具诊断证明书
      ↓
送达当事人
```

### 3.4 诊断时限

- **受理时间**：材料齐全后15个工作日内
- **诊断时间**：受理后60日内完成
- **特殊情况**：可延长30日
- **集体诊断**：复杂病例可组织专家会诊

### 3.5 诊断结论

**诊断结论类型：**

| 结论 | 含义 |
|------|------|
| 确诊为职业病 | 属于职业病 |
| 观察对象 | 需进一步观察 |
| 不是职业病 | 排除职业病 |
| 需进一步检查 | 资料不足，需补充 |

## 四、职业病鉴定

### 4.1 鉴定类型

**（一）首次诊断鉴定**

- 职业病诊断机构首次作出的诊断
- 是后续鉴定的基础

**（二）再鉴定**

- 当事人对首次诊断结论有异议
- 向鉴定委员会申请鉴定

### 4.2 鉴定机构

**职业病鉴定委员会：**

- 省级卫生健康行政部门组织设立
- 由相关专业的专家组成
- 负责职业病鉴定的组织工作

### 4.3 鉴定申请

**（一）申请主体**

- 劳动者
- 用人单位
- 劳动者家属

**（二）申请时限**

- 收到诊断证明书之日起30日内

**（三）申请材料**

| 材料 | 说明 |
|------|------|
| 职业病鉴定申请书 | 填写鉴定请求 |
| 职业病诊断证明书 | 原诊断结论 |
| 相关病历资料 | 支持性医学资料 |
| 身份证复印件 | 身份证明 |

### 4.4 鉴定程序

```
提交鉴定申请
      ↓
审查材料
      ↓
组织鉴定专家组
      ↓
召开鉴定会
      ↓
出具鉴定结论
```

### 4.5 鉴定专家组的组成

- 必须是单数，一般为5人或以上
- 应当抽取相关专业的专家
- 与当事人有利害关系的专家应当回避
- 用人单位代表可参加鉴定会

### 4.6 鉴定结论

- 鉴定结论为最终结论
- 维持原诊断结论
- 变更原诊断结论
- 鉴定结论应当载明理由

## 五、职业病诊断标准

### 5.1 国家职业病诊断标准

**（一）尘肺病**

- GBZ 70-2015 职业性尘肺病的诊断

**（二）职业中毒**

- GBZ 68-2022 职业性苯中毒诊断标准
- GBZ 37-2015 职业性铅中毒诊断标准
- GBZ 15-2002 职业性急性氨气中毒诊断标准
- GBZ 10-2002 职业性急性氯气中毒诊断标准
- GBZ 83-2013 职业性砷中毒诊断标准

**（三）物理因素所致职业病**

- GBZ 49-2014 职业性噪声聋诊断标准
- GBZ 7-2014 职业性手臂振动病诊断标准

**（四）其他职业病**

- GBZ 18-2013 职业性皮肤病诊断标准
- GBZ 57-2019 职业性哮喘诊断标准
- GBZ 95-2014 职业性放射性白内障诊断标准

### 5.2 诊断标准的构成要素

1. **职业接触史**：可靠的有害因素接触史
2. **临床表现**：符合职业病的症状体征
3. **辅助检查**：实验室检查异常
4. **排除其他疾病**：排除非职业性疾病

### 5.3 诊断分级

以尘肺病为例：

| 分期 | X线表现 | 劳动能力 |
|------|---------|----------|
| 壹期 | 小阴影达一定程度 | 部分丧失 |
| 贰期 | 小阴影增多 | 明显丧失 |
| 叁期 | 大阴影出现 | 严重丧失 |

## 六、工伤认定

### 6.1 工伤认定的法律依据

- 《工伤保险条例》
- 《职业病防治法》
- 相关配套文件

### 6.2 职业病工伤认定条件

1. 职业病诊断证明书
2. 与用人单位存在劳动关系
3. 依法缴纳工伤保险

### 6.3 工伤认定程序

**（一）申请主体**

- 用人单位
- 劳动者或其家属
- 工会组织

**（二）申请时限**

- 用人单位：事故伤害发生之日起30日内
- 劳动者：1年内

**（三）申请材料**

| 材料 | 说明 |
|------|------|
| 工伤认定申请表 | 填写基本情况 |
| 劳动关系证明 | 劳动合同等 |
| 职业病诊断证明 | 职业病诊断机构出具 |
| 身份证明 | 劳动者身份证 |

### 6.4 工伤认定时限

- 受理后60日内作出决定
- 复杂情况可延长30日

### 6.5 工伤认定结论

- 认定为工伤
- 不认定为工伤

## 七、劳动者权利与义务

### 7.1 劳动者权利

**（一）诊断权利**

- 选择诊断机构的权利
- 获得免费诊断的权利
- 委托他人代为申请的权利

**（二）知情权利**

- 了解诊断标准的权利
- 获得诊断结果的权利
- 了解鉴定程序的权利

**（三）异议权利**

- 对诊断结论有异议可申请鉴定
- 对鉴定结论可申请再次鉴定（一次）

**（四）法律救济权利**

- 申请行政复议
- 提起行政诉讼
- 申请劳动仲裁

**（五）获得赔偿权利**

- 工伤医疗待遇
- 伤残待遇
- 死亡待遇

### 7.2 劳动者义务

1. 如实提供诊断所需材料
2. 配合诊断机构进行检查
3. 遵守诊断程序要求
4. 尊重诊断鉴定人员

### 7.3 用人单位责任

**（一）配合诊断**

- 不得拒绝提供职业史证明
- 不得阻碍劳动者申请诊断
- 应当如实提供相关材料

**（二）费用承担**

- 诊断费用
- 鉴定费用
- 治疗费用

**（三）法律责任**

- 不依法配合诊断的法律责任
- 不支付费用的法律责任
- 不落实待遇的法律责任

## 八、常见问题解答

### Q1：没有签订劳动合同可以诊断吗？

A：可以。需要提供其他劳动关系证明，如工资单、工作证、考勤记录等。

### Q2：用人单位不提供职业史证明怎么办？

A：可以向劳动监察部门投诉，或申请劳动仲裁确认劳动关系。

### Q3：诊断时需要自己付费吗？

A：不需要。职业病诊断费用由用人单位承担。

### Q4：对诊断结论有异议怎么办？

A：在收到诊断证明书之日起30日内，向省级职业病鉴定委员会申请鉴定。

### Q5：诊断为职业病后如何享受待遇？

A：需要进行工伤认定，然后根据伤残等级享受相应待遇。

### Q6：用人单位解散了怎么办？

A：职业病病人的医疗费用和生活保障由用人单位承担。用人单位解散的，应当从其资产中拨付。

### Q7：职业病诊断需要等很长时间吗？

A：诊断机构应当在受理后60日内作出诊断结论。

### Q8：可以同时申请多个诊断吗？

A：不可以。应当在一个诊断机构完成诊断后，如有异议再申请鉴定。

## 九、维权建议

### 9.1 证据收集

1. 保存劳动合同
2. 保存工资单
3. 保存体检报告
4. 保存工作证、出入证
5. 保存职业危害检测报告
6. 记录相关事实

### 9.2 程序要点

1. 及时申请诊断
2. 完整提交材料
3. 关注时限要求
4. 保留送达凭证
5. 依法维护异议权

### 9.3 寻求帮助

1. 工会组织
2. 职业卫生技术服务机构
3. 法律援助机构
4. 职业病防治机构

## 十、总结

职业病诊断与鉴定是劳动者依法维护自身权益的重要途径。

**核心要点：**

1. 诊断机构需省级卫生健康行政部门批准
2. 诊断费用由用人单位承担
3. 职业史证明是诊断的关键材料
4. 对诊断结论有异议可申请鉴定
5. 鉴定结论为最终结论
6. 工伤认定是享受待遇的前提

**劳动者应当：**

- 了解职业病诊断程序
- 及时进行诊断
- 依法维护权益
- 保留相关证据

让我们共同关注职业病诊断与鉴定，依法维护劳动者权益！
