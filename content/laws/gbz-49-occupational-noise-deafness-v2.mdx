## 1 范围

本标准规定了职业性噪声聋的诊断原则、诊断分级及处理原则。

本标准适用于长期职业接触噪声作业人员的职业性噪声聋诊断。

## 2 规范性引用文件

GBZ 188 职业健康监护技术规范

GBZ/T 189.8 工作场所物理因素测量 第8部分：噪声

GB/T 7582 声学 听阈与年龄关系的统计分布

## 3 诊断原则

### 3.1 职业接触史

连续3年以上职业性噪声接触史，年噪声暴露量≥80dB(A·年)。

### 3.2 临床表现

- 渐进性听力下降
- 耳鸣
- 纯音测听为感音神经性聋
- 听力损失双耳对称

### 3.3 排除其他原因

- 中耳炎
- 外伤
- 药物中毒
- 老年性聋
- 传染病源性聋

### 3.4 职业卫生调查

结合声级、接触时间等职业卫生调查资料。

## 4 诊断及分级

### 4.1 分级原则

根据较好耳500、1000、2000、3000、4000、6000 Hz 平均听阈进行分级。

### 4.2 分级标准

| 级别 | 双耳平均听阈(dB HL) | 表现 |
|------|---------------------|------|
| 轻度 | 26-40 | 耳鸣、听力下降 |
| 中度 | 41-55 | 日常对话困难 |
| 重度 | 56-70 | 明显听力障碍 |
| 极重度 | >70 | 严重听力障碍 |

### 4.3 诊断要点

4.3.1 连续3年以上职业性噪声接触史是诊断的基础条件

4.3.2 双耳高频（3000、4000、6000 Hz）平均听阈≥40 dB HL作为诊断听力损失的参考指标

4.3.3 听力损失为不可逆性改变

4.3.4 应排除其他原因所致的听觉损伤

## 5 处理原则

### 5.1 观察对象

听力损失达轻度者，应加强防护措施，避免听力进一步下降。

### 5.2 职业性噪声聋患者

5.2.1 应及时调离噪声作业岗位

5.2.2 进行听力康复治疗

5.2.3 根据听力损失程度给予适当赔偿

5.2.4 定期复查听力

## 6 正确使用说明

### 6.1 综合分析

诊断需综合考虑噪声暴露史、临床表现、听力检查结果，并排除其他原因。

### 6.2 纯音测听要求

纯音测听应按照GB/T 7582进行年龄修正，并按GBZ/T 189.8进行设备校正。

### 6.3 诊断机构

职业性噪声聋诊断应由省级卫生行政部门批准的职业病诊断机构进行。

### 6.4 鉴别诊断

| 疾病 | 鉴别要点 |
|------|----------|
| 药物中毒性聋 | 有耳毒性药物使用史 |
| 老年性聋 | 年龄相关，听力下降缓慢 |
| 传染病源性聋 | 有传染病史 |
| 中耳炎后遗症 | 有中耳炎病史 |
| 外伤性聋 | 有头部外伤史 |

## 7 职业健康检查

### 7.1 检查项目

| 检查项目 | 上岗前 | 在岗 | 离岗 |
|----------|--------|------|------|
| 纯音测听 | ✓ | ✓ | ✓ |
| 耳科检查 | ✓ | ✓ | ✓ |
| 询问症状 | ✓ | ✓ | ✓ |

### 7.2 检查周期

| 接触强度 | 检查周期 |
|----------|----------|
| ≥85 dB(A) | 每年1次 |
| 80-85 dB(A) | 每2年1次 |

## 8 噪声作业分级

| 级别 | 8h等效声级[dB(A)] |
|------|---------------------|
| 轻度 | 80-85 |
| 中度 | 85-90 |
| 重度 | 90-100 |
| 极重度 | >100 |

## 9 噪声聋分期

| 分期 | 表现 |
|------|------|
| 观察对象 | 耳鸣、听力下降，纯音测听无明显改变 |
| 轻度 | 双耳高频听力下降 |
| 中度 | 日常对话困难 |
| 重度 | 明显听力障碍 |
| 极重度 | 严重听力障碍，几乎听不到 |
