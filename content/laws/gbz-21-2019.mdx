## GBZ 2.1-2019 工作场所有害因素职业接触限值 第1部分：化学有害因素

标准编号：GBZ 2.1-2019

规定了工作场所化学有害因素的职业接触限值，包括时间加权平均容许浓度、短时间接触容许浓度和最高容许浓度。

### 1 范围

本标准规定了工作场所空气中化学有害因素的职业接触限值。

### 2 术语和定义

- **PC-TWA**：时间加权平均容许浓度，以时间为权数规定的8h工作日、40h工作周的平均容许接触浓度
- **PC-STEL**：短时间接触容许浓度，在遵守PC-TWA前提下容许短时间（15min）接触的浓度
- **MAC**：最高容许浓度，工作地点在一个工作日内任何时间有毒化学物质均不应超过的浓度

### 3 常见化学有害因素限值

| 化学物质 | PC-TWA (mg/m³) | PC-STEL (mg/m³) | MAC (mg/m³) |
|----------|----------------|------------------|-------------|
| 苯 | 6 | 10 | - |
| 甲苯 | 50 | 100 | - |
| 二甲苯 | 50 | 100 | - |
| 甲醛 | 0.5 | - | - |
| 铅及其化合物 | 0.05 | - | - |
| 汞及其化合物 | 0.02 | - | - |
| 锰及其化合物 | 0.15 | - | - |
| 一氧化碳 | 20 | 30 | - |
| 硫化氢 | - | - | 10 |

### 4 使用说明

4.1 本标准适用于工业企业卫生设计及工作场所化学有害因素的职业卫生检测与评价。

4.2 职业接触限值是职业病危害因素的接触限制量值，不是安全与危险的界限。

4.3 超标处理应采取工程控制、管理措施和个人防护等综合措施。
