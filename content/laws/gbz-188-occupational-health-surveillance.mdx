## 1 范围

本规范规定了职业健康监护的基本原则和接触人群、目标疾病、检查方法、复查、离岗后检查、应急检查及禁忌证等内容。

本规范适用于接触职业病危害因素劳动者的职业健康监护工作。

## 2 规范性引用文件

GBZ 2 工作场所有害因素职业接触限值

GBZ/T 157 职业病诊断标准

## 3 术语和定义

### 3.1 职业健康监护

为及时发现劳动者的职业健康损害，根据劳动者的职业接触史，对劳动者进行定期或不定期的医学健康检查。

### 3.2 职业健康检查

包括上岗前、在岗期间、离岗时和应急的健康检查。

### 3.3 禁忌证

劳动者从事特定职业或者接触特定职业病危害因素时，比一般职业人群更易于遭受职业病危害和可能导致原有自身疾病病情加重。

## 4 职业健康监护的基本原则

4.1 用人单位应建立职业健康监护制度。

4.2 职业健康检查应由取得资质的医疗卫生机构承担。

4.3 职业健康检查应遵循分类管理原则。

## 5 接触人群

### 5.1 接触职业病危害因素人群

| 危害因素 | 接触人群 |
|----------|----------|
| 粉尘 | 采矿、冶金、建筑工人 |
| 化学因素 | 化工、印刷工人 |
| 物理因素 | 噪声、高温、振动作业人员 |
| 生物因素 | 医护人员、兽医 |

### 5.2 特殊人群

| 人群 | 特殊要求 |
|------|----------|
| 未成年工 | 不得从事禁忌作业 |
| 妊娠哺乳期 | 不得从事禁忌作业 |
| 老年人 | 缩短接触时间 |

## 6 上岗前健康检查

### 6.1 检查目的

发现职业禁忌证，确定劳动者是否适合从事该作业。

### 6.2 检查时间

上岗前，或开始从事有害作业前。

### 6.3 必检项目

| 检查类型 | 项目 |
|----------|------|
| 常规 | 血常规、尿常规 |
| 肝功能 | ALT、AST |
| 肾功能 | 肌酐、尿素氮 |
| 心电图 | 常规12导联 |
| 影像学 | 胸部X射线 |

### 6.4 特殊检查

| 危害因素 | 检查项目 |
|----------|----------|
| 噪声 | 纯音听力测试 |
| 粉尘 | 肺功能 |
| 高温 | 血糖 |
| 苯 | 血常规 |

## 7 在岗期间健康检查

### 7.1 定期检查周期

| 危害因素 | 检查周期 |
|----------|----------|
| 噪声≥85dB | 每年1次 |
| 噪声80-85dB | 每2年1次 |
| 粉尘 | 每年1次 |
| 苯 | 每年1次 |
| 高温 | 每年1次 |
| 铅 | 每年1次 |
| 锰 | 每年1次 |

### 7.2 检查项目

| 检查类型 | 项目 |
|----------|------|
| 症状询问 | 咳嗽、胸闷、头晕、耳鸣 |
| 体格检查 | 内科、神经系统 |
| 辅助检查 | 根据危害因素确定 |

## 8 离岗时健康检查

### 8.1 检查时间

劳动者准备调离或脱离有害作业岗位前。

### 8.2 检查项目

与在岗期间相同。

## 9 应急健康检查

### 9.1 适用情况

发生急性职业病危害事故时。

### 9.2 检查范围

事故暴露人群。

### 9.3 检查项目

根据危害因素确定。

## 10 常见危害因素健康检查

### 10.1 噪声

| 检查类型 | 周期 | 项目 |
|----------|------|------|
| 上岗前 | - | 纯音测听 |
| 在岗 | 每年1次 | 纯音测听 |
| 离岗 | - | 纯音测听 |

**禁忌证**：
- 永久性感音神经性听力损失
- 中度以上传导性耳聋
- 眩晕综合征

### 10.2 粉尘

| 检查类型 | 周期 | 项目 |
|----------|------|------|
| 上岗前 | - | 胸部X射线、肺功能 |
| 在岗 | 每年1次 | 胸部X射线、肺功能 |
| 离岗 | - | 胸部X射线 |

**禁忌证**：
- 活动性肺结核
- 慢性阻塞性肺疾病
- 慢性肺间质疾病

### 10.3 苯

| 检查类型 | 周期 | 项目 |
|----------|------|------|
| 上岗前 | - | 血常规 |
| 在岗 | 每年1次 | 血常规 |
| 离岗 | - | 血常规 |

**禁忌证**：
- 血液系统疾病
- 肝肾疾病

### 10.4 高温

| 检查类型 | 周期 | 项目 |
|----------|------|------|
| 上岗前 | - | 内科检查 |
| 在岗 | 每年1次 | 内科、心电图 |
| 离岗 | - | 内科 |

**禁忌证**：
- 心血管疾病
- 中枢神经系统疾病
- 呼吸系统疾病

### 10.5 铅

| 检查类型 | 周期 | 项目 |
|----------|------|------|
| 上岗前 | - | 血常规、尿铅 |
| 在岗 | 每年1次 | 血常规、尿铅 |
| 离岗 | - | 血常规、尿铅 |

**禁忌证**：
- 神经系统疾病
- 肝肾疾病

### 10.6 锰

| 检查类型 | 周期 | 项目 |
|----------|------|------|
| 上岗前 | - | 神经系统检查 |
| 在岗 | 每年1次 | 神经系统检查 |
| 离岗 | - | 神经系统检查 |

**禁忌证**：
- 神经系统疾病
- 精神疾病

### 10.7 振动

| 检查类型 | 周期 | 项目 |
|----------|------|------|
| 上岗前 | - | 末梢循环、神经检查 |
| 在岗 | 每年1次 | 末梢循环、神经检查 |
| 离岗 | - | 末梢循环、神经检查 |

**禁忌证**：
- 雷诺病
- 末梢血管疾病

## 11 健康检查结果处理

### 11.1 禁忌证

发现禁忌证，应调离原岗位。

### 11.2 异常

发现异常，应复查、治疗。

### 11.3 职业病

发现职业病，应诊断、治疗。

## 12 档案管理

### 12.1 档案内容

- 职业史
- 既往病史
- 职业健康检查结果
- 诊断结论

### 12.2 保存期限

- 离岗后至少20年
- 职业病患者终身保存
