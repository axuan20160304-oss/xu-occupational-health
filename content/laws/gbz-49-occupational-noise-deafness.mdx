## 1 范围

本标准规定了职业性噪声聋的诊断原则、诊断分级及处理原则。

本标准适用于长期职业接触噪声作业人员的职业性噪声聋诊断。

## 3 诊断原则

3.1 连续3年以上职业性噪声接触史（年噪声暴露量≥80dB(A·年)）

3.2 出现渐进性听力下降、耳鸣，纯音测听为感音神经性聋

3.3 听力损失双耳对称

3.4 排除其他原因所致听觉损伤，如中耳炎、外伤、药物中毒等

3.5 结合职业卫生调查资料（声级、接触时间等）

## 4 诊断及分级

4.1 诊断及分级原则

根据较好耳500、1000、2000、3000、4000、6000 Hz 平均听阈进行分级。

4.2 分级标准

| 级别 | 双耳平均听阈(dB HL) |
|------|---------------------|
| 轻度 | 26-40 |
| 中度 | 41-55 |
| 重度 | 56-70 |
| 极重度 | >70 |

4.3 诊断要点

4.3.1 连续3年以上职业性噪声接触史是诊断的基础条件

4.3.2 双耳高频（3000、4000、6000 Hz）平均听阈≥40 dB HL作为诊断听力损失的参考指标

4.3.3 听力损失为不可逆性改变

4.3.4 应排除其他原因所致的听觉损伤

## 5 处理原则

5.1 观察对象

听力损失达轻度者，应加强防护措施，避免听力进一步下降。

5.2 职业性噪声聋患者

5.2.1 应及时调离噪声作业岗位

5.2.2 进行听力康复治疗

5.2.3 根据听力损失程度给予适当赔偿

5.2.4 定期复查听力

## 6 正确使用本标准的说明

6.1 职业性噪声聋的诊断需综合分析

诊断需综合考虑噪声暴露史、临床表现、听力检查结果，并排除其他原因。

6.2 纯音测听要求

纯音测听应按照GB/T 7582进行年龄修正，并按GBZ/T 189.8进行设备校正。

6.3 诊断机构

职业性噪声聋诊断应由省级卫生行政部门批准的的职业病诊断机构进行。

6.4 鉴别诊断

应与以下疾病鉴别：
- 药物中毒性聋
- 老年性聋
- 传染病源性聋
- 中耳炎后遗症
- 外伤性聋

### 7.1 职业健康检查项目

| 检查项目 | 上岗前 | 在岗 | 离岗 |
|----------|--------|------|------|
| 纯音测听 | ✓ | ✓ | ✓ |
| 耳科检查 | ✓ | ✓ | ✓ |
| 询问症状 | ✓ | ✓ | ✓ |

### 7.2 噪声作业分级

| 级别 | 8h等效声级[dB(A)] |
|------|---------------------|
| 轻度 | 80-85 |
| 中度 | 85-90 |
| 重度 | 90-100 |
| 极重度 | >100 |

### 7.3 噪声聋分期

| 分期 | 表现 |
|------|------|
| 观察对象 | 耳鸣、听力下降，纯音测听无明显改变 |
| 轻度 | 双耳高频听力下降 |
| 中度 | 日常对话困难 |
| 重度 | 明显听力障碍 |
| 极重度 | 严重听力障碍，几乎听不到 |
