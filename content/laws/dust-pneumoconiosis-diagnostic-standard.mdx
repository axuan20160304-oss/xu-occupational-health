## 一、尘肺病概述

### 1.1 定义

尘肺病（Pneumoconiosis）是指在职业活动中长期吸入生产性矿物性粉尘并在肺内潴留而引起的以肺组织弥漫性纤维化为主的全身性疾病。

### 1.2 分类

| 类型 | 主要粉尘来源 |
|------|--------------|
| 矽肺 | 二氧化硅粉尘（石英、花岗岩） |
| 煤工尘肺 | 煤矿粉尘 |
| 石棉肺 | 石棉粉尘 |
| 水泥尘肺 | 水泥粉尘 |
| 陶工尘肺 | 陶瓷、耐火材料 |
| 铝尘肺 | 铝金属粉尘 |
| 电焊工尘肺 | 电焊烟尘 |
| 铸工尘肺 | 铸造粉尘 |

## 二、主要诊断标准

### 2.1 尘肺病诊断标准（GBZ 70-2015）

**诊断原则：**

1. 可靠的生产性粉尘接触史
2.  chest X-ray 表现符合尘肺病的影像学特征
3. 结合职业卫生调查资料
4. 排除其他疾病

**诊断分期：**

| 分期 | X射线表现 |
|------|-----------|
| 壹期 | 总体密集度1级，分布范围达到2个肺区 |
| 贰期 | 总体密集度2级，分布范围达到4个肺区；或总体密集度3级，分布范围达到2个肺区 |
| 叁期 | 总体密集度3级，分布范围超过4个肺区；或出现大阴影 |
| +/s | 小阴影聚集 |

### 2.2 矽肺诊断标准（GBZ 70-2015）

**诊断要点：**

- 接触游离二氧化硅粉尘史
- X射线胸片特征性表现
- 肺功能损害程度
- 结合流行病学调查

**并发症：**

- 肺结核
- 肺部感染
- 肺气肿
- 肺心病
- 呼吸衰竭

### 2.3 煤工尘肺诊断标准

**诊断要点：**

- 煤矿粉尘接触史
- X射线胸片表现
- 肺功能检查
- 排除其他疾病

**临床特点：**

- 煤斑为特征性表现
- 进展相对缓慢
- 并发肺结核率较高

### 2.4 石棉肺诊断标准（GBZ 188-2025相关）

**诊断要点：**

- 石棉粉尘接触史
- X射线胸片表现
- 肺功能损害

**并发症：**

- 肺癌
- 间皮瘤
- 肺结核

## 三、接触限值

### 3.1 工作场所粉尘浓度限值（GBZ 2.1-2007）

| 粉尘种类 | PC-TWA (mg/m³) |
|----------|----------------|
| 总粉尘 | 4 |
| 呼吸性粉尘 | 2 |
| 游离二氧化硅≥10% | 1 |
| 游离二氧化硅50%-80% | 0.7 |
| 游离二氧化硅＞80% | 0.5 |
| 石棉 | 0.8 (纤维/mL) |
| 煤尘 | 4 |
| 水泥粉尘 | 4 |
| 铝尘 | 3 |
| 电焊烟尘 | 4 |

### 3.2 粉尘监测要求

| 监测类型 | 频次 |
|----------|------|
| 日常监测 | 每月至少1次 |
| 定期监测 | 每年至少1次 |
| 个体监测 | 接触浓度≥50%PC-TWA者 |

## 四、职业健康检查

### 4.1 上岗前检查

**必检项目：**

- 详细职业史
- 症状询问
- 内科常规检查
- 胸片
- 肺功能

**禁忌症：**

- 活动性肺结核
- 慢性阻塞性肺疾病
- 慢性肺间质疾病
- 肺切除术后

### 4.2 在岗检查

| 粉尘类型 | 检查周期 |
|----------|----------|
| 矽尘 | 每年1次 |
| 煤尘 | 每年1次 |
| 石棉 | 每年1次 |
| 其他粉尘 | 根据浓度 |

**检查项目：**

- 症状询问（咳嗽、咳痰、胸闷、气短）
- 体格检查
- 胸片
- 肺功能

### 4.3 离岗检查

- 离岗时应进行职业健康检查
- 检查项目同在岗检查

## 五、诊断程序

### 5.1 诊断机构

- 省级卫生健康行政部门批准的职业病诊断机构
- 3名以上医师集体诊断

### 5.2 诊断材料

1. 职业病诊断申请书
2. 劳动者职业史和职业病危害接触史
3. 职业健康检查结果
4. 工作场所职业病危害因素检测结果
5. X射线胸片等影像学资料

### 5.3 诊断时限

- 诊断机构应当自收到材料之日起30日内作出诊断结论
- 疑难病例可延长至60日

## 六、治疗与康复

### 6.1 治疗原则

- 病因治疗：脱离粉尘作业
- 对症治疗：缓解症状
- 康复治疗：改善肺功能
- 并发症治疗

### 6.2 肺康复

- 呼吸训练
- 运动训练
- 营养支持
- 心理支持

### 6.3 健康教育

- 戒烟
- 预防感染
- 呼吸防护
- 定期复查

## 七、预防措施

### 7.1 工程控制

| 措施 | 说明 |
|------|------|
| 湿式作业 | 降低粉尘浓度 |
| 通风除尘 | 全面通风、局部排风 |
| 密闭隔离 | 设备密闭、操作隔离 |
| 工艺改革 | 用低粉尘材料替代 |

### 7.2 个人防护

- 防尘口罩（过滤效率≥95%）
- 防护服
- 防护眼镜

### 7.3 管理措施

- 危害告知
- 培训教育
- 健康监护
- 定期监测
