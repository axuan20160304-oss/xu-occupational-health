## 1 范围

本标准规定了职业性中暑的诊断原则、诊断分级及处理原则。

本标准适用于从事高温作业人员的职业性中暑诊断。

## 2 规范性引用文件

GBZ 188 职业健康监护技术规范

GB/T 4200 高温作业分级

## 3 诊断原则

3.1 职业接触史

3.2 临床表现

3.3 体温测定

3.4 排除其他疾病

## 4 诊断分级

### 4.1 先兆中暑

在高温环境下，出现以下表现：

- 头晕
- 头痛
- 乏力
- 口渴
- 多汗
- 心悸
- 注意力不集中

**体温**：正常或略升高

### 4.2 轻度中暑

除先兆中暑症状外，出现以下表现：

- 面色潮红
- 面色苍白
- 大量出汗
- 脉搏快弱

**体温**：体温升高达38℃以上

### 4.3 重度中暑

出现以下表现：

- 意识障碍
- 抽搐
- 昏迷

**体温**：体温超过40℃

## 5 重度中暑分类

### 5.1 热射病

| 表现 | 说明 |
|------|------|
| 高热 | 体温骤升达40℃以上 |
| 无汗 | 皮肤干燥 |
| 意识障碍 | 意识模糊、昏迷 |
| 器官损伤 | 肝、肾、心脏损伤 |

### 5.2 热痉挛

| 表现 | 说明 |
|------|------|
| 肌肉痉挛 | 阵发性四肢肌肉痉挛 |
| 疼痛 | 剧烈疼痛 |
| 多汗 | 大量出汗 |
| 体温 | 正常或轻度升高 |

### 5.3 热衰竭

| 表现 | 说明 |
|------|------|
| 头晕 | 眩晕、站立不稳 |
| 恶心 | 恶心、呕吐 |
| 血压下降 | 脉搏细速、血压下降 |
| 体温 | 正常或轻度升高 |

## 6 处理原则

### 6.1 先兆中暑

- 转移至阴凉通风处
- 补充水分和盐分
- 休息观察

### 6.2 轻度中暑

- 脱离高温环境
- 物理降温
- 补充液体
- 对症处理

### 6.3 重度中暑

- 立即拨打120
- 迅速降温
- 保持呼吸道通畅
- 纠正水电解质紊乱
- 防治并发症

## 7 职业健康检查

### 7.1 上岗前检查（GBZ 188-2025）

**禁忌症**：

| 系统 | 疾病 |
|------|------|
| 心血管 | 高血压、冠心病、心律失常、心脏瓣膜病 |
| 神经系统 | 癫痫、重症神经症、脑外伤后遗症 |
| 呼吸系统 | 肺气肿、支气管哮喘、慢性支气管炎 |
| 消化系统 | 消化性溃疡、慢性胃炎、肝炎 |
| 泌尿系统 | 肾炎、肾功能不全 |
| 内分泌 | 糖尿病、甲亢 |
| 其他 | 严重贫血、皮肤病、妊娠期、哺乳期 |

### 7.2 在岗检查

- **周期**：每年1次（夏季前）
- **项目**：症状询问、体格检查、辅助检查

### 7.3 检查项目

| 检查类型 | 项目 |
|----------|------|
| 症状 | 头晕、乏力、心悸、胸闷 |
| 体格 | 体温、脉搏、血压、心肺 |
| 辅助 | 心电图、血常规、尿常规、肝功能 |

## 8 接触限值

### 8.1 高温作业分级（GB/T 4200）

| 接触时间率 | 26℃ | 28℃ | 30℃ | 32℃ |
|-----------|------|------|------|------|
| 25% | - | - | - | 轻度 |
| 50% | - | - | 轻度 | 中度 |
| 75% | - | 轻度 | 中度 | 重度 |
| 100% | 轻度 | 中度 | 重度 | 重度 |

### 8.2 气象条件监测

| 项目 | 要求 |
|------|------|
| 监测频次 | 夏季每7天1次 |
| 监测时间 | 10:00-14:00 |
| 监测项目 | 气温、湿度、风速、热辐射 |

## 9 预防措施

### 9.1 技术措施

| 措施 | 说明 |
|------|------|
| 隔热 | 隔热材料、水幕、空气幕 |
| 通风 | 全面通风、局部通风 |
| 降温 | 空调、冷风机 |
| 布局 | 热源布置在下风向 |

### 9.2 管理措施

- 合理安排作业时间（避开12:00-15:00）
- 实行轮换作业
- 减少持续接触时间
- 健康监护

### 9.3 个人防护

- 穿着白色或浅色透气工作服
- 佩戴防护帽、防护眼镜
- 补充水分和盐分

### 9.4 健康教育

- 防暑降温知识培训
- 中暑症状识别
- 急救知识培训

## 10 急救设备

| 设备 | 数量 | 位置 |
|------|------|------|

| 淡盐水 | 充足 | 休息区 |
| 急救箱 | 1 | 车间 |
| 担架 | 1 | 车间 |
