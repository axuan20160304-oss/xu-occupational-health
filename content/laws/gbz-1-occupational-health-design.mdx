## 1 范围

本标准规定了工业企业选址与总体布局、工作场所基本卫生要求、防护设施的基本技术要求，以及辅助用室的基本卫生要求。

本标准适用于新建、改建、扩建和技术改造工业企业的设计以及工业企业运行过程中的管理。

## 3 选址与总体布局

### 3.1 选址

3.1.1 工业企业选址应符合国家规定的基本卫生防护距离要求。

3.1.2 选址应满足以下要求：

- 空气质量良好
- 地势干燥
- 交通便利
- 远离污染源

### 3.2 总体布局

3.2.1 功能分区

| 区域 | 要求 |
|------|------|
| 生产区 | 主导风向的下风侧 |
| 生活区 | 生产区最小频率风向的上风侧 |
| 辅助区 | 介于两者之间 |

3.2.2 防护距离

| 污染类型 | 距离 |
|----------|------|
| 粉尘 | ≥100m |
| 有毒气体 | ≥50m |
| 噪声 | ≥30m |

## 4 工作场所基本卫生要求

### 4.1 采光与照明

4.1.1 采光

| 场所 | 采光系数 |
|------|----------|
| 精密工作室 | ≥1/6 |
| 一般工作室 | ≥1/8 |
| 辅助用房 | ≥1/10 |

4.1.2 照明

| 场所 | 照度(lx) |
|------|----------|
| 精细作业 | ≥300 |
| 一般作业 | ≥100 |
| 走廊 | ≥50 |

### 4.2 通风

4.2.1 自然通风

- 应有足够的天窗、侧窗
- 气流组织合理

4.2.2 机械通风

- 局部排风
- 全面通风
- 通风量根据污染程度确定

### 4.3 保温与隔热

4.3.1 高温车间

- 设置隔热措施
- 设置通风降温设施
- 供给清凉饮料

4.3.2 寒冷地区

- 设置防寒设施
- 室内温度≥16℃

## 5 防护设施

### 5.1 防尘设施

| 设施 | 适用 |
|------|------|
| 密闭罩 | 粉尘发生源 |
| 通风除尘 | 设备除尘 |
| 湿式作业 | 粉尘作业 |
| 个体防护 | 防尘口罩 |

### 5.2 防毒设施

| 设施 | 适用 |
|------|------|
| 隔离操作 | 剧毒作业 |
| 通风排毒 | 有毒物质 |
| 净化装置 | 排放废气 |
| 个体防护 | 防毒面具 |

### 5.3 防噪声设施

| 设施 | 适用 |
|------|------|
| 隔声 | 噪声源 |
| 消声 | 气流噪声 |
| 减振 | 设备振动 |
| 个体防护 | 防噪声耳塞 |

### 5.4 防振动设施

| 设施 | 说明 |
|------|------|
| 隔振 | 振动设备 |
| 减振垫 | 设备基础 |
| 个体防护 | 防振手套 |

### 5.5 防高温设施

| 设施 | 说明 |
|------|------|
| 隔热 | 热源隔热 |
| 通风 | 强制通风 |
| 空调 | 特殊岗位 |
| 防护用品 | 防热服 |

### 5.6 防非电离辐射

| 类型 | 防护 |
|------|------|
| 紫外辐射 | 防护眼镜、防护服 |
| 红外辐射 | 隔热防护 |
| 射频辐射 | 屏蔽防护 |
| 激光 | 防护眼镜 |

### 7.7 防电离辐射

| 类型 | 防护 |
|------|------|
| X射线、γ射线 | 铅屏蔽、时间防护、距离防护 |
| 放射性物质 | 密封操作、通风 |

## 6 辅助用室

### 6.1 卫生用室

| 用室 | 要求 |
|------|------|
| 厕所 | 车间附近、清洁、通风 |
| 盥洗室 | 设有洗手设施 |
| 淋浴室 | 高温作业设置 |
| 休息室 | 清洁、安静 |

### 6.2 生活用室

| 用室 | 要求 |
|------|------|
| 食堂 | 远离污染源 |
| 宿舍 | 通风、采光良好 |
| 托儿所 | 远离生产区 |

### 6.3 职业健康用室

| 用室 | 用途 |
|------|------|
| 急救站 | 紧急救援 |
| 卫生室 | 医疗保健 |
| 监测室 | 职业病危害监测 |

## 7 职业病危害控制

### 7.1 危害等级（GBZ/T 194）

| 等级 | 危害程度 |
|------|----------|
| 0级 | 安全 |
| Ⅰ级 | 轻度危害 |
| Ⅱ级 | 中度危害 |
| Ⅲ级 | 高度危害 |
| Ⅳ级 | 极度危害 |

### 7.2 控制措施

| 等级 | 措施 |
|------|------|
| 0级 | 保持现有防护措施 |
| Ⅰ级 | 危害告知、培训 |
| Ⅱ级 | 采取工程控制 |
| Ⅲ级 | 限期整改 |
| Ⅳ级 | 立即停止作业 |

## 8 监测与体检

### 8.1 职业病危害因素监测

| 监测类型 | 频次 |
|----------|------|
| 日常监测 | 每月1次 |
| 定期监测 | 每年1次 |
| 事故监测 | 发生时 |

### 8.2 职业健康检查

| 检查类型 | 对象 | 频次 |
|----------|------|------|
| 上岗前 | 全部接害人员 | 上岗前 |
| 在岗 | 全部接害人员 | 每年或按规定 |
| 离岗 | 全部接害人员 | 离岗时 |
