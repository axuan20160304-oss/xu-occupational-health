## 1 范围

本标准规定了职业性尘肺病的诊断原则、诊断分期及处理原则。

本标准适用于职业活动中长期吸入生产性矿物性粉尘引起的尘肺病的诊断。

## 3 诊断原则

3.1 可靠的生产性粉尘接触史

3.2 符合尘肺病X射线胸片特征性表现

3.3 结合职业卫生调查资料

3.4 排除其他疾病

## 4 诊断分期

### 4.1 壹期尘肺

X射线胸片表现为：

- 总体密集度1级
- 分布范围至少2个肺区

### 4.2 贰期尘肺

X射线胸片表现为：

- 总体密集度2级
- 分布范围至少4个肺区

或

- 总体密集度3级
- 分布范围2个肺区

### 4.3 叁期尘肺

X射线胸片表现为：

- 总体密集度3级
- 分布范围超过4个肺区

或

- 出现大阴影

### 4.4 尘肺病X射线表现

| 类型 | 表现 |
|------|------|
| 小阴影 | 圆形、不规则形 |
| 大阴影 | 长径≥20mm的阴影 |
| 胸膜斑 | 厚度≥5mm的钙化胸膜增厚 |
| 密集度 | 1级、2级、3级 |

## 5 常见尘肺病

### 5.1 矽肺

- **病因**：游离二氧化硅粉尘（石英、花岗岩）
- **特点**：进展快、后果严重
- **并发症**：肺结核、肺癌

### 5.2 煤工尘肺

- **病因**：煤矿粉尘
- **特点**：煤斑为特征性表现
- **并发症**：肺结核、慢性支气管炎

### 5.3 石棉肺

- **病因**：石棉粉尘
- **特点**：胸膜斑常见
- **并发症**：肺癌、间皮瘤

### 5.4 其他尘肺

| 病名 | 粉尘来源 |
|------|----------|
| 水泥尘肺 | 水泥 |
| 陶工尘肺 | 陶瓷、耐火材料 |
| 铝尘肺 | 铝金属 |
| 电焊工尘肺 | 电焊烟尘 |
| 铸工尘肺 | 铸造粉尘 |

## 6 处理原则

### 6.1 尘肺患者

- 及时调离粉尘作业岗位
- 根据病情进行康复治疗
- 对症治疗并发症
- 定期复查

### 6.2 观察对象

- 定期复查
- 动态观察
- 必要时调离

## 7 职业健康检查

### 7.1 上岗前检查

**禁忌症**：

- 活动性肺结核
- 慢性阻塞性肺疾病
- 慢性肺间质疾病
- 肺切除术后
- 严重心血管疾病

### 7.2 在岗检查

| 粉尘类型 | 检查周期 |
|----------|----------|
| 矽尘 | 每年1次 |
| 煤尘 | 每年1次 |
| 石棉 | 每年1次 |
| 其他 | 根据浓度 |

### 7.3 检查项目

- 症状询问（咳嗽、咳痰、胸闷、气短）
- 体格检查
- 胸部X射线
- 肺功能检查

## 8 接触限值

### 8.1 工作场所粉尘浓度限值（GBZ 2.1-2007）

| 粉尘种类 | PC-TWA (mg/m³) |
|----------|----------------|
| 总粉尘 | 4 |
| 呼吸性粉尘 | 2 |
| 游离SiO₂≥10% | 1 |
| 游离SiO₂50%-80% | 0.7 |
| 游离SiO₂＞80% | 0.5 |
| 石棉 | 0.8 (纤维/mL) |
| 煤尘 | 4 |
| 水泥粉尘 | 4 |
| 铝尘 | 3 |
| 电焊烟尘 | 4 |

### 8.2 监测要求

| 监测类型 | 频次 |
|----------|------|
| 日常监测 | 每月至少1次 |
| 定期监测 | 每年至少1次 |
| 个体监测 | 接触≥50%PC-TWA者 |

## 9 预防措施

### 9.1 工程控制

- 湿式作业
- 通风除尘
- 密闭隔离
- 工艺改革

### 9.2 个人防护

- 防尘口罩
- 防护服
- 防护眼镜

### 9.3 管理措施

- 危害告知
- 培训教育
- 健康监护
