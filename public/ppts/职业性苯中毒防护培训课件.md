# 职业性苯中毒防护培训课件

## 第一部分：苯的基本知识

### 什么是苯？
苯是一种有机溶剂，常用于：
- 制鞋行业：鞋底粘合
- 印刷行业：油墨稀释
- 电子行业：清洗剂
- 化工行业：溶剂

### 苯的危害
- 急性中毒：头晕、恶心、昏迷
- 慢性中毒：贫血、白细胞减少
- 致癌性：白血病
- 生殖毒性：影响生育

## 第二部分：苯中毒的诊断

### 诊断标准（GBZ 91）

**诊断原则：**
1. 职业接触史（6个月以上）
2. 出现临床表现
3. 血常规异常
4. 排除其他疾病

### 诊断分期

| 分期 | 标准 |
|------|------|
| 观察对象 | 血常规异常，无明显症状 |
| 轻度中毒 | 头晕、乏力，白细胞＜4×10⁹/L |
| 中度中毒 | 贫血，血小板减少 |
| 重度中毒 | 再生障碍性贫血、白血病 |

## 第三部分：苯中毒的预防

### 企业措施

1. **替代工艺（首选）**
   用水性胶代替苯系胶

2. **工程控制**
   - 通风设施
   - 隔离操作
   - 监测报警

3. **个人防护**
   - 防毒面具
   - 防护手套
   - 工作服

### 员工防护

1. 严格防护：戴防毒面具
2. 卫生习惯：班后沐浴
3. 健康监测：每年体检

## 第四部分：苯中毒的治疗

### 治疗原则
```
脱离苯接触 → 对症治疗 → 营养支持 → 康复
```

### 治疗方法
- 药物治疗：维生素B族、铁剂
- 支持治疗：输血、隔离
- 分期治疗：轻/中/重度不同方案

## 第五部分：案例警示

### 典型案例
某制鞋厂工人：
- 工龄：5年
- 症状：反复发热、乏力
- 诊断：职业性苯中毒（重度）

### 防范要点
- 正确佩戴防毒面具
- 严禁在车间进食
- 身体不适及时就医
